    Pitt Fuel #

    最後更新日期2021.09.08


    Moocho提供各個校園的第三方支付平台，可在當地合作商家付錢。其中，Pitt Fuel是匹茲堡地區的平台，開放任何人註冊。

    Moocho app或Pitt Fuel app均可在匹茲堡使用，目前商家主要分佈在Oakland與Shadyside。

    Pitt Fuel註冊對象
    任何人均可註冊，不限定Pitt學生。

    特色 #

    • 可連結Credit Card或Debit Card儲值，並提供額外儲值金。

      購買Git Card
      可在Market District購買Gift Card。如果以儲值$300可獲得$330計算，等於獲得10%折扣。
      避免囤積大量儲值金
      由於Moocho屬於新創的第三方支付平台，如果倒閉有損失儲值金的風險。

    • 每$5可獲得1 Mooche，可對換贈品 。

    • 完成Mission可獲得額外Mooches。

    付費方法 #

    1. 在手機app點到指定商家 ，右滑啟用
    2. 到人工櫃台跟店員說要用Pitt Fuel支付
    3. 店員會確認名字
    4. 完成支付

    尚未普及
    該支付尚未普及，因此如果店員不知道如何付費，可能要請經理協助。

    退貨政策不明
    Moocho並未提供退貨操作，因此建議使用於生鮮或消耗品支出。
