    禮卡 #

    最後更新日期2021.05.24

    禮卡最初的目的是一個不失禮貌送禮方法。當結合店家的促銷手段、二手禮卡折扣或信用卡回饋，買禮卡也可以是一個省錢手段。

    購買禮品卡前，首先需要了解下禮品卡都有使用條件與分類：

    • 部分禮卡僅能用線上或現場支付。
    • 使用禮卡能不能與其他優惠共用。
    • 有沒有時間限制。
    • 單筆交易使用張數限制。
    • 收單機構信用卡發行禮卡，例如如Visa Gift Card、American Express Gift Card 、Master Prepaid Card等，這種禮品卡和Debit Card使用方式一樣，但是購買這類禮卡通常需要支付額外費用。
    • 實物卡（Physical Card）需要等待郵寄；eGift Card一般透過email寄達，可立即使用。

    購買禮卡省錢概念： #

    • 購買禮卡、用禮卡支付款項通常都可以合併使用Shopping Portals，可以獲得多重回饋，詳情參考Shopping Portals章節。
    • 如果持有的信用卡有高額超市回饋，可以到超市購買禮卡。
    • 二手禮卡通常都會打折，通常越難用的店家（例如特定的服飾店或小眾餐廳）折扣越好。
    • 如果剛辦的信用卡信用額度不夠高額消費，透過分批買入禮卡則避免超用信用額度。

    購買與使用禮卡注意事項 #

    • 購買禮卡時可能會被要求看Photo ID。

    • Amex信用卡不能透過購買禮卡來達成開卡禮刷卡要求。

    • 二手禮卡比價可以參考Gift Card Wiki網頁。


    • 使用禮卡支付，商品就不享受到信用卡的延長保固、購買保護等福利。

    • 二手禮卡通常都有一定的保障時間。如果卡內餘額有問題，在符合條款的情況下，都可以通過購買平台追回錢。買完二手禮卡建議馬上使用，一有問題馬上回報退款！
