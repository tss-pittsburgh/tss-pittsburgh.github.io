    折扣季 #

    最後更新日期2021.05.24

    如同台灣一樣，美國各大商家會在特定時間推出大力折扣，以下簡單提幾個常見的折扣季。

    Back to School #

    時間：8月中旬至9月初。
    各大商家都會推出針對學生市場的一些促銷折扣，例如電子產品、學生用品。

    Labor’s Day #

    時間：9月第一個禮拜一。
    主要為夏季清倉拍賣。

    Thanksgiving Day #

    時間：11月~11月第四個星期四。
    從感恩節前1個月左右開始，各大商家都在這個時候推出最大折扣。感恩節的促銷打折囊括了各種產品。主要為電子產品、大型電器等。

    Black Friday #

    時間：11月第四個星期五。
    以往黑五這天會有Early birds special，提供最優惠的價格或贈品，所以商場或Outlet都是大排長龍。但近年，網上商店也提供有同等的促銷活動了。

    Christmas to New Year #

    時間：12/25~1/1。
    主要打折商品為聖誕禮物、美妝、服飾、鞋類。

    Cyber Monday #

    時間：各大節日後的星期一。
    延續各大節日，打折活動主要限於網路上。
