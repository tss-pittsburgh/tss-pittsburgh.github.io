    Outlet與折扣季 #

    最後更新日期2021.05.24

    美國的打折商品通常有三個渠道來購買，一是官網、品牌實體門市的季節性或假日折扣；二是百貨公司的專櫃的季節性或假日折扣；以及三是品牌專有的打折店，也就是俗稱的Outlet店。基本上，透過第一或第二種方法購買到的商品幾乎沒有差異，頂多是打折商品是過季商品或一些福利品。然而，在Outlet店所購買的商品，與上述前兩者並非相同。

    Outlet商品來源 #

    Outlet產品的來源主要有兩種：

    • 某些品牌生產的商品如果不能完全符合市場的需求，該商品就可能轉去Outlet店。
    • 部分Outlet店商品的生產線是獨立於正價商品的，而Outlet店也就是常聽見的Factory Store。通常產地、Logo、材質與製造方式都可能有所不同。

    無論是專櫃款式還是Outlet商品，符合自己需求的商品就能愉快購物。

    匹茲堡附近Outlet #

    匹茲堡附近主要有兩間Outlet。

    Tanger Outlets Pittsburgh #

    Tanger Outlets Pittsburgh車程約40分鐘。除了網路上免費索取折價券，亦可花費（一次性費用，$10）加入終身會員得到額外折扣、累積消費。

    Grove City Premium Outlets #

    Grove City Premium Outlets車程約1小時。網路上免費註冊會員後，可以客服中心索取折價券（疫情期間線上下載）。

    Ross Park Mall
    Ross Park Mall為匹茲堡附近最大商場，車程約20分鐘，並非Outlet店。

    折扣季 #

    如同台灣一樣，美國各大商家會在特定時間推出大力折扣，以下簡單提幾個常見的折扣季。

    Back to School #

    時間：8月中旬至9月初。
    各大商家都會推出針對學生市場的一些促銷折扣，例如電子產品、學生用品。

    Labor’s Day #

    時間：9月第一個禮拜一。
    主要為夏季清倉拍賣。

    Thanksgiving Day #

    時間：11月~11月第四個星期四。
    從感恩節前1個月左右開始，各大商家都在這個時候推出最大折扣。感恩節的促銷打折囊括了各種產品。主要為電子產品、大型電器等。

    Black Friday #

    時間：11月第四個星期五。
    以往黑五這天會有Early birds special，提供最優惠的價格或贈品，所以商場或Outlet都是大排長龍。但近年，網上商店也提供有同等的促銷活動了。

    Christmas to New Year #

    時間：12/25~1/1。
    主要打折商品為聖誕禮物、美妝、服飾、鞋類。

    Cyber Monday #

    時間：各大節日後的星期一。
    延續各大節日，打折活動主要限於網路上。
