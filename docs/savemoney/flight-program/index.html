    飛行常客獎勵計畫 #

    最後更新日期2021.05.24

    飛行常客獎勵計劃（Frequent Flyer Program）是許多航空公司給忠實乘客的一種獎勵方案，其中一個最重要的獎勵是航空哩程計畫，允許使用累積的哩程來兌換免費的機票、艙位升等或其他服務、商品的權利。


    航空哩程計畫規則

    以下為航空哩程計畫的基本概念

    • 航空哩程主要可以透過搭乘飛機、使用聯名信用卡以及銀行點數（例如Chase的UR點、Amex的MR點）轉入等方式獲得。

    • 世界上有三大航空聯盟：星空、寰宇、天合。共享航空代碼、轉機與部分服務（例如貴賓室）。

      • 星空聯盟常見航空公司：長榮、全日空、加拿大航空、聯合航空。
      • 寰宇一家常見航空公司：美國航空、日本航空、阿拉斯加航空、國泰航空。
      • 天合聯盟常見航空公司：華航、達美航空、法國航空、荷蘭皇家航空。

    • 搭乘聯盟內航空公司的航班時，可以累積到同一聯盟內的任一航空公司，必須在購票時輸入或Check-in時給地勤對應的哩程計畫的會員號碼。

      • 舉例來說：搭乘聯合航空（UA），可以要求存入聯合或長榮（EVA）等星空聯盟的哩程帳戶。惟須注意，事先需要到網站註冊好哩程帳戶以及存入比例依艙等不同，艙等存入比例可以使用Where to Credit網站查詢。

    • 哩程帳戶是獨立、不互相通的，哩程一旦存入某個航空公司的哩程帳戶後就不能轉到其他航空公司。相反地，銀行點數可以自由轉入特定航空哩程計畫。

      • 舉例來說，一旦哩程已經存入了UA帳戶，就不能轉移到任何的帳戶。除非有類似全日空的全家桶帳戶，哩程也不能與其他人共用。因此，如果哩程東存一個西存一個，最後會都不足以使用，必須好好規畫什麼哩程計畫是自己感興趣的。

    • 一個哩程帳戶可以兌換同一聯盟任一個航空公司的機票。惟須注意，每個航空公司的哩程計畫兌換自家或別家航空公司航班所需的哩程均有所差異。

      • 舉例來說：可以使用95,000的ANA哩程兌換台美來回的長榮商務艙。當然也可以使用長榮哩程兌換，但就需要160,000的長榮哩程。
      • 哩程票搜尋比較可以使用AwardHacker網站查詢，該網站可以具體提供從A到B各種哩程計畫所需里程以及銀行轉點方式。
