    學校資源 #

    謝牧君主編
    最後更新日期2021.05.24

    學校經常提供的免費資源或會議補助。

    Pitt #

    Pantry #

    Pitt Pantry是Pitt學生免費食物領取的渠道，每月可以申請2次。

    • 每週二或四中午12:00前必須填寫完表單，填寫學生證2P的後四碼pitt Email。
    • 每週三15:00~19:00領取。
    • 分發地點為Thackeray Avenue side porch of Bellefield Presbyterian Church。
    • 如果有你不想要的食物，將它放置在附近標有"donations"的盒子。

    CMU #

    Pantry #

    CMU Pantry是CMU學生免費食物領取的渠道，每週可以申請1次。

    • 可領取多種罐頭、米飯、義大利麵、蔬菜水果、牛奶雞蛋、零食、醬料、生活用品等等。每天的東西都不太一樣且依照類型有數量限制。
    • 建議提早去排隊，許多食物的數量有限，早點排隊可有較多的選擇
    • 疫情期間改為填寫 Online Order Form 然後由工作人員幫你挑選打包，無法自行選擇
    • 領取前需填寫問卷 Pantry Registration Survey 與 Online Order Form兩份文件。
    • 問卷填寫完畢後，於開放時間內 (Pick-Up Hours) 攜帶學生證領取食物。

    Conference Funding #

    CMU Conference Funding限定CMU Graduate Student申請。

    • 每學年（academic year）可申請一次獎學金補助，採用lottery抽籤（中獎機率約1/2）。
    • 補助金額為：一個人上限 $750，一個團體上限 $1,500。
      補助平分
      如果一個科系有三個人中籤並獲得補助，且都參加同一個conference，則這三人自動視為同一個團體平分$1,500的補助，建議可與系上承辦人員討論以了解其他同學的申請狀況。
    • 補助金額可報銷參加 conference 的所有花費包含：門票、交通、食物、住宿（不包含衣著與社交費用）。疫情期間報銷範圍可能有調整，建議參照頁面最新公告。
    • 申請流程：
      1. 確認會議舉辦時間，在學校網頁確定申請funding的截止時間。一年有四次申請時間。
      2. 參照頁面內的 APPLICATION 區塊了解申請流程。其中包含填寫申請表、與系上 Advisor 討論流程與提供申請需要的資料。
      3. Advisor 需要於申請截止前送出申請。
      4. 如果獲得獎學金，需要於會後繳交心得報告（參考 REPORTING REQUIREMENTS）。
