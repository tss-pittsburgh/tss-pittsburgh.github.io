    COVID-19疫苗接種 #

    最後更新日期2022.01.11

    目前全美不論有沒有保險，任何人都可以「免費」施打COVID-19疫苗。

    美國核准疫苗 #

    目前美國FDA緊急授權的疫苗：

    • Pfizer（輝瑞，2劑，每劑間隔至少21天，12歲以上）
    • Moderna（莫德納，2劑，每劑間隔至少28天，18歲以上)
    • Janssen（嬌生，1劑，18歲以上）

    施打場所 #

    如何預約：線上預約為主，且方便。電話預約亦可。少數提供Walk-in。施打地點可能是醫院、藥局、體育中心或是賣場等。預約時就會告知這個地點會提供什麼疫苗牌子，基本上只要有提供都可以自由選擇。

    • PittCoVax：可以Walk-in，不限Pitt學生。
    • CDC VaccineFinder
    • Allegheny County Vaccine Registration
    • Rite Aid
    • Giant Eagle Pharmancy（可以Walk-in）
    • CVS（可以Walk-in）
    • Costco

    所需文件：Photo ID（如護照或美國駕照）、保險卡（如果有保險就提供，沒有也非必要）、COVID-19疫苗卡（施打第二劑用）

    加強劑 #

    Pfizer或Moderna是美國主要推薦的COVID-19加強劑，可任意選擇，除了低於18歲的兒童仍僅能施打Pfizer。原先施打Pfizer或Moderna的人須間隔5個月後可施打加強劑，原本施打Janssen的人僅須間隔2個月。

    原圖摘錄自美國CDC官網

    已在美國外接種疫苗
    如果在美國境外完成了Pfizer、Moderna或Janssen疫苗的人，可應遵循上述指示獲取加強劑。
    如果在美國外施打WHO緊急授權完整疫苗或混合施打WHO緊急授權與FDA授權的疫苗，可以在施打所有推薦劑量後的6個月施打Pfizer或Moderna作為加強劑。

    加強劑劑量
    加強劑成分完全相同，但Moderna劑量為一半。
