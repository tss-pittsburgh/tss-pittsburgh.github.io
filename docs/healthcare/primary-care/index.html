    初級保健 #

    最後更新日期2022.01.11

    初級保健涵蓋常見疾病的預防、保健和治療的健康服務。初級保健提供者（Primary Care Provider, PCP）會與病患保持長期關係，並就一系列健康相關問題提供建議和治療。初級保健提供者包括醫生、註冊護士、執業護士和醫師助理，主要專攻家庭醫學或內科。

    主要業務 #

    PCP通過提供定期檢查、預防性保健檢查和疫苗接種來幫助病患保持健康。

    • 了解病患、病患的生活方式和家族病史
    • 血壓、血糖、膽固醇監測
    • 篩查（Screening）：例如子宮頸癌篩查、乳房X光檢查、結腸鏡
    • 抑鬱症評估
    • 疫苗接種
    • 補充處方籤
    • 提供健康建議和轉介健康教育、預防諮詢服務
    • 減重、戒菸、身體活動（Physical Activity）、營養、性病預防等

    接受篩查檢測的時間除了可能是例行檢查，其頻率取決於年齡、性別、健康狀況、風險因素和檢測費用。Pitt和CMU詳細的預防性護理可參閱UPMC 2022 Preventive Services Reference Guide和Highmark 2021 Preventive Schedule，一般保險需要涵蓋的內容可參照HealthCare.gov。

    一般來說，正常健康的成年人仍建議每年體檢一次，也就是所謂的年度健康訪問（Annual Wellness Visit）。

    第一次看診
    除了常規的體重、血壓量測，首次看診會詢問家族病史、抽血、補打疫苗。抽血結果沒太大問題的話，大約3~4年會再重新檢測。

    費用
    Pitt和CMU學生保險對於預防性護理在In-Network都是免費的。此外，Pitt、CMU學生保險對於PCP的首次與年度回診是不收取Copay。

    多問問題
    如果有什麼不清楚的健康問題或不清楚要找哪一個專科，都可以向PCP諮詢。

    地點 #

    CMU學生保險是Highmark，可透過3 easy ways to find a doctor網站尋找。

    Pitt學生保險是UPMC Health Plan。UPMC的初級保健辦公室涵蓋整個大匹茲堡，光Central區域就有25個醫院診所提供服務。大匹茲堡區域範圍涵蓋賓州西部和俄亥俄州東部。尋找PCP可以透過Find a Doctor or Provider，也可以參考 UPMC Primary Care Locations Guide 2020。

    別選到兒科
    除了家庭醫學科或內科，PCP搜尋結果是包含專長是兒科（Pediatrics）的醫生。特別注意別選到兒科作為PCP。

    建立聯繫 #

    PCP可以指名或由醫院/診所分配，通過電話或網站首次約診（會索取保險訊息）。

    • 指名：要先確認PCP是否收新病人。由於PCP為了確保舊的病人都能收到足夠的照顧，通常不會開放太多時間見新病人，因此第一次預約可能會要排一段時間。

    • 醫院/診所分配：如果想要盡快預約也不太挑人的話，可以直接打電話給較大型的醫院、診所要求幫忙分配一個，如果想要同性別醫生或其他要求可以在電話中一併說明。

    首次看診、建立聯繫後，以後看病或醫療諮詢就會比較順暢。以UPMC為例，可以透過MyUPMC messages直接傳訊息跟PCP聯繫。如果不滿意，亦可更換。

    常見問題 #

    家庭醫學和內科有什麼差異？

    兩人都是訓練有素的醫療專業人員，在識別和治療各種健康狀況的初級保健扮演重要角色。

    • 家庭醫學 — 家庭醫學醫生為所有年齡段的人提供護理.家庭醫生通常會接受許多醫學專科的培訓。
    • 內科 — 內科醫生通常為 18 歲及以上的成年患者提供不涉及手術的診斷與治療.內科醫生通常會有針對不同領域或年齡層的次專科。
