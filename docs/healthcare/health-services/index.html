    就醫指南 #

    最後更新日期2022.01.17

    美國不像台灣有如便利商店林立的醫療診所，一般看診都需要提前相當久的時間約診。雖然都有健康保險，但是如何獲得即時、有效、負擔較小的醫療服務仍是一大學問。

    就醫評估 #

    就醫前依序思考一下問題以選擇適當的醫療服務：

    1. 是否有醫療保險？

      留學生強制購買醫療保險。強烈建議了解自己的保險權益，詳情參考學生健康保險章節。以下分析均已Pitt或CMU學校保險作為基礎。

      眼睛疾病、不適
      眼睛疾病診斷、護理使用的是Medical Plan，而不是Vision Plan。

    2. 是否需要急診（Emergency Room）？

      與台灣不同，美國急診通常是有生命危險才會需要的醫療服務。在一般常識下合理預期不立即就醫會導致嚴重危害生命、身體機能嚴重損害、器官等功能損害。例如：事故造成的身體傷害（刀傷、槍傷、車禍），嚴重的急性症狀或劇烈疼痛、突然的醫療狀況（呼吸困難、胸部持續疼痛或壓力、嘴唇或臉發青）。如果是或不確定時，立即撥打911，接線人員會判斷是否需要提供救護車等醫療支援。

      急診不需要是In-Network
      當需要急救時，請立即前往/送往最近的醫院急診室。無論是否在In-Network或Out-of-Network的急診（Emergency Room Services），Pitt和CMU學校保險的Copayment均一樣，且Coverage均是100%。

      救護車
      大多數情況下，Pitt和CMU的學校保險對救護車都是100% Coverage，僅CMU保險在非緊急的Out-of-Network需要部分負擔。如果不確定是否屬於緊急狀況，撥打911可協助判斷。

    3. 是否需要緊急護理（Urgent Care）？

      緊急護理是指需要即時治療但通常沒有嚴重到需要急診的疾病或傷害。常見需要Urgent Care的情況可參考連結。

      出發前確認該醫療院所是否有營業。

      1. 學校保健中心對學生提供免費的Urgent Care，因此只要是學校保健中心的營業時間，可優先考慮預約處理。

      2. 如果學校保健中心沒開，則需要尋求鄰近的Urgent Care機構治療（例如UPMC Urgent Care Shadyside或MedExpress Urgent Care）。

      3. 一般Urgent Care營業時間為9:00am - 9:00pm，匹茲堡地區僅UPMC Urgent Care Pleasant Hills有較長的營業時間7:00am - 11:00pm。

      4. 如果在深夜仍需要緊急護理（例如嚴重割傷、刀傷），請直接前往急診。

      5. 如果人不在匹茲堡、不方便出門、找不到In-Network Urgent Care、非物理性擦、割傷，可以考慮In-Network Virtual Visits或Telemedicine Services。線上看診同樣可以提供處方簽，而且看診快速（通常排隊不需要超過半小時）、便宜。以Pitt的Graduate Student Plan為例，使用UPMC AnywhereCare是$0 Copayment，如何使用詳情參考UPMC AnywhereCare章節。

    1. 是否清楚自己需要看的科別？

      如果只是想定期身體檢查或有不確定的病痛，可以選擇初級保健提供者（Primary care provider, PCP）。PCP可以是主治醫生、住院醫生（Resident）、助理醫生（Physician assistant）或執業護理師（Nurse practitioner, NP），其專業包含家庭醫科或內科，擅長對各類疾病綜合診斷並評估是否需要轉診。此外，Primary Care的Copayment都低於專科。

      如果短期約不上PCP或聯絡不上自己的PCP，可撥打24/7 Nurse Line，全天候均可撥打，通常在保險下這服務是免費的。護理師會根據描述的病徵提供醫療建議或協助預約醫生。

      與PCP建立長期醫療關係
      可以透過電話或網站指定一個並建立長期醫療關係（可指定性別），如果不滿意也可以隨時更換。如果有突發的小病情，向長期配合的PCP預約優先度都較高，通常可當週或甚至隔天看診。特別是在一些大型醫療院所（例如UPMC）裡，即使長期配合的PCP近期無法看診，院所也能安排其他當值的醫生來看診。詳情參考初級保健章節。

      看專科不須透過PCP轉診
      Pitt或CMU的學生保險類型均是PPO，專科看診不要求透過PCP轉診。

    2. 是否附近有In-Network Provider？

      In-network Provider會比Out-of-network Provider便宜很多外，在Out-of-network看診通常要自行墊款並且向保險報銷較為麻煩。特別是Pitt的學校保險，離開賓州後大多數都在Out-of-network。如果仍有定期看病的需求，可以使用MyUPMC Video Visits約線上看診，詳情參考MyUPMC Video Visits章節。

    1. 確認附近的藥局

      每次看完診如果有需要領取處方藥時，醫療院所大多會詢問你要去哪裡領藥，他們會直接將處方簽傳至該藥局。可以選擇家附近CVS、Rite Aid或Pitt Pharmacy等。如果人不在匹茲堡，就選擇鄰近的藥局即可。

    2. 是否有疾病史、藥物過敏、偏好用藥、治療方式？

      第一次約診時，會被要求填寫確認是否一串疾病史、藥物過敏。通常用字較難，如果是紙本，可以用Google Translate App以相機翻譯。此外，如果總覺得台灣的皮膚科開的藥、之前在台灣接受過什麼治療比較有效，可以將資訊提供給醫療院所，醫生評估後可直接幫你進藥，藥價甚至有時候比台灣的自費藥還便宜。

    3. 是否需要翻譯？

      醫療院所通常會提供免費的翻譯服務。雖然當場提出要求翻譯服務即可，但在約診時就先提出需求可避免現場等待翻譯人員。以UPMC為例，是以多方電話進行翻譯，有些則是提供在場翻譯服務。如果覺得翻譯服務麻煩，通常醫療人員都很有耐心，可以事先查好字典講述症狀或是現場查詢溝通。

    學校保健中心 #

    對於日常小病，學校保健中心是看病的首選。學校保健中心提供比較基礎的醫療服務（感冒、經痛、基本皮肉傷等）、開立處方藥、體檢（如年度健康檢查、駕照體檢）、疫苗接種以及心理諮商。

    使用學校保健中心的服務是不用負擔Copayment，但在藥局領取的藥費仍需自行負擔。

    疫苗施打費用
    Pitt的UPMC保險任何疫苗均可免費施打。CMU的Highmark保險施打Affordable Care Act (ACA)要求的疫苗為免費，部分國外旅遊用的疫苗可能需要自費。

    Pitt學校保健中心位於Nordenberg Hall二樓的Wellness Center；同棟大樓的一樓有學校藥局，可以直接購買成藥與處方藥，較一般CVS、Rite Aid便宜。

    由於疫情因素，目前僅提供電話預約412-383-1800，周一至周五9:00-17:00。


    CMU學校保健中心位於Forbes Ave上，學校正門的對面。

    由於疫情因素，目前僅提供電話預約412-268-2157，僅平日到17:00。 如果在保健中心旁停車場停車，記得跟櫃台索取停車證放置車上，以免被開罰。

    地址： 1060 Morewood Avenue Pittsburgh, PA 15213

    校外醫院 #

    在匹茲堡，UPMC是最好的校外醫院選擇。UPMC提供線上預約、線上病歷與檢測結果、線上看診以及直接傳送訊息給你的PCP諮詢、24/7 Nurse Line。UPMC資源詳情參考UPMC章節。
