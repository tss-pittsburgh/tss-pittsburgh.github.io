    租約 #

    林奕丞、劉又齊協編
    最後更新日期2020.08.03

    簽約時要特別注意看清條約款項。

    合約基本內容 #

    簽約對象：分為與房東/物業管理公司（以下簡稱房東）簽約或第三方轉租（sublease）若對方為房東/物業管理公司，應確認產權屬於對方；若對方為轉租，應確認對方原始租約。

    所有房客名單：確認自己與共同簽約者是否載明於合約上。除了受到保障外，未來在申請駕照等可作為居住證明（Proof of Address）。

    房租所包含的費用：確認合約載明租金、相關費用（例如申請費等）、是否涵蓋水電瓦斯（統稱Utility）以及電話、網路、哪些修繕費用由房東負擔等。此外，於租約期間房東不能調整相關費用。

    房租付款方式與日期：確認每月交房租日。付款方式除了常見的支票、eCheck、Money Order，如果可以使用信用卡的手續費。部分私人租屋可能會要求以Venmo等其他第三方支付。

    租約期限：應確認合約載明租約起訖日，租金交付期限，押金退還期限等。起租日通常以每月1號（月初）開始，全新租約大多為一年一簽，少部分公寓提供半年約（不建議簽超過1年以上）。如果想調整起租日，簽約前先洽詢房東可行性。此外，合約通常會載明若想續約的最後期限，一般須在期滿兩個月前（每年4、5月）書面或其他方式告知房東。

    押金（Security Deposit）：確認合約載明押金，一般押金不得超過二個月房租（加預繳第1個月最多3個月房租），並應確認押金用法與退還方法、扣款名目（allowable deductions）以及押金退還期限（一般來說房東必須於離開30天內退還）。為保障自身權益不被亂扣押金，最好於退房前請房東檢查並退還押金。

    居住環境權利及義務：是否與他人共用客廳/廚房等，客廳是否有其他租客等，修繕、積雪、垃圾處理等。

    租房保險（Renters Insurance）：為有強制或非強制的。許多公寓會在簽約時同時提供相關保險，也有可能要求租客自行購置保險。一般理賠範圍分成個人財產險，責任險與額外開銷險三大部分。分別為保障個人財產的遺失與損壞維修，個人意外引起採產損害與人身傷害以及當房屋因故無法居住造成額外開銷之保障。

    其他：是否可以轉租、 房客人數限制、違約賠償條件、退租手續、 房東在何種緊急情況下可以進入房屋、 非緊急情況須多久前告知、 解約責任等。

    屋況與設施 #

    屋況檢查：建議能視訊就視訊看房，照片往往跟現實有落差。入住時務必檢查房間各項設備狀況，以拍照/影片搭配紀錄房間之損壞情形，並與房東回報相關問題。檢查房屋的清潔及通風情況。

    家具：租約應載明提供家具與損壞維修賠償相關事宜。

    地板：美國多數房間鋪設地毯。若對塵蟎等過敏，可考慮屋內鋪設木質地板而非地毯之房子。

    爐子：美國多數提供電熱爐而非瓦斯爐。電熱爐僅對應平底鍋具，且火力掌控較瓦斯爐受限。若對此有要求應注意申租房子提供爐子類型。

    隔音：對於噪音敏感之同學應注意房間房內的隔音與室外隔音是否能夠接受。可於承租前詢問或是實際看房評估。一般隔音好的房子相對來說冬天保暖效果也較佳。

    抽油煙機：基於法律規定，美式抽油煙機是一般禁止把油煙抽到室外，也就是抽進的油煙經由過濾後又循環回室內，對於台灣人炒菜產生的大量油煙通常不足。熱衷於煮飯的人須斟酌廚房旁是有窗戶或擁有較好的抽油煙設備。

    裝飾：退租時大多會要求復原房子，因此有不可逆性的裝修大多須要經過房東核准（例如：漆油漆、牆上打釘子等）。

    水電瓦斯 #

    簽約時先確認：

    • 水電瓦斯三項費用有哪些包含於租金中。
    • 確認是否需要額外申請使用（新房還可能有沒接通問題）。
    • 哪些費用需繳交給房東或水電瓦斯公司。如果繳給房東的話，是否以租客名義承擔費用。
    • 冷暖氣費用是否各戶獨立計算。（部分公寓為統一控管）
    • 詢問平均費用（詢問房客較佳），但使用習慣差異仍會反應在費用多寡上。

    自行聯絡申請：

    • 公司：匹茲堡地區的電力公司僅有Duquesne Light 一家, 瓦斯常見有 Peoples Gas、Dominion Gas 或 Equitable Gas 等。各公司的網站電話等連絡資訊可自行上網搜尋。
    • 申請流程：各公司有提供網路申請方式，惟建議直接先以電話聯絡加速流程，並依照相關指示提交文件完成後續步驟。
    • 申請文件：一般須填好公司所提供的申請文件，包含個人資訊，電話與地址等。申請時可能會要求提供SSN，若無則需附上護照等證明文件。
    • 押金：若無SSN，公司可能會要求付押金（例如2018年電力服務押金為$165）。
    • 繳費：多半提供線上繳費服務，利用帳單上的網址即可繳費。Duquesne Light可以從此頁面申請帳號並進行繳交。

    網路/有線電視 #

    少數房屋提供網路與有線電視（TV Cable），大多都必須自己重新申請。匹茲堡地區主要供應商為Xfinity（Comcast）與Verizon二家，然而並非所有房屋都可以申裝所有電信商的服務，須先向房東以及電信商確認自己所承租的房屋是否可以申辦指定電信商的服務。

    一般來說網路與有線電視服務通常都是綁在一起的，分開申裝網路及有線電視會比共同申裝方案來得昂貴。可依自己需求及方案價位申裝純網路方案或是網路+有線電視方案。基礎方案（60Mbps、不限流量）約在每月$40~$50左右。搜尋離自己最近的店家，備妥申請相關身份證明文件，臨櫃即可辦理。

    申裝時務必確認店員登錄資料無誤。店員輸入錯誤可能往後要耗費近1小時透過線上客服更正才得以正常使用。

    空調 #

    冷氣：冷氣部分通常可由房客自行控制，主要差別在於為中央空調或各房獨立。不管是哪種，房客通常都須依據使用量負擔費用。有些需要自己自行購買窗型冷氣。

    獨立供暖：每個房間、每戶一台，靠電或瓦斯驅動，通常可以自己調整溫度及何時開關，但也常常代表要使用者付費（房租不包含）。

    中央供暖：公寓會統一暖氣系統打開，整棟建築的每間房間可能靠中央空調送出的熱風，通常沒辦法關掉。雖然牆上通常有調溫度或暖氣強度的功能，但調了不代表有改變。

    衣物清潔 #

    公寓：如果沒有戶內獨立的洗衣機及烘衣機，則通常會有洗衣房提供共用設備（類似大學宿舍內的公用投幣洗衣機），具備基本水溫、烘衣溫度、洗/烘衣行程時長調控。洗衣服的價位每次$1~$1.5不等。如果有洗衣房，請注意使用時間跟機器的數量，如某些公寓規定半夜不能洗衣服，而有些地方洗衣機太少往往需要等待。

    非公寓：如果沒有提供洗/烘衣機，甚至可能得到附近的「洗衣店」花錢洗烘衣服，相對不方便。

    晾衣服：美國普遍使用烘衣機，雖然很多高級公寓有陽台，也很少看到有人在晾衣服，更不用說室外的晾衣場是非常少見。如果不想使用烘衣機，由於氣候乾燥、冬天暖氣，就算是厚牛仔褲大多一天以內就會乾，T-shirt可能只需數小時。女性的貼身衣物大多晾在浴室或衣櫃或房間的某個角落，因為氣候乾燥，通常不會太久就乾了。就算晾在浴室，由於浴室在洗澡後的濕氣往往不用一小時就完全消失，尤其有窗/通風設施的浴室，所以幾乎不用擔心不會乾。

    寵物 #

    大部分公寓對於養寵物均有不同的限制。有些是根據寵物的重量作限制；有些是規定可以養貓不可以養狗；有的是養寵物要額外的押金。若有養寵物需求，應於承租前詢問相關規定。

    停車 #

    如果已經是車主或確定近期內會買車，在找房子時須注意停車的相關事項。如果不確定是否會買車，建議找房子時在這方面多問清楚並做一些考慮，留一些彈性空間。

    On-street：也就是路邊停車，沒有固定車位，好不好停則要看附近環境而定。如果建築物本身沒有停車場，則可以到線上申辦Residential Parking Permit（連結，$20/年），屬於劃區停車，辦理需要居住證明、駕照、行照。部分路段路邊車位有計費器，停車就要照規定付錢並且限時（逃繳被抓到罰很重！）。 此種車位可以下載Go Mobile PGH App以手機進行繳費與延長等操作。

    Off-street：按提供者分為公寓及專門出租公司。住在擁有車位的公寓有時允許申請市政府的parking permit，萬一公寓本身車位需要排隊等很久，可找尋住處附近對外出租的車位（其他公寓也可能對外開放）。以台灣學生聚居的三個區域而言，一般來說，Squirrel Hill的公寓有附屬的車位的比例稍低，Shadyside與North Oakland的公寓有車位的比例稍高。按位置分為是室外及室內的車庫，室內車位對於冬天使用相對友善。在North Oakland或Shadyside，室內車位的月租金多在$40~$100之間，室外的車位多在$20~$80。按屬性又分為先搶先贏或是個人專屬車位。租個人專屬車位時，通常會和房間分開計算租金，至於是否接受殺價則看機運與造化。

    剷雪責任分配
    如果是公寓提供的車位，通常房東會負責下雪天的車道剷雪工作。住House則更要先問清楚車道及社區內道路（如果有）的剷雪事宜，以免冬天出入不便。

    反歧視政策 #

    房東不得因為下列原因歧視房客或申請人：種族、膚色、宗教、原國籍（包括有限的英語能力）、年齡、性別表述、性別認同、性別及性取向、懷孕、出生地、殘疾、政治傾向或信仰。例如，房東以沒有SSN的國際學生必須要付兩倍的押金，這有可能構成針對原國籍的歧視。如果遇到房東惡意歧視形，請聯絡匹茲堡市的 Commission on Human Relations，可以透過電話或本人到辦公室申訴，若案件成立，市政府會提供免費律師代表。
