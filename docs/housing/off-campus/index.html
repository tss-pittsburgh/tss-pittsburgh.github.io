    校外租屋 #

    林青億協編
    最後更新日期2021.05.24

    每個人對於房屋的喜好不同，推薦「好」房子是極為困難的事情，列出自己的喜好（例如預算、距離、採光、地板等等）會比較容易尋找適合的房子。

    校外租屋考量因素 #

    價格 #

    基本上就考量預算與性價比。需不需要希望生活品質換取較低的租金，例如出租客廳、住遠一點。評估住宿費用占學費、生活花費的比例。

    地理位置 #

    由於學生需要到學校上課，大多台灣學生校外租屋大多落於North Oakland、Shadyside、Squirrel Hill，入住情報較多也比較容易與大家有互動。如果對於時間較不敏感的人，特別是疫情期間大多人在家上課、上班，選擇住South Side或是East Liberty會有不錯的性價比，但入住情報較少。如果沒有交通工具，則需要考慮附近是不是有公車站牌與超市。

    對於剛來的人來說，買車並不是急需。匹茲堡的公車網絡發達，加上Pitt與CMU的學生證、工作證可以免費搭乘公車，學校、超市、藥妝店大多快速方便抵達。另外，學期間Pitt、CMU白天有提供校車、CMU晚上提供Escort校車（行駛時間6:30 pm~6:30 am）。詳情可參照匹茲堡交通。

    Pitt與CMU鄰近常見公車路線與超市及藥妝店分佈，百合超市與Sambok Korean Groceries位於沿54、86公車路線上的左側地圖外區域。

    Apartment/House #

    Apartment與House的優缺點比較可參照連結。

    室友 #

    在合租前，問清楚彼此的生活習慣與要求，從房租與Utility比例分攤、合買東西、清潔分配、可不可帶訪客、異性等。

    環境清潔和金錢經常是紛爭源頭，千萬不要一時的佔小便宜而傷了室友和氣。如果對於房租與Utility比例分攤不熟悉，可以多向他人詢問，積極與未來室友討論達成共識。

    物業/房東 #

    房子狀況難免有些問題，好的房東是能即時回應、盡速維修等，在入住前有沒有清潔、驅蟲等。這方面主要需要靠口耳相傳，需要多多問已經來匹茲堡的人。

    如果你是承接他人租約，特別是公寓承接，物業管理可能就沒有入住前清潔的責任。

    看屋與徵室友 #

    通常4月多開始會有人陸續釋出空位，5月多通常是房源最多的時候，過了5月房源會漸漸減少。雖然現場看房子比較有踏實感，但也是需要考量房源的多寡。找室友可以利用PTT、TSS Facebook社團、中國留學生組織CSSA或加入租屋微信群。

    • 能視訊就視訊看房，照片往往跟現實有落差。
    • 多詢問其他人該區域或建築的風評。

    租屋查詢 #

    物業管理公司通常有專屬的網站，提供數間公寓出租；私人租屋則通常是到租屋網站或微信群。

    常見問題 #

    客廳是否值得出租或入住？

    出租方面
    過去許多人會將2b2b的客廳出租以換取較低的租金。然而根據統計，2b1b與2b2b有巨大的價差，比起將客廳出租以壓低價錢2b2b，無論在價格或能夠客廳使用上，2b1b都具有很好的競爭力，值得考慮。

    入住方面
    客廳的價錢約$500~$600。通常客廳僅能用布幕或辦公室檔板作為空間隔離，除了隱私不佳外，室友們廚房油煙是難以避免的。相對於，studio僅有自己在使用廚房較好控制。此外，室友進門或訪客的交談都會有一定程度的干擾。因此，一個月多花$100~$250（約一年學費的5~10%）自己租studio或共享2b1b，是可以大幅提升生活品質的。此外，亦可考慮合租House，雅房人均租金約是$550~$700，也是一相對性價比高的選擇。
