    租房保險 #

    最後更新日期2021.08.24

    租房保險（Renters Insurance）是保護租客的個人財產和責任的保險。雖然法律並沒有強制規定租客要購買租房保險，但如果房東或物業管理公司要求購買租房保險，依然是合法的。簽約前確認租屋協議。

    為何需要租房保險 #

    • 房東的房東保險（Landlord Insurance）一般僅保障房屋的整體結構、公共設施及場所，並不涵蓋火災、天災、竊盜等因素造成租客的財產損失。

    • 如果是租客造成的損害，例如廚房失火造成的廚房設備損壞或冬天誤關暖氣造成水管爆裂、甚至造成其他租客的財產損失，租客可能都要自行負擔。即便房東保險可承擔建築物的損害賠償，房東可能為其支付的自付額、保額增加向租客索賠。

    • 避免糾紛。不論突發意外、他人或自身過失，只要有租房保險都不用為了財物損失而煩惱，也不用花大費經歷捍衛自己的責任或損失。

    • 因契約要求。一般租約上面通常都會載明購買租客保險。有些房東希望看到保險單或保險聲明的副本；有些房東口頭告知即可。即便合約有載明購買義務，但許多房東並不在乎租客有沒有沒購買保險，畢竟意外發生大多是租客自身損失。

      附加利益人
      有些房東要求租客將其列為保單的附加利益人（Additional Interest）。如果保單失效或被取消，保險公司會馬上通知附加利益人（即房東）。惟須注意，「附加利益人」不同於「附加被保險人」（Additional Insured），後者是針對與租客一起居住的室友、家人或其他希望被保單覆蓋的人。千萬不可將房東列為附加被保險人，否則將喪失對房東求償的權利。

    保險範圍 #

    標準的租客保險涵蓋以下類別：

    • 個人財產（Personal Property）：泛指租客自身的個人物品，包括家具、衣物以及電器等。惟須注意，部分貴重的個人財產可能是獨立選項，包含手機、電子產品、樂器、運動器材、珠寶、手錶等。

      隨身物品
      除了屋內的東西，隨身帶的東西依照合約也可能是受保的。例如：隨身攜帶的筆電被偷，可以申請保險理賠。

    • 責任險（Liability）：由於租客過失或其租屋範圍內造成的他人（房東、朋友、鄰居、路人等）的經濟損失，包含財產和醫療賠償等。例如：房內水管堵住造成他們房屋淹水；住在House冬天忘記剷雪造成路人滑倒。

    • 使用損失（Loss of Use）：如果因為意外後住處暫時無法居住，住飯店的開銷將由保險公司理賠。例如：變電箱、暖氣損壞暫時無法居住。

    • 醫療支出（Medical Expense）：他人在租客的住處受傷所需要的醫療開銷，不限於租客過失。例如：朋友來家幫忙切菜切到手去急診。

    不理賠情況 #

    大多數房屋保險都會將以下情況排除在理賠範圍以外：

    • 洪水
    • 地震
    • 坍方
    • 發霉
    • 除了水管或者家電導致的情況以外的水害
    • 蟲害鼠害等
    • 地基變形沉降等
    • 寵物動物
    • 由寵物造成的破壞
    • 磨損、損耗
    • 故意丟失
    • 核事故
    • 政府行動

    匹茲堡夏天偶爾會有暴雨情形，特別是如果住的House處於易淹水地區，可考慮額外購買洪水保險（Flood Insurance）。

    保險費用 #

    以下列AllState 2021保單為例，年費為$184（約$15/月），購買租客保險並不會造成太大負擔。

    如何選購租客保險 #

    選購租客保險時，可參考以下幾點：

    • 根據自己擁有的個人物品價值選擇Personal Property保額。沒有樂器、運動器材、珠寶、手錶就不需要購買此項目。家徒四壁就可以選擇保額低點的。

    • 一般Liability額度會在$100K~$300K之間。

    • 選擇合理的免賠額（Deductible）。越低的免賠額，保費越高 反之亦然。

    • 審慎思考是否要跟室友共買租客保險。雖然保費上可以省錢，除了分帳外，其中一人提出索賠之後也會出現在你的保險紀錄持續3年。

    • 如果有汽車，與汽車保險綑綁購買有相當的折扣。

    • 貨比三家。到Allstate、Amica、AAA、Lemonade等大型保險公司。
