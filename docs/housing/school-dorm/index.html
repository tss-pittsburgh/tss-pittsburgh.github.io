    學校宿舍 #

    最後更新日期2021.05.24

    Pitt校園宿舍 #

    Pitt校園宿舍（On-campus）通常僅提供大學生住宿、提供基本床具、書桌並包含餐點購買計畫，費用納入學費中（費用已含Utilities）。大一強制住宿，大二以上基本上都需要抽籤排隊（一人排到，同隊室友可以一起入住）。部分樓只提供給大一住宿，部分樓只收大二以上。大部分樓都是多人一間（沒有獨立房間）。強制購買Dining Pass＋Dining Dollars，大一限定購買至少Tier 2以上的（2020年價格為$2,050/學期）。建議選最少次的Dining Pass即可。以下僅列舉四間校園宿舍。

    Litchfield Tower #


    指定樓層限大一住宿，其他樓層提供給高年級。位於主校區的中心，總共有三棟。每間房間配有微波爐、冰箱；每層樓共用1~2個廁所、沖澡區，每天會有專人清潔；每棟共用一個洗衣房。一樓是學生中心（Panther Central），負責辦理學生證、學生住宿事宜，24小時營運。地下一樓是學生餐廳（Market Central）。

    Holland #


    限大一住宿。位於主校區的中心，Litchfield Tower隔壁。每間房間配有微波爐、冰箱；頂樓有Lounge/Study Area。一樓是University Store，可購買書、文具、日用品與紀念品。

    Nordenburg #


    位於主校區的中心，Holland正對面。每間房間配有微波爐、冰箱、電視。每一層共用洗衣房。一樓有PNC、AT&T以及Pitt藥局。

    Irvis Hall #


    限高年級住宿。位於校園上層，Petersen Events Center附近。每間有獨立浴室的雙人間。

    Pitt學校公寓 #

    Pitt學校公寓（apartment-style）為大學擁有公寓，大學生、研究生皆可入住，均不帶家具，也不用購買Dining Pass，租金採月收。

    • College Gardens（大學生、研究生、教職員）
    • Darragh Street（研究生）
    • Hyacinth Place（大學生、研究生）
    • Mayflower Apartments（研究生）

    CMU校園宿舍 #

    CMU校園宿舍包含家具，費用納入學費中（費用已含Utilities）。大一強制住宿，如果需要指定特定室友，須提供他人的Andrew ID。大二以上、研究所以上需要排隊。大一強制購買Traditional Dining Plan。

    CMU未提供研究生用的學校公寓，僅提供Off-Campus Housing網站查詢校外租屋。
