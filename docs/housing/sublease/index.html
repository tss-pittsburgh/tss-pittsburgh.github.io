    轉租 #

    最後更新日期2021.05.24

    轉租是有許多互惠的優勢，但也要注意有許多風險。

    雖然承接他人推薦的房子是相當便利的事情，但仍須注意這些轉租人所提供的推薦是不是符合現實。由於越多人住的地方轉租消息就會越多，這些正面評價數目並不代表這地方更好，建議仍多多找房與比較。除了轉租人的評價外，參考不住在該建物人的評價亦是一種選擇。

    轉租類型 #

    • 承接他人剩下租約。這種租約需要確認屋況、押金轉移等。

    • 不承接他人租約，但承接他人房間。由於不需要向物業、房東退房，可能需要確認房間清潔、屋況權責。

    簽約模式 #

    • 房東簽約：建議選項，通常會有額外手續費，但雙方較有保障。轉租人可以拿走押金；承租人不用擔心合約問題。

    • 私下協議：務必詳閱轉租人原始合約，確認租約原始期限、是否允許轉租，同時釐清租金付款方式、押金轉移、轉租人/承租人毀約賠償事宜、租金補貼（如果轉租人降價出租）等事宜。

    轉租注意事項 #

    • 確認客廳是否有住人、確認室友組成、生活習慣（例如異性同戶、常有聚會是否介意）。

    • 談妥承接家具事宜。注意時價、物品狀態，避免當冤大頭。
