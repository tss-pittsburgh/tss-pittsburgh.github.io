    旅遊平安險/留學險 #

    最後更新日期2021.05.24

    學校大多是從8~9月開始生效。如果早於這時間前往美國，會有醫療保險的空窗期。如果剛好遇上水土不服，可能就需要花費大筆醫療支出。建議離開台灣前可先選購短期的旅遊平安險或留學險（均須在出國前購買）。由於美國的醫療費昂貴，所以記得買醫療給付較高類型的保險。
