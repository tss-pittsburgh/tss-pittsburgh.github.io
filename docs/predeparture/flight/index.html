    機票 #

    最後更新日期2021.06.04

    機票的購買是一門大學問，本篇主要討論從台灣出發。機票價格跟許多因素有關，沒有所謂的最佳解。除了轉機次數，轉機時間、是否為聯程機票、轉機是否須換機場、是否想於轉機點中停都需要注意以及到達匹茲堡的時間。暑期經濟艙單程票價介於$850~$1,400。

    機票查詢平台 #

    如果不想要直接透過旅行社處理，可以藉由搜尋平台機票比價。常見搜尋平台：

    • Google Flight
    • Skyscanner
    • KAYAK
    • Studentuniverse
    • Expedia

    機票組合 #

    聯程機票 #

    多航段機票由單一航空公司開票（每個航段執飛的航空公司可不同）。途中如果行李轉運、航班誤點等，航空公司會負責處理，確保旅客能到達終點。

    聯程機票的優缺點
    ✅ 航段間提供協助轉機、托運行李免費直掛、國內線托運行李免額外付費。
    ❌ 每個航段都要搭乘，不能放棄起始或途中的航班。例如，台北飛美國在日本轉機，不能放棄台灣起飛直接跑去日本搭飛機。

    相對於非聯程機票是航段間是獨立出售過程中轉機、行李轉接大多都是要自己負責，可能需要領了行李後再重新check-in。美國國內線托運行李要額外支付費用，兩件共約$70~$80）。旅行社提供機票可能是將非聯程機票套裝出售，須特別注意不同航班的行李規定。

    轉機時間 #

    抵達美國的國際線至國內轉機或加拿大前往美國轉機前，將會需要通過移民官、領出託運行李並再通過海關入境美國，因此保留最少3小時的轉機時間為佳。

    美國境內轉機通常登機口相當靠近。此外，如果是購買聯程機票，航空公司會盡力確保你轉機成功，如果因為班機延誤或轉機時間較短，可跟空服員告知轉機需求，通常會安排你降落後優先離開座位。通常保留40~60分鐘就相當足夠了。如果是非聯程機票，通常如果轉機失敗，可能就要自行跟下班航空公司交涉。

    中停 #

    在查詢聯程機票時，一般預設總航程時間會追求最小化，因此每一次轉機時間會大多會在24小時內完成。在有些情況下（例如想遊玩城市、拜訪親友），乘客不想要馬上轉機離開該城市，中停（Stopover）的目的即是使轉機時間超過24小時甚至數日。通常轉機超過12小時候，就會需要將托運行李取出並於等待下個航段開始前重新check-in。期間可以離開機場並在城市過夜。

    想要轉機中停某個城市，除了直接分開買機票外，也可以使用Multi-city搜尋功能搜尋聯程機票。可先使用One way查詢大致了解可行航段，在Multi-city模式下設定中停城市抵達與再次起飛的時間（通常中停城市必須相同）。有中停的票價不一定會更貴。如果不介意拖著行李移動，是個值得考慮的選擇。

    航程中停的優點
    ✅ 從容不迫地轉機、換機場。
    ✅ 拜訪親友。
    ✅ 開學前遊玩另一個城市。
    ✅ 在大城市開花旗銀行帳戶，詳情參照花旗全球速匯章節。
    ✅ 至駐紐約辦事處紐約現場辦駕照譯本，省去準備money order、寄送資料，詳情參照免試申換賓州駕照章節。
    ❌ 需要托著許多行李跑來跑去。

    使用中停航班範例：
    聯程機票，美國航空官網票價：25,555 NTD（~$870）
    （JFK, LGA都是鄰近紐約機場，算同城市）

    • 2018/07/24 TPE→NRT→JFK
    • 2018/07/31 LGA→PIT

    台美航線航空公司 #

    台灣與匹茲堡國際機場（PIT）之間沒有直飛航班，所以至少要轉1~2次機。選擇亞洲航空公司的班機大多會在晚上離開亞洲，大清早到匹茲堡，美國或加拿大的公司則多是早上離開晚上到。1次轉機航線較少且昂貴，由於匹茲堡沒有直飛亞洲航班，轉機點將位於美國境內或加拿大，飛行時間約20~28小時。

    直飛美加的常見航空公司與航點：

    • 華航：紐約甘迺迪JFK、洛杉磯LAX
    • 長榮：紐約甘迺迪JFK、洛杉磯LAX、芝加哥ORD、西雅圖SEA、休士頓IAH、多倫多YYZ
    • 外籍航空：舊金山SFO（聯合）、多倫多YYZ（加航）

    美國境內航空公司 #

    如果購買非聯程機票，會需要單獨購買美國境內航班，通常是需要單獨購買行李。

    • 美國航空（AA）：航班多。
    • 達美航空（Delta）：航班多。
    • 聯合航空（UA）：航班多，不要購買Basic Economy，該艙等在美國境內航班只允許攜帶小型的隨身個人行李（能塞到座位下的）。
    • 阿拉斯加航空（Alaska）：美國相對服務品質比較好的航空公司。
    • 捷藍航空（Jetblue）：美國相對服務品質比較好的航空公司。
    • 西南航空（Southwest）：全部航班均有免費兩件行李。全機不選座位，起飛前24小時check-in決定登機順序，座位先強先贏。無法從搜票平台搜到機票。
    • Spirit、Frontier：屬於最廉價機票，但經常有隱形費用，服務品質較差。

    24小時退票政策
    任何前往美國或美國境內的航班，只要出發前7天以外購買，都可以在購買的24小時內免費退票。

    建議官網購買
    雖然航空公司官網的價格通常不是最便宜的，但未來如果發生改時段、停飛等因素，可以直接找航空公司尋求幫助，不需要透過旅行社更改而被收取額外的手續費。除非旅行社自創的套裝航段，一般搜尋結果會有航空公司官網或其他旅行社選項。

    行李限制/限重 #

    大部分航空公司的免費托運行李經濟艙為兩件各23 kg （50 lb）。由於行李限重的規定隨時可能變動，請務必一定要到航空公司網站或打電話詢問確認。托運行李超重看運氣是否罰錢。隨身行李的規定多半是一件隨身行李加上一件個人行李。

    出發日期 #

    領取F學生簽證的新生只能在I-20表格上所列的課程報到日（Program Start Date）之前30天內才會被准許入境美國，早一天都不行。在規定期間前提早赴美的簽證持有人在可能會在入境關口被拒絕入境。該規定不適用於繼續就讀的學生。疫情期間，國際學生最早以及最晚可以何時抵達，可以參考Pitt FAQ。

    邊境入境
    可以提前以eTA入境加拿大，並於Program Start Date的前30天內從邊境入境美國。

    無法課程開始日期前抵達
    若無法在課程開始日期前到達美國，請要求您的學校在SEVIS系統裡更改該日期。該規定不適用於繼續就讀的學生。

    日期差異
    Program Start Date和Start of Classes日期並不相同，不要看錯了。

    提前入境
    有人會以ESTA方式早於30天入境，並之後再出入境美墨邊境開通學生身分，但具有相當風險。

    抵達時間 #

    抵達時間需要考慮交通、入住手續以及辦事方便性。

    • 交通：匹茲堡機場接駁公車（28X）只行駛至晚上12點，錯過這個時段則需要叫Uber / Lyft。如果需要他人接機，時段也是相當重要，詳情參考匹茲堡交通。
    • 入住手續：部分公寓只接受特定時段入住、看房。
    • 辦事：週末銀行/手機店沒開或營業時間較短，要拖更久才能拿到手機號碼。
