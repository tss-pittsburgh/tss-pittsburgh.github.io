    主編的話 #


    歡迎加入匹茲堡這個大家庭，一起在美國追求自己的夢想！無論是否是第一次赴美或是從其他地方搬來的學子，新生活總是令人忐忑不安。匹茲堡生存手冊希望提供最全面與即時的資訊，從簽證辦理、找房到金融小知識儲備，幫助在匹茲堡地區甚至其他地區的台灣學人適應全新的美國生活。手冊的每個章節都它的背後故事，來自於我們曾受到的大小幫助，最後將這些感恩回饋成文字。這些經驗將帶你繞過許多彎路與掙扎，與你攜手克服恐懼與迷茫，陪伴你生活在匹茲堡。如果你喜歡上手冊的熱情與精神，期待未來的你能將你遇到的挑戰與歡樂，分享給未來的每一個匹村學子。

    匹茲堡生存手冊囊括通用的美國行前準備與生活技巧以及匹茲堡當地特色資訊，其中參考了許多公開資訊、私人經驗以及其他學人的著作，如PITT CSSA的2016版新生生存手冊與UCSD TGSA新生手冊等。部分資訊可能隨時間推移而不合時宜，特別是疫情期間，許多措施和政策變化迅速。如果有錯誤、建議或需要補充的地方，歡迎利用選單左下方的「意見回饋」告訴我們你的想法，我們將會持續更新。如果你有想要撰寫的主題與經驗分享，也歡迎透過意見回饋聯絡我們。
