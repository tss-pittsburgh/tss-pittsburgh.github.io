    活動 #

    最後更新日期2021.05.24

    疫情下的活動會與歷年有所差異，敬請期待新的一年活動規劃！

    新生說明會（台灣） #

    每年5月至6月期間，在台灣的學長姐會在台灣舉辦新生說明會。在新生開始在踏上匹茲堡的冒險之前，我們將介紹來美國前的準備事項以及生活資訊。

    2019年新生說明會@Taipei

    迎新（匹茲堡） #

    每年在8月底至9月初，在匹茲堡校園舉辦迎新活動，我們將提供一些有用的資訊，舉凡交通、超市以及社團活動，這也是個認識彼此的絕佳機會。

    2018年迎新@Pitt

    中秋烤肉 #

    中秋節是團員的日子。每年的10月份，我們會在CMU校園辦烤肉活動。不僅可以享受美食，並且可以和來自不同領域、學校的朋友一起活動。

    2019年中秋烤肉@CMU

    農曆新年晚會 #

    一年將過去了，忙碌的學業生活多少會有些思鄉情緒。每年的新年附近，我們會舉辦一個盛大聚會，一起享用美食與表演活動。通常會有上百人參與活動。

    2020年農曆新年晚會@蜀香園
