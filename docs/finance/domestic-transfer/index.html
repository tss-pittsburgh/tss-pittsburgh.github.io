    美國境內轉帳 #

    最後更新日期2022.03.30

    美國境內轉帳系統與台灣相差甚大，具有不同的轉帳方式，到帳時間、手續費、轉帳金額均有不同的規定。

    ACH Transfer #

    ACH是美國一個低成本的轉帳系統，Transfer在銀行網頁或手機App上被簡稱為Transfer或Money Transfer。這類授權轉帳都會需要提供Routing Number和Account Number。ACH主要有四個用途：

    • 同一個人不同銀行帳戶的資金轉移。
    • 部分銀行支持同一間銀行不同人之間轉帳。
    • C2P：如薪資發放、退稅等。
    • P2C：如學費、房租繳納等。

    優點：通常是免費的，適合用於同一人帳戶之間的資金調動。
    缺點：轉帳通常需要2~3個工作日。轉出金額有每日或每月上限。

    Direct Deposit
    許多銀⾏會要求設定Direct Deposit來滿足開⼾獎勵或免除管理費，通常須透過公司透過ACH發放薪資滿足。

    Routing Number (ACH) #

    由9位數代碼構成，會根據開⼾分⾏的地區有所不同，銀⾏App/網⾴上可能標示為：

    • Routing Number (Chase)
    • Paper & Electronic Routing Number (BoA)
    • ACH Routing Number
    • Direct Deposit

    可以在App、線上或支票左下角找到該號碼。

    使用ACH資金轉移 #

    第一次個人轉帳前須要認證帳戶是自己並綁定，類似台灣的約定帳戶。帳戶綁定方式有兩種，均可在網路上完成：

    • 直接登入該銀行帳戶。
    • 銀行會存些暫時性金額，需到綁定帳戶確認後填寫出該數字。

    ACH綁定是單向的，在A銀行綁定B銀行後，可以從A銀行推錢（ACH Push）至B銀行或從A銀行發起從B銀行拉錢（ACH Pull）。然而，並不能在B銀行對A銀行發起推錢或拉錢，須要在B銀行綁定A銀行才能進行。

    Wire Transfer #

    類似於台灣的銀行轉帳，通常用於轉帳金額較大或時間急迫。需要收款人的姓名、分行地址、Routing Number和Account Number。

    優點：當天到帳、可跨行、接受較大額轉帳。
    缺點：匯款方與收款方都有手續費（$15~$20）。

    Routing Number (Wires) #

    由9位數代碼構成，通常代表銀行機構，因此不管是國內/國際電匯或哪個分行開戶，電匯的Routing number通常都相同。在銀⾏App/網⾴上可能標示為 Routing Number for Wire Transfers、 Wires、 Domestic Routing Number等。

    常見的Wire Transfer Routing Number：

    • Chase: 021000021
    • BoA: 026009593
    • PNC: 043000096
    注意Routing Number
    ACH Transfer 和 Wire Transfer所需要的Routing number可能不⼀樣。如果有不確定請務必向銀⾏或⾦融機構查證。
    Routing Number歷史
    Routing Number⼜稱Routine Transit Number (RTN)，由美國銀⾏家協會（American Bankers Association, ABA）分發的9位數代碼。從早期的紙本支票交易到近期的電子支付（例如Fedwire、ACH），均是採用ABA Routing Number作為金融機構的辨識。

    Zelle #

    Zelle是銀行間的即時轉帳系統，只須知道收款人的手機號碼或Email即可以轉帳。如果開戶銀行已經是Zelle的合作夥伴（大銀行通常都是），銀行App通常已經內建Zelle。

    優點：即時到帳、可跨行、無手續費、操作方便。
    缺點：每日或每月轉出金額上限（例如：PNC $1,000/日、$5,000/月；Chase $2,000/日、$16,000/月），且收款方需要有Zelle。目前只和美國本土銀行有合作。

    第三方轉帳 #

    付款是投過帳戶餘額或綁定的個人金融帳戶（支票帳戶、簽帳金融卡及信用卡），收款方則是即時存入系統帳戶裡，帳戶餘額需要自行轉入綁定的銀行帳戶。由於是即時轉帳，一旦提交是不可以撤回，交易平台並不會為付款方提供保護，因此需要注意匯款金額與對象。

    PayPal #

    老牌的第三方轉帳軟體，兼具網路購物付款渠道。轉帳方面，透過Email尋找轉錢對象，使用簽帳金融卡和信用卡轉帳均需要手續費。網路購物方面，買方的付款方式均無手續費，且具有保護信用卡信息的功能。經姓名認證後，單筆支付上限為$10,000，總額沒有限制。支援25種貨幣。

    Venmo #

    具社交元素的新興轉帳軟體，申辦僅須要姓名、Email、美國電話號碼即可。透過帳號或QR-code找到轉錢對象，亦可以申請好友（非必須）。除了信用卡轉帳是需要手續費外，其餘方式均為免費。每周有轉出金額上限，未驗證帳號為$299.99、驗證帳號為$2,999.99。僅支援美元。

    票據 #

    Personal Check、Money Order、Cashier’s Check是美國常見的票據，三者用途、簽發方式不同。

    Personal Check #

    由支票帳戶（Checking Account）的ACH Routing Number和Account Number所構成的付款票據。帳戶持有人填寫金額、簽字，即可用於付款。填寫方式可以參照網路或請教Banker。個人支票可向銀行、支票訂製公司或倉儲式大賣場（Costco、Walmart）購買，而部分銀行會免費提供高級帳戶免費的個人支票。兌現須要有銀行帳戶，除臨櫃外，可使用ATM或手機（有上限）。

    個人支票不等於現金，支票有跳票、撤銷、簽名不符等風險而無法兌現。因此不要接受來路不明的支票。

    Money Order #

    在台灣金融系統稱為「匯票」，屬於保證兌換的票據，不會有跳票的風險。票面包含收款人、付款人、雙方地址以及金額（上限為$1,000），上述資料皆是行員印製、非手寫，而且僅收款人可以兌換，是一種相當安全、不會被盜領的付款方式。可在郵局（USPS）、商店（Walmart、CVS）等地方購買，手續費約$1。收款人可以至銀行、郵局或其他有出售Mondey Order的地方兌現。


    購買Money Order注意事項

    • 僅可以Debit Card或現金購買，不接受信用卡支付。
    • 記得保留收執聯。如果付不小心丟失是可以依據收據聯退款的。
    • 雖然購買的Money Order地址是非必填項目，但部分支付對象（例如：美國政府機構）的Money Order需要完整填寫收款人和付款人地址。
    • 如果單日買超過$1,000的Money Order可能會被要求查看Photo ID。

    Cashier’s Check #

    在台灣金融系統稱為「銀行本票」，是由銀行簽發的保證兌換票據，不會有跳票的風險。票面包含簽發銀行及其行員簽名、收款人資訊，購買是經由個人銀行帳戶扣款，手續費約$10~$20。通常用於大額購買（例如：購車）。
