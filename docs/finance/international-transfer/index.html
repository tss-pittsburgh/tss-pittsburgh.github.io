    國際電匯 #

    最後更新日期2020.08.13

    繳學費、房租或未來繳貸款親自搬運大筆資金往來台美不太可行，勢必得靠電匯處理。

    台灣匯至美國 #

    台灣的匯款大多需要臨櫃匯款，匯款人不一定要在美國有銀行帳戶，只要任選一家有開設跨國匯款的銀行即可。因此如果是親人操作，建議親人自己有美金帳戶較為方便。轉帳所需要資料有（以下資料均為英文）：

    • 受款人姓名
    • 受款銀行全名
    • 受款銀行Swift Code
    • 受款銀行地址
    • Routing Number
    • Account Number

    轉帳的手續費可以分成三筆：

    • 由台灣的銀行收取
    • 由中轉銀行收取（不一定，依據美國的銀行與台灣的銀行沒有直接匯款渠道時）
    • 美國收款銀行

    轉帳時台灣的銀行僅預收自身銀行的手續費，但由於仍有中轉行和收款行的手續費，因此匯出去的金額會比實際收到的少。

    美國匯至台灣 #

    匯款流程大致與台灣匯款到美國相同，大多也需要親自臨櫃。因此未來離開美國想關閉銀行帳戶時，建議先行操作。

    花旗全球速匯 #

    台美兩邊均有花旗銀行帳戶，即時、長年無手續費、可以網路操作。匹茲堡無花旗銀行，須至鄰近的紐約、芝加哥或華盛頓特區開戶。

    台灣速匯至美國 #

    需要先綁定美國帳戶。第一次需要臨櫃台灣花旗綁定美國帳戶，第二次後可以書面綁定、變更美國帳戶。因此如果尚未抵達美國，可以考慮綁定親友的美國帳戶，等到美國開戶後，再以書面方式綁定自己的美國帳戶。

    美國速匯至台灣 #

    網路輸入台灣花旗銀行帳戶資訊即可以匯款。
