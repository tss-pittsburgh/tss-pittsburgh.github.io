    E-ZPass #

    最後更新日期2021.06.21


    E-ZPass是一種收費公路、橋樑和隧道的電子道路收費系統（ETC），類似台灣高速公路的收費模式，其覆蓋範圍為美國東部、中西部，南至佛羅里達州，西至伊利諾州。

    與台灣單一公司運營模式不同，E-ZPass目前是由19個州、40+個公路局或委員會組成「E-ZPass Interagency Group (IAG)」，每個代理處均有自己獨立的結算、客服中心與應營政策，其範圍包括可返還或無返還存款的加值、帳戶管理費、最低帳戶限額和每次加值額度。其中，匹茲堡地區主要的代理處為Pennsylvania Turnpike Commission (PA Turnpike)。

    選擇代理處
    IAG並不要求在居住所在地開戶，使用者可以在IAG下的任何一個代理處申請帳戶。這意味使用者可以依據裝置價格、帳戶管理費等因素自行選擇代理處。本文僅介紹PA Turnpike的申辦流程。

    E-ZPass優勢 #

    相對於低廉的帳戶管理費（$3），E-ZPass提供相當便利與高額折扣。因此，不論自己是否有車，只要有開車需求的人都推薦申辦E-ZPass。

    • 便利：過收費站不再需要使用人工付費，此外部分收費站僅收現金。
    • 折扣：通常E-ZPass會提供額外折扣，通常比人工收費便宜30%。
    • 免用租車服務：使用租車公司內建的服務，費用約為$5/日，且只要使用一次，整趟旅途每天都要計費。只要將租來的車牌登記至帳戶後，可以使用自己的E-ZPass。
    • E-ZPass限定：部分道路或疫情期間，很多收費道路不提供人工收費。如果沒有E-ZPass，則需要依照車牌郵寄收費。通常郵寄收費比E-Zpass收費貴上50%~100%。此外，如果是租車公司的車輛，可能會被額外收取服務費。

    帳戶申請 #

    帳戶申請可在官方網頁線上申請。除了個人資料外，需要填寫駕照、車牌、信用卡以及電子信箱。帳戶管理費$3/年。

    如果沒有車
    如果自己沒有車，車牌可以任意填寫，之後隨時可以更動。推薦填寫方式：First Name的前3個英文字母、生日4位數字。惟須注意，一個車牌僅能加入至一個帳戶。

    裝置購買 #


    E-ZPass裝置（Transponder）購買途徑主要有三種：

    • 線上購買：註冊帳戶時同時購買，預繳$35，免運費。
    • 實體店購買：地點包含AAA辦公室、高速公路上的便利商店、Giant Eagle等。裝置初始額度$38（其中$3用於帳戶管理費），但實體店販售價會高於$38。需要購買的72小時內將裝置註冊並開新帳戶。
    • 郵寄購買：不推薦。

    帳戶加值 #

    E-ZPass帳戶加值購買途徑主要有三種：

    • Credit, Debit or Check Card：綁定在帳戶中。除了手動加值外，當帳戶餘額少於$10或更少時，可自動加值$35以上。
    • 銀行帳戶：不推薦。
    • 支票或現金：不推薦。

    E-ZPass Plus #

    透過信用卡加值的用戶，可以開通E-ZPass Plus選項。用戶可以在JFK、LGA、EWR等紐約、紐澤西機場停車使用E-Zpass付費。
