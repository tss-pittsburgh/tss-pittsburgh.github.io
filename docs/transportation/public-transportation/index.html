    匹茲堡交通 #

    最後更新日期2021.05.29

    校車/接駁車 #

    Pitt Shuttle #


    Pitt校車主要在學期間提供往來分散的校園建築，通常不需要驗證Pitt學生證。路線與運營時間可使用網站，或是使用Rider System app或TransLoc app查詢。

    CMU Shuttle #


    由於匹茲堡公車系統對CMU相對不友善，CMU校車主要在學期間提供往來North Oakland、Shadyside與Squirrel Hill，有時候會需要驗證CMU學生證。路線與運營時間可使用網站，或是使用Rider System app或TransLoc app查詢。

    Safe Rider #

    Safe Rider是Pitt教職員工生的晚上接送服務，須撥打電話412－648－2255或使用TransLoc app預約。搭乘時需要出示Pitt ID，可攜帶2名客人。每晚可使用一次，一學期最多25次。接送地點符合以下任一條件即可：

    • Safe Rider的接送區域內，區域定義參照連結。
    • 校園建築間、住處到校園或校園到住處。
    • 不在公車路線或接駁車路線上。
    • 從Greyhound、Megabus或Amtrack站返回匹茲堡校園，限週日、開學前或開課日當天。

    行駛時間：

    • 學期間，週日至週三行駛時間7:00 pm ~3:00 am。
    • 學期間，週四至週六行駛時間7:00 pm ~5:00 am。
    • 暑期間，行駛時間均為9:00pm~3:00 am。

    Escort Shuttle #

    CMU晚上提供Escort校車，有固定4條線路，固定站上下車。學期間行駛時間6:30 pm~6:30 am，非學期間行駛使間6:30 pm~1:30 am。乘坐會需要CMU學生證，可攜帶1名客人。

    UPMC Blue Shuttle #


    UPMC在往來Oakland 與Shadyside分院、診所及Family House之間有免費的Shuttle，無須任何證件。平日運營時間為5:00 am到7:00 pm；假日運營時間為8:00 am到7:00 pm，班次較少。詳細路線與時刻表可在參考連結或撥打412－647－3194。

    公車 #


    公車顏色有紅色、藍色、土黃色等，但功能上沒有差異。上車從前門上車並刷卡或付費（不找零），部分時候因為票務機故障等因素則省略。下車從前後門均可。公車的路線查詢可以用Moovit網站；公車即時位置及路線可下載Moovit、Transit app、Busgazer（iOS限定）等app追蹤，其餘app可參考網址map對於公車路線規劃也相當有。使用Google map對於公車路線規劃也相當有用。

    票價 #

    Pitt與CMU學生證可以免費搭乘市內所有公車。現金為$2.75/趟，無轉乘優惠；持ConnectCard為$2.5/趟，轉乘為$1/趟。ConnectCard可購買日票、周票、月票及年票。ConnectCard在匹茲堡機場、公車車站或Giant Ealge等地方皆有販售，詳情參考網址。

    71A/C：沿著Downtown進入Fifth Ave穿過Pitt，轉至Centre Ave並經過Shadyside hospital、Giant Eagle、ALDI後，接著轉向South Negley ，並最後沿著Penn Ave經過Target、Trader Joe’s。

    71B/D：從Downtown進入並貫穿Fifth Ave。其中71B轉至Highland Ave並經過SSA – East Liberty；71D轉至Hamilton Ave。

    54：沿著South Side過橋進入Fifth Ave穿過Pitt，轉至Centre Ave沒多久後就轉入Liberty Ave，Liberty Ave途中會經過好多多、百合超市、Strip District（傳統市集區），最後抵達Allegheny Center（博物館區）。

    61A/B：從Downtown進入並貫穿Forbes Ave（回程到Pitt時走Fifth Ave）抵達Squirrel Hill，往匹茲堡東邊開去。

    61C/D：從Downtown進入並貫穿Forbes Ave（回程到Pitt時走Fifth Ave）抵達Squirrel Hill，轉至Waterfront。其中61C會繼續沿著河往匹茲堡南端開；61D會繞行Costco及Waterfront購物區後返回。

    75：沿著South Side過橋進入Fifth Ave穿過Pitt，轉至Ellsworth Ave，最後轉向北方經過Pittsburgh Zoo & PPG Aquarium，過橋後抵達Walmart購物區。

    28X：機場專線，起站在CMU校門口，中途會經過Pitt、Downtown、Ikea。惟須注意，屬於快速公車，前往Downtown的路線部分站是不停的，時刻表請參照連結。

    每過一段時間，port authority 會對公車線路些微調整，請定期查看常用的公車線路。

    共享單車 #


    Healthy Ride是匹茲堡的共享單車系統。使用方法是使用Nextbike app租用，在腳踏車輸入對應PIN碼。Pitt教職員工生都可以註冊後，每次30分鐘內免費。

    共享汽車 #


    Zipcar是美國主要的共享汽車系統。需要繳年費，學生年費有折扣。租車以小時計費、每天有最高費率上限。提供保險、加油不用錢。使用完後需要將汽車停回原處。

    纜車 #


    Duquesne Incline和Monongahela Incline均是匹茲堡相當有特色的觀光、夜景行程。持有Pitt與CMU學生證可以免費搭乘。

    輕軌 #


    Pittsburgh Light Rail目前主要線路為Downtown至Downtown的河對岸南邊，持有Pitt與CMU學生證可以免費搭乘。

    機場交通 #

    主要有4種方式來回機場。

    以往許多人使用的機場巴士SuperShuttle已於2019.12.31停止服務。

    28X公車 #

    起站為CMU校門口，中間停靠Cathedral Learning、Forbes Ave at Atwood St.（UPMC本院門口）。公車不經過North Oakland、Shadyside與Squirrel Hill，由於28X營運時間較一般公車晚，如果是深夜沒有公車時，需要先擬定前往鄰近公車站的方法。雖為機場快捷公車，但公車設計與一般公車相同，沒有特別行李架設計。

    Uber / Lyft #

    離峰時間約為$35~$40，尖峰時間可能到$70。尖峰或深夜時最好先預約，除了可確保叫得到車外，可以得到比較好的價格。機場有特定的接送區域，但仍是機場大廳門口搭乘。

    他人接送 #

    以North Oakland為例，一般到機場平均需要30分鐘。平日下午通常會塞車，可能會到40~50分鐘。

    自駕 #

    自駕前往機場可停收費停車場，主要分為Short Term、Long Term、Extended和Economy（自2021.06.07設置） 停車區，費率由便宜至貴，詳情參考連結。Extended停車區有接駁巴士，但也可步行至機場大廳（約10~15分鐘）。Long Term和Economy停車區未提供接駁巴士，僅可步行至機場大廳（約5~10分鐘）。

    長途客運 #

    匹茲堡的長途客車的選擇較少，最普遍的為Greyhound和Megabus。

    Greyhound #


    Greyhound即是俗稱的灰狗巴士，覆蓋相當全面，全美本土48個州與加拿大的大部分地區均有他的車站。匹茲堡車站在Downtown，離主要公車路線需要步行5~10分鐘左右。車上提供Wifi與充電，不需要對號入座，由上車順序決定。提供學生折扣。惟須注意，僅提供一件托運行李，額外件數需要加錢購買。

    Megabus #


    Megabus是廉價長途巴士，特點是不使用市區的傳統車站，而是在每個城市的特定街口上下乘客，也沒有遮風避雨的候車亭，建議提前到達候車區，有時候候車區會更動。訂車時會需要選位，部分位置需要額外收費。

    火車 #


    Amtrak（美國國鐵）屬於國營事業。雖然Amtrak遍及全美46州，甚至部份路線延伸至加拿大，但其實每個州的火車站點並不多，可到達的地方有限且班次少（經過匹茲堡的路線幾乎是一天一班）。相對於客運，走的是觀光、精緻服務路線。有臥鋪、餐廳區、一般座位也比一般長途客運大兩倍，缺點是許多線路上是沒有提供Wifi。除了芝加哥–匹茲堡路線外，其餘路線都比長途客運慢上不少。票價不便宜，火車票比飛機票貴是很正常的。
