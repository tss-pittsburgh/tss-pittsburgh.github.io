    全球入境計劃 #

    最後更新日期2021.06.21

    全球入境計劃（Global Entry）是由美國海關及邊境保護局（CBP）推出的旅客信任計劃。讓事先申請並通過審核、低風險的旅客在抵達美國時可以快速完成通關檢查，並同時享有TSA Pre✓®機場快速安檢的功能。

    計劃福利 #

    效期5年。全球入境計劃的會員在入境美國時可以在美國主要機場及全球各地預先核可的地點，使用全球入境計劃的自動查驗機完成海關及移民通關檢查。此外，會員也有資格自動加入TSA PreCheck®。

    TSA PreCheck®是由美國國土安全部的運輸保安署所規劃的旅客快速安檢措施，其好處包含：

    • 享有TSA Pre✓®專屬通道。
    • 不需要脫下鞋子、皮帶、薄外套。
    • 不需取出收納於隨身行李中的筆記型電腦與液態物品。
    • 隨行12歲（含）以下兒童可以一同使用。

    本身TSA Pre✓®僅限美國公民和綠卡持有者申請。然而，只要參與全球入境計劃的會員，都自動享有TSA PreCheck®的福利。

    申請資格 #

    全球入境計劃限美國公民、綠卡持有者或台灣等11個國家申請，並持有有效的ESTA或特定美國簽證（包含F、J簽證、H-1B等）。

    申請方式 #

    1. 備齊必要文件

      • 台灣警察刑事紀錄證明：俗稱警察紀錄或良民證。必須是在申請全球入境計劃前1年內核發的證明，線上申請表會要求填寫警察紀錄上的文號。需要於台灣警察局領取，可請人代領。
      • 有效的台灣護照
      • 有效的ESTA旅行許可或美國簽證
      • 另一種含照片的證件(例如，駕駛執照或身分證)
      • 身分證字號

    2. 在線上向美國海關及邊境保護局提出申請
      在旅客信任計劃（TTP）網站填寫線上申請表並支付$100。無論通過與否，申請費均不退費。一般而言，外國公民需要等到半年以上才會獲得條件式的初步核准（Conditionally approved）。

      申請費用
      大多的高階信用卡提供報銷申請費用的福利。

    3. 預約面談或在抵達美國時落地註冊
      一旦申請獲得初步核准，TTP帳戶會收到訊息，並按指示到線上面談地點（通常在機場）與時間，或參加落地註冊計劃（Enrollment on Arrival）。

      落地計劃
      初審通過的全球入境計劃申請人在搭乘國際航班抵達的時候，可以在入境前面談而不用事先預約。

    4. 面試
      帶著有效護照及另一種身分證件（例如駕駛執照或身分證）以及其他簽證相關文件（例如I-20）面談。如果在申請後、面試前搬家，需要額外攜帶2份地址證明。以PIT機場面試為例，抵達西南售票處的機場大廳後，撥打412-472-0805 Ext. 2500 報到。審核的官員會詢問問題、拍照、採集生物識別特徵（例如指紋）、並解說如何使用機場內的自動查驗機器等。

      面試問題
      問題舉凡：確認email address、媽媽的婚前姓氏、申請Global Entry後還去過哪些國家、列舉5年內去過的國家，並詢問原因、申請地址是不是現在地址等。

    5. 核准
      除美國公民、綠卡持有者或墨西哥公民外，都不會獲得Global Entry卡片，只會有已知旅客號碼（Known Traveler Number, KTN），這組號碼就是線上申請時的TTP會員編號。

    使用方式 #

    美國境內航班 #

    購買機票時可以填寫Known Traveler Number或是報到時請地勤輸入方可使用TSA Pre✓的福利。惟須注意，並非所有航空公司都參加TSA Pre✓。

    入境美國 #

    使用自動查驗機會掃描護照身份頁、指紋按壓最後報關。自動查驗機會顯示旅客的航班資訊並作最後確認。自動查驗機會列一張收據判斷是否需要人工查核。惟須注意，由於F-1簽證必須驗證I-20，所以收據上一定顯示會需要經過移民官查核。雖然沒辦法直接通關，但仍可跳過大批排隊等待入境的人群。
