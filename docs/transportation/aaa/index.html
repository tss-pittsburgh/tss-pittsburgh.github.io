    AAA會員 #

    最後更新日期2021.05.24

    美國汽車協會（American Automobile Association，簡稱AAA或Triple A），是一個百年歷史非營利北美汽車聯盟。AAA所提供的服務是會員制，服務內容包括緊急道路救援、保險、飯店租車等旅遊優惠、旅遊指南等。

    AAA會員的會員費用、福利與網站會依地區有所不同，匹茲堡的地區屬於East Central。

    會員費用 #

    AAA會員分為3個等級，價格由低到高分別是Classic，Plus和Premier。2021年會員年費分別為$54、$90、$129，可選擇月繳分別是$4.5/月、$7.5/月、$10.58/月。

    除非遇到年繳的促銷活動，由於年繳有時有Admission fee，不然選擇月繳更加划算。如果網路上沒有優惠活動時，到AAA據點現場辦年費會員有機會得到額外的折扣。

    會員福利 #

    越高級的會員享有越多AAA福利，以下提幾個主要福利。

    道路救援 #

    道路救援服務是AAA的主要服務。值得一提的是，AAA道路救援服務是認卡不認車，因此朋友遇難時也是可以出手相助的。AAA服務對會員大多是免費的，包括：

    • 拖吊
    • 鑰匙鎖在車裡的開鎖
    • 車輛電池沒電的跳接
    • 爆胎時的換備胎
    • 沒油需要有人送油
    • 遇到困在雪堆、沙堆需要將車拖出（Extrication/Winching）
    • 小故障排除
    無論是租車或自用的車，如果沒有購買road emergency insurance，道路救援是非常花錢的。如果是購買車險的道路救援，如果使用到的話，還有漲保險的風險。購買AAA會員可以省去購買此保險的費用。

    Hertz租車 #

    AAA會員在租Hertz享有多重福利，主要包含：

    • 免Young Renter Fee （~$20/day）
    • 免Additional driver fee （~$10/day）
    • 額外租車折扣

    額外折扣 #

    許多飯店、門票（包括主題公園、電影與運動等門票）、商店有提供AAA會員額外的折扣，詳情參考AAA網站。

    據點服務 #

    AAA的據點提供公證（Notary Service）、護照拍攝等服務。
