    牙科保險簡介 #

    最後更新日期2022.12.20

    雖然牙科保險並非強制的，但牙科保險保費一年僅約$220並提供免費檢查與每半年一次的洗牙，推薦購買。如果沒有保險要檢查牙齒，可能光X光片就要$99。

    常用術語 #

    牙科保險項目主要分為4類服務（可能根據保險公司有所差異）：

    • Class I：預防診斷類別，包含口腔檢查（Exam）、X-rays、常規洗牙（Cleanings）、塗氟（Fluoride Treatment）等，學校保險都是100%報銷且不計入Deductible。一般無保險的情況下，光檢查費用就要$200。
    • Class II：基礎治療，包含補牙（Fillings）、拔牙（Extractions）、根管治療（Root Canal/Endodontics）、深度洗牙（Periodontal Maintenance）等。大多數保險的報銷為70%~80%。
    • Class III：主要治療，包含烤瓷牙（Crown）、植牙（Implant）、假牙（Denture）。大多數保險的報銷為50%。
    • Class IV：牙齒矯正（Orthodontia），通常報銷與上述3類分開計算，通常僅限19歲以下。
    牙齒意外損傷
    部分醫療保險承保意外所致的牙齒損傷，例如Pitt和CMU提供的醫療保險。

    校外牙科保險 #

    如果臨時有看牙需求但沒有學校保險的情況下可考慮購買校外牙科保險。如Delta Dental，購買校外保險前需要確定是否有Waiting Period。如果來不及等Waiting Period結束，可以告知診所狀況，診所可以協助以適當的方法規避Waiting Period。

    牙科折扣計畫 #

    牙科折扣計畫（Dental Discount Plan）類似會員折扣計畫，費用為年付，如DentalPlans.com。通常沒有Waiting Period、Dedutible與Maximum Benefits，但限定於In-Network且沒有免費口腔檢查與護理。
