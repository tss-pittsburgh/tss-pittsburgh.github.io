    保險選擇 #

    最後更新日期2023.01.06

    本文主要闡述留學生、學人及其家屬的醫療保險選擇。另外，本文也將提及過去不在美國生活的美國公民與綠卡持有者醫療保險選擇。

    F簽證 #

    F簽證包含F-1在校生、OPT實習生以及F-2家屬（配偶與子女）。美國政府並未對F簽證持有者有醫療保險的最低要求。

    F-1在校生 #

    學校通常會F-1在校生生有最低保險要求，獲取保險方式主要為：

    • 學校學生保險：通常分為強制購買或有條件豁免。
    • 校外留學生保險：大部分學校允許學生投保符合規定的校外保險以豁免學校要求。通常校外保險公司都有自己的學校資料庫，以提供留學生符合規定的保險。詳情參考校外留學保險章節。

    OPT #

    OPT實習生通常沒有資格購買學校學生保險（畢業前已投保的年度仍可繼續但下年度無法續保），也不受到學校保險規範。獲取保險方式主要為：

    • 雇主團體保險
    • 校外留學保險：雇主未提供保險、自雇者或失業期的空窗期銜接。

    F-2眷屬 #

    通常學校也不會強制F-2簽證的配偶或子女有醫療保險。獲取保險方式主要為：

    • 學校學生保險：通常不能獨立投保，必須依附在F-1在校生的保險下。此外，部分學校也不提供F-2眷屬的保險。
    • 校外留學保險：大部分F-2眷屬可以獨立投保。

    J簽證 #

    美國政府對J1/J2簽證有最低投保要求的，具體要求如下：

    • 每宗傷病的最高醫療保額至少有$100,000
    • 遺體遣返費用保障達$25,000
    • 緊急醫療運送的費用保障達$50,000
    • 每宗傷病的Deductible不超過$500
    • 承擔保險計劃醫療福利的承保商符合最低要求的評級

    J-1學生學人 #

    J-1簽證持有人主要分為長期的J-1 Scholar（包含Research Scholar、Professor，至多5年）、與短期的J-1 Short-term Scholar（至多6個月）與J-1 Student（包含Intern與Non-degree，至多1年）。獲取保險方式主要為：

    • 計畫資助人提供的保險：例如美國國務院辦理的訪問計畫。
    • 學校員工保險：限長期的J-1 Research、Scholar/Professor，且必須要在學校編制內才能投保。，由於受到學校補助，保費通常相當實惠。 惟需注意，不能選擇High deductible類型的保險。
    • 校外留學保險：部分保險經銷商會提供給J-1 Scholar比F-1/J-1 Student更優惠的保險，例如PSI Health Insurance。
    J-1簽證持有者務必確認保險來源
    J-1學生學人可能不屬於學校編制內，因此不能投保學校員工保險。可試著跟雇主（教授）或系上爭取編入成為staff，則可以獲得實惠又全面的學校員工保險。

    J-2眷屬 #

    獲取保險方式主要為：

    • 依附J1持有者的保險：如果J-1持有者可以辦理學校員工保險或計畫資助人提供的保險，通常其眷屬可以依附其保險。
    • 校外留學保險：大部分J-2家屬可以獨立投保。
    • 雇主團體保險：由於J-2眷屬可以經授權後工作，因此J-2眷屬可以透過雇主獲得保險。

    美國公民與綠卡持有者 #

    即便長期不在美國生活，美國公民與綠卡持有者並不會被視為留學生，也因此不能投保校外留學保險。獲取保險方式主要為：

    • 依附父母保險：如果父母在美國有保險可選擇依附保險。縱使不在開放投保期，「搬遷至美國」符合特殊投保期的資格。
    • 學校學生/員工保險
    • 雇主團體保險：由於美國公民與綠卡持有者可就學期間兼職，因此也可能從雇主那獲得保險。
    • Marketplace保險：至聯邦政府的健康保險販售平台HealthCare.gov選購。縱使不在開放投保期，「搬遷至美國」符合特殊投保期的資格。投保可依據收入享有不等的醫療補貼。詳情參考主要醫療險章節。
    • Non-Marketplace保險：不需要符合ACA法案規範下的商業保險。由於無法受到美國聯邦政府的醫療補貼，不一定比較划算。
    • 聯邦醫療補助計劃 (Medicaid) ：俗稱白卡，主要提供給低收入人群，適合沒有收入的學生且沒有被父母撫養，隨時可以投保。
