    申辦SSN / SSN Denial Letter #

    最後更新日期2022.05.29

    社會安全碼（Social Security Number, SSN）是一組9位數字號碼（XXX – XX – XXXX），由社會安全局（Social Security Administration, SSA）免費提供。

    謹慎保管SSN
    非必要不要提供SSN或攜帶SSN卡，也盡量避免使用Email發送SSN。通常銀行、信用卡、薪資發放、保險等辦理會需要繳交SSN。

    申辦目的 #

    SSN：主要是為個人報稅、信用紀錄查詢（如申辦信用卡、貸款）、社會安全福利和用於其他識別目的等。

    SSN Denial Letter：申請駕照或State ID時，如果沒辦法申請SSN，必須先向SSA申請SSN Denial Letter (SSA-L676)取代SSN要求。

    申辦資格 #

    除了美國公民外，綠卡或有美國工作的人均須申辦。一般是如果沒有綠卡、無工作的外籍人士不能申請SSN。

    由於有了SSN會在美國再辦理很多事物上方便許多，F1簽證學生可由校內工作、CPT或OPT獲得SSN，參照工作章節。

    申辦時機 #

    無論申請SSN或SSN Denial Letter，均需要參加完OIS/OIE Orientation和Immigration Check-In Session，並開通SEVIS以及抵達美國至少10天後方可申請。

    申辦方式 #

    多數SSA辦公室不須預約
    自2022.04.12起，大部分SSA辦公室已經恢復Walk-in（不用事先預約）辦理SSN、SSN Denial Letter。少部分辦公室仍須電話預約，詳細清單參考SSA官網。

    無轄區限制
    可以親自到任何SSA 辦公室辦理SSN、SSN Denial Letter。鄰近SSA可參考轄區查詢。

    SSN #

    須得到學校提供的Social Security Letter（校內工作）或Work Eligibility Document（例如：CPT I-20、OPT I-20、AT DS-2019）。

    流程參照CMU、Pitt提供的官方文件。主要流程：

    1. 準備護照、I-94、I-20、Social Security Letter、EAD (if on OPT)等文件。
    2. 親自前往SSA提交申請。
    3. SSN卡約7個工作天會郵寄收到住處。
    預約申辦（疫情間適用，部分辦公室已不適用） 由於每間SSA有不同的申辦流程，建議先致電欲前往的SSA詢問流程。如果不允許Walk-in辦理，申請參考流程： 1. 寄送文件（I-20、簽證等重要文件為影本即可）。 2. SSA會連絡至現場辦理。 3. SSN卡會郵寄至住處。

    SSN Denial Letter #

    此文件不需要從學校得到許可文件即可自行申請。

    親自前往SSA提交申請後，攜帶護照（含簽證）、I-94，當場會提供SSN Denial Letter (SSA-L676)。

    郵寄申辦（疫情間適用，可能已不適用） 以下流程適用於匹茲堡地區：
    1. 填寫Form SS-5，並額外寫一份簡單說明,如"requesting an SSN denial letter for a Pennsylvania driver’s license"。
    2. 護照資料頁（即照片頁）以及簽證頁影本。
    3. 將文件郵寄至 6117 Station St., Pittsburgh, PA 15206。
    4. SSN Denial Letter (SSA-L676)會郵寄至住處。

    SSA地點 #

    學校附近有兩個可供申請SSN的辦公室。轄區查詢請參照連結。一個在Downtown、一個在East Liberty，都是公車可以便捷到達的地方。

    常見問題 #

    沒有SSN是否可以工作？ 規定上只要有合法授權的工作（例如校內工作、EAD），沒有SSN仍是允許工作。實務上，許多公司會利用SSN作背景調查，導致許多人在沒有獲得SSN前是不被允許工作的。

    沒辦法申辦SSN，但仍須繳稅需要如何處理？

    如果F簽證（或其親屬簽證）沒有資格獲得SSN，但在美國期間獲得了非工資收入（例如，獎學金、助學金、股票利息、賭博/彩票獎金），必須申請個人納稅人識別號 (ITIN)。需更多信息，可參考ITIN介紹。
