    前往美國 #

    林奕丞協編
    最後更新日期2021.06.22

    行前注意與準備 #

    航程安排 #

    抵達美國的國際線至國內轉機或加拿大前往美國轉機前，將會需要通過移民官、領出託運行李並再通過海關入境美國，因此保留最少3小時的轉機時間為佳。疫情期間，國際學生最早以及最晚可以何時抵達，可以參考Pitt FAQ。

    隨身攜帶 #

    記得隨身攜帶護照（含簽證）、I-20/DS-2019。由於入境相關文件在到達美國本土提領行李前即須出示，請勿置於托運行李中。

    行李限制 #

    一般來說美洲航線行李上限規定為托運行李2件各23 kg，隨身行李為隨身包與登機箱各1件。若為美國航空公司一般不限制登機箱重量，若所訂機票為亞洲航空公司，一般會有7~10 kg限制，請於出發前上航空公司官方網站進行確認。

    防疫措施 #

    由於COVID-19疫情流行，身處封閉空間應注意防疫措施並評估適合個人之防護裝備。謹記室內必須隨時配戴口罩之原則，並可選擇穿著防護衣、防護面罩、拋棄式手套或攜帶容器小於100ml之酒精或乾洗手隨時進行消毒。若想避免使用機上盥洗室可選擇穿著尿布。多數機場有強制配戴口罩之規定，請隨身攜帶多片口罩進行替換以防旅途中口罩污損。

    第三國轉機 #

    除非需要過夜，在第三國轉機時行李通常直接由台灣直掛美加入境第一個機場，毋需提領托運行李。入境過程只須配合機場指示，前往指定登機口等待下一段航班即可。除非該機場具有美國境外預先清關功能（加拿大為主），則需要在該機場提領托運行李並且通過移民官與海關。

    各個國家可能隨時更新旅遊相關規定。以加拿大為例，免簽證的台灣人在前往加拿大或在加拿大轉機時，需要線上申辦電子旅行證（ETA）。此外，疫情期間，應於出發前確認該國是否允許台灣出發之旅客轉機以及是否有其他檢疫要求。相關資訊可於外交部網站或各國官方網站獲取。各機場可能有額外手續需配合（如疫調、旅行紀錄等），配合相關指示與流程即可。

    入境美國與轉機 #

    抵達美國國內轉機前須先入境美國，入境程序大致為：

    A.填寫申報單 #

    抵達美國前，空服員會於機上發放海關申報單。填寫皆以誠實為原則，如果申報不實輕則罰款，重則列入黑名單，不建議以身試罰。

    如果有帶食物，入境表格上要勾選攜帶食物。如果未申報或攜帶禁止食物若身上攜帶大於$10,000等值之現鈔需額外填寫FinCEN Form 105表單進行申報。為求通關順利、旅途安全性，盡量避免攜帶超過此量之現鈔，以電匯代替，詳情參見國際電匯章節。

    B.移民官詢問 #

    備妥護照、I-20 / DS-2019 與先前填寫之海關申報單等必要文件。可額外準備Offer Letter等補充資料以備不時之需。美國公民者請務必使用美國護照入境、其餘務必使用F1/J1簽證入境，切勿使用ESTA或與此行身份不符之其他簽證入境。移民官標準問題包含入境原因、時間長度、目的、就讀學校科系名稱與未來居住地等。若申報單上有勾選攜帶食物，可能會詢問食物內容，所有回覆以誠實至上原則即可。若不幸因為隨機抽樣或其他特殊原因被要求進一步檢查，無須慌張，依照移民官指示提供相關資訊，並配合檢查攜帶物品即可。

    學校授課型態也可能是移民官詢問問題之一。請注意美國政府2020年疫情下的政策是不允許新生以全線上授課形式入境美國。CMU、Pitt於2020 Fall採取Hybrid授課形式，即同時包含in-person與online二種授課模式，可作為回答移民官的參考。

    C.提領托運行李 #

    依照機場指標，找到搭乘航班行李轉盤領取托運行李即可。若不幸遭遇行李遺失，馬上尋找航空公司的地勤人員查詢。地勤人員可以了解行李最終紀錄之時間與地點，若當場無法找回，請記得向地勤人員登記行李遺失並領取收據（也記得通知接機人員），後續航空公司尋獲行李會依照登記資訊進行聯絡並運送至登記地址。領取完托運行李時記得確認旅途中購買的物品是否有不能以隨身行李登機之品項（如免稅店購買之酒品等），於再次托運前將其置於托運行李中。

    D.連接下段航程 #

    依照機場標示找到執飛下段航程之航空公司櫃台進行行李再托運與國內線報到等手續，再次通過安檢程序並到達指定登機口即可，具體流程與初次報到相仿。

    • 美國國內線航班通常不提供餐點只有簡單零食，若有飲食上需求可於轉機時解決或購買餐點至飛機上。
    • 過海關後的轉機點，只要轉機時間不超過12小時或沒有過夜的話，行李會直掛至匹茲堡。如果有提領需求，可跟航空公司說明。

    抵達匹茲堡機場 #

    由於通常已是到達美國的第二或第三站，無須經過移民關、海關，只須提領行李即可離開機場。機場前往住處之方式：

    • Uber / Lyft：約 $40~80（24小時皆提供服務，特殊時段會加價。）
    • 28X公車：$2.75單程（抵達Oakland約一個多小時，營運時間：4:00 am~0:00 am）
    • TSS組織的的「匹茲堡機場–住處」接送服務（有收費），詳情請見TSS社團資訊。

    常見問題 #

    是否可以從加拿大邊境入境美國？ 疫情期間，美加邊境仍未開放。如果美加邊境開放後，可以飛達多倫多並從陸上交通入境美國。惟須注意，邊境上的車道無法啟用SEVIS，因此會被帶去辦公室（俗稱小黑屋）辦手續，不需要緊張。
