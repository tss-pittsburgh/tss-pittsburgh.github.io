    申辦手機門號 #

    最後更新日期2021.05.24

    電信方案經常更換，本文僅概略介紹電信方案嘗試與各電信公司特色。推薦方案價錢皆不含稅且可能隨時間有所差異。

    電信方案 #

    美國的電信方案通常包含美國境內無限通話和簡訊。主要是以付費方式、網路流量、流速區分價錢。在選擇電信方案前，可閱讀以下美國常見的分類。

    通訊技術 #

    美國主流通訊技術分為CDMA與GSM。台灣的電信頻段通常為700、900、1800、2100與2600MHz，與美國有所差異，詳情可參照維基百科。台灣手機晶片所採用的訊號頻段是符合台灣電信公司提供的頻段，因此可能到美國某些電信商或全部電信商都訊號差或收不到訊號。如果沒準備換手機，最好到店裡試過再辦。

    美國常見電信商頻段

    付費方式 #

    美國的手機方案的付費方式主要分為Prepaid和Postpaid。

    Prepaid
    類似預付卡，設定每月自動扣繳通常可享$5~10折扣。無合約、無須信用查核（credit check），線上申辦通常也不會有啟動費，也可以隨時改方案。通常Prepaid是Indivdual Plan，即一個帳戶只辦一個門號。講究個人隱私或覺得合辦套餐麻煩者適用。

    Postpaid
    類似台灣的簽約方案。通常提供購買手機折扣、額外的福利（例如：免費串流影音會員、雲端空間等），因此通常比較貴。由於是使用後付款，通常須要信用查核，如果沒有SSN則以押金取代。 通常Postpaid是Family Plan，即允許一個帳戶申辦多個門號，不限於家人，可以與任何人組。當一個帳戶人數越多，每條線路的電信費都會有折扣。當然人數達到最大折扣時，價錢就與Prepaid方案有競爭力。

    使用Family Plan須注意以下幾點：

    • 由一個人付錢，再向其他成員收取費用。須注意金錢糾紛。
    • 進入或退出都需要管理者操作。
    • 為維持最大折扣，退出時可能被要求找人遞補空缺。
    • 有些Family Plan是共享流量的。

    吃到飽方案 #

    美國的吃到飽方案稱為Unlimited Plan。如果研究吃到飽方案，同一家電信都是吃到飽方案卻有不同的價錢，這是由於是否允許熱點分享、網速以及去優先度的差異。

    熱點分享
    Prepaid和Postpaid的基礎版吃到飽都不允熱點分享。Postpaid的高階版吃到飽會提供一定流量內的熱點分享，超過降速。

    網速
    Prepaid的任何方案和Postpaid的基礎版吃到飽都是低速版，即使用串流影音時網速限制在480p（即SD畫質，約1.5Mbps）。Postpaid的高階版吃到飽會提供720p以上（即HD畫質，>2Mpbs）的網速。大部分SD網速足夠大部分使用，然而多數串流平台（例如：Netflix）會由網速調控畫質，因此如果特別在意高畫質的串流影音，建議選擇高階版吃到飽。

    去優先度（Deprioritization）
    當網路雍塞時（例如：大型賽事、演唱會、大都會等人多地方），會暫時降低網速。在網路雍塞時，Prepaid的所有方案和Postpaid的基礎版吃到飽通常順位較低。Postpaid的高階版吃到飽在一定流量內保證優先度，使用超過一定流量時（通常為22GB或50GB），原本優先度就會被降低而導致網速下降。

    Throttling與Deprioritization的差異
    Throttling是發生在非吃到飽用為流量時，網速自動被降至2G（約128kbps）直至該帳單週期為止。與非吃到飽不同，吃到飽的降速僅發生在網速雍塞時。

    網路覆蓋 #

    通常城市的網路覆蓋都沒什麼問題。但如果熱衷於跑國家公園或比較偏僻的地方，覆蓋率是很重要的因素。

    三大電信運營商 #

    美國三大電信運營商分別是Verizon、AT&T與T-Mobile。

    Verizon #

    全美3G與LTE覆蓋率最高、訊號穩定度最好的電信運營商。通常價錢也是最貴的。使用CDMA技術。

    Prepaid推薦方案

    • $25/月（初始費率$35/月，使用3個月後降$5、9個月後降$10）、5GB。

    Postpaid推薦方案

    • $30/月、Unlimited、5+線路。（Start Unlimited）
    • $40/月、Unlimited、5+線路、50GB內保證優先度、允許熱點分享、免費Disney+、Hulu、ESPN+會員。（Play More Unlimited）

    AT&T #

    全美3G與LTE覆蓋率、訊號穩定度排名第二。使用GSM技術。最高階吃到飽方案提供免費HB Max會員。

    Prepaid推薦方案

    • $30/月、5GB。
    • $25/月、8GB，須預付一年費用（$300）。

    Postpaid推薦方案

    • $35/月、Unlimited、4+線路。（Unlimited Starter）

    T-Mobile #

    全美3G與LTE覆蓋率、訊號穩定度排名第三，價格通常是最便宜。使用CDMA技術。特定吃到飽方案提供免費Netflix會員。於2021年4月併購原本美國第四大運營商Sprint共同打造5G網路，目前5G信號覆蓋率最好。

    Prepaid推薦方案

    • $15/月、2.5GB，每年免費每月增加500MB至2025年。
    • $25/月、5.5GB，每年免費每月增加500MB至2025年。

    Postpaid推薦方案

    • $24/月、Unlimited、5+線路，50GB內保證優先度。（Essentials）
    • $32/月、Unlimited、5+線路，100GB內保證優先度、免費Netflix Basic、允許熱點分享。（Magenta）

    虛擬電信運營商 #

    虛擬電信運營商是租用三大電信運營商中的網路，再自行包裝成方案販售。通常較為便宜，但訊號強度、流速、覆蓋率以及優先度只會低於採用的電信公司。以下是常見的電信運營商：

    Cricket #

    採用AT&T。其中的Unlimited Family Plan（$25/月，4+線路）最受歡迎。有實體店面。

    Mint #

    採用T-Mobile。相當便宜（$15/月、4GB；$20/月、10GB，須年繳），有時黑五有特價。根據多人回報，訊號覆蓋率偏低、高速行駛或人多地方通話品質、網速較差。如果常離開城市活動的人，須審慎使用。 可先使用3個月體驗方案再決定使否繼續使用。

    Google Fi #

    採用T-Mobile。 費率採流量計費（全球均一價），可以申請多張Data-only SIM卡，給不同人使用。支援全球200+國家網路、電話，國際電話、簡訊費率便宜，適合經常出國的人使用。也有Unlimited Plan與Family Plan。
    Google Fi的Unlimited Plan並非美國常見的吃到飽方案，而是如同一般非吃到飽方案，在使用超過一定流量後，網速直接降至256kbps。

    CTExcel #

    採用T-Mobile。中國電信推出，支援中文服務，無限20國國際通話與無限全球簡訊，支援台灣手機頻帶。

    CUniq #

    採用T-Mobile。中國聯通推出，支援中文服務，支援台灣手機頻帶。可以在台灣網路上直接購買。

    攜號轉網 #

    如果想更換運營商並保留原本門號，可以跟原運營商索取門號的Account Number和Pin Number，交給想更換的運營商轉網。

    常見問題 #

    是否建議到實體店辦理門號？ 實體店面分為直營店與授權店。授權店往往有額外費用、坑殺消費者，因此剛到美國不建議跑到授權店辦理。此外，網路辦理通常會有最好的優惠，但通常需要等待幾天郵寄SIM卡，少部分允許到店家領取SIM卡。
