    遵守疫情措施 #

    最後更新日期2021.06.05

    疫情期間，抵達美國首要事情為安排COVID-19疫苗接種與檢測。

    預約COVID-19疫苗 #

    目前全美不論有沒有保險，任何人都可以「免費」施打COVID-19疫苗。詳情可參閱疫情生活指南章節。

    抵達美國後，即可先去預約施打COVID-19疫苗，不需要等到安排檢測與隔離完成。

    安排檢測與隔離 #

    根據美國CDC的國際旅行後指南：

    • 未獲得疫苗完全保護的人在國際旅行後3~5天後建議做COVID-19檢測並且在旅行後在家隔離7天，如果不想接受檢測，可選擇在家隔離10天。

    • 獲得完全疫苗保護的人仍建議在國際旅行後3~5天後做COVID-19檢測，但僅需要自我監測是否有COVID-19症狀，不需要隔離。

    檢測方式可以為核酸增幅檢測或抗原檢測，如何預約檢測可參考疫情生活指南章節。

    強制措施
    美國與賓州對於入境後的檢疫與隔離均為建議，並沒有強制措施。

    完全疫苗保護（Fully Vaccinated）

    • 接受兩劑型疫苗的兩周後（例如輝瑞或莫德納）
    • 接受一劑型疫苗的兩周後（例如嬌生疫苗）
