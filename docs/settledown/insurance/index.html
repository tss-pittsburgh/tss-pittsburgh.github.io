    投保健康保險 #

    最後更新日期2021.07.18

    健康保險（Health Insurance）可分為醫療保險（Medical Plan）、牙科保險（Dental Plan）、眼科保險（Vision Plan）。

    健康保險簡介 #

    通用術語 #

    • Premium：即保費，通常是月付。
    • Deductible：保險公司開始賠付前，投保人須在投保一年內自行支付的累積費用。
    • Copayment：每次看病、治療時須支付的固定金額，類似台灣的掛號費。
    • Coinsurance：投保人所須要支付醫療費用的百分比，類似台灣的部分負擔。
    • Out-of-pocket Limit：投保人在投保一年內最多須支付的金額，支付的金額包括Deductible、Copayment、Coinsurance等。滿足後，保險公司會100%報銷所有醫療費用。
    • Maximum Benefit：保險公司最高為投保人報銷的費用。依據險種不同，可以限定於單次、投保的一年內或終身。如果沒有上限，則為無。
    • In-/Out-of-network：In-network是指跟保險有合作關係的醫療院所，報銷比例高。Out-of-network的醫療院所報銷低或甚至不報銷。
    • Waiting Period：某些疾病或治療在投保一定時間後才會報銷。

    低Deductible、Copayment、Coinsurance、Out-of-pocket Maximum，高或無上限Maximum Benefit通常保費會較為昂貴。

    醫療保險 #

    國際生是強制購買醫療保險，但不強制購買學校方案。醫療保險依照不同看病模式，主要可分為4種類型：

    • Health Maintenance Organization (HMO)：投保人須要指定家庭醫生，專科診察須要家庭醫師轉診，只報銷In-network的醫療服（緊急狀況除外），自由度低。
    • Exclusive Provider Organization (EPO)：，專科診察不須經由轉診，但仍只報銷In-network的醫療服務（緊急狀況除外），自由度比HMO稍高。
    • Preferred Provider Organization (PPO)：專科診察不須經由轉診外，也報銷Out-of-network的醫療服務，但報銷費用會少In-network許多，自由度最高，保費一般較貴。
    • Point of Service (POS)：投保人須要指定家庭醫生，專科診察須要家庭醫師轉診。與PPO相同，報銷Out-of-network的醫療服務。計畫介於HMO與PPO之間。

    牙科保險 #

    非強制性，推薦購買。牙科保險項目主要分為4類服務（可能根據保險公司有所差異）：

    • Class I：預防診斷類別，包含口腔檢查（Exam）、X-rays、常規洗牙（Cleanings）、塗氟（Fluoride Treatment）等，學校保險都是100%報銷且不計入deductible。一般無保險的情況下，光檢查費用就要$200。

    • Class II：基礎治療，包含補牙（Fillings）、拔牙（Extractions）、根管治療（Root Canal/Endodontics）、深度洗牙（Periodontal Maintenance）等。大多數保險的報銷為70%~80%。

    • Class III：主要治療，包含烤瓷牙（Crown）、植牙（Implant）、假牙（Denture）。大多數保險的報銷為50%。

    • Class IV：牙齒矯正（Orthodontia），通常報銷與上述3類分開計算，通常僅限19歲以下。

    眼科保險 #

    非強制性。報銷/折抵驗光、眼睛檢查、眼鏡（配鏡架和鏡片）、隱形眼鏡等方面的費用。惟須注意，除了眼鏡、隱形眼鏡可在台灣多買幾份備份替代此保險，眼睛疾病、治療是使用醫療保險而非眼科保險。

    Pitt學生保險 #

    以下主要介紹Pitt學生保險。如果變成Staff（例如在圖書館打工），Pitt提供員工保險。

    保期 #

    過去Pitt起保日期一般從9/1開始，即便疫情因素，有提供投保8/16~8/31的醫療險選項，但8月的空窗期建議以旅遊險或留學險填補。

    醫療保險 #

    Pitt的醫療保險與UPMC合作。

    Graduate Student Medical Plan #

    限定研究生、博士後研究員、健康科學研究員及其眷屬可以投保。學校會免費幫有TA、RA獎學金的研究生投保。昂貴，但無論是自付額、部分負擔皆是最頂級的。

    General Medical Plan #

    任何學生及其眷屬皆可以投保。較為便宜，自付額、部分負擔比例較差。

    牙科保險 #

    Pitt的醫療保險與United Concordia合作。任何學生及其眷屬都可以保，分為研究生和大學生方案。


    檢查、洗牙（每半年）和X-ray都是免費的。投保後，在United Concordia網頁中的Covered Services可以查到所有治療的費用，在哪家接受治療價格都一樣（僅Pitt牙醫學院會有額外折扣）。

    眼科保險 #

    Pitt的醫療保險與Davis Vision合作。任何學生及其眷屬都可以保。

    CMU學生保險 #

    2021-2022的CMU醫療、牙科、眼科保險是與Highmark合作。保險方案大學生和研究生一樣。

    保期與保費 #

    August 1, 2021 - July 31, 2022

    醫療保險 #

    CMU的醫療保險在In-Network醫療主要負擔copayment，剩下由保險公司100%負擔，詳細可參考醫療保險權益摘要。

    In-Network主要內容包含：

    • Deductible: $0
    • Retail Clinic / Primary Care Provider: $25 per visit
    • Specialist: $25 per visit
    • Urgent Care: $25 per visit
    • Emergency Room: $125 per visit
    • Telemedicine (Amwell): $20 per visit
    • Outpatient Metal Heatl: $0 per visit
    • Physical Medicine / Occupational Therapy: $0 (Limit: 30 visits)
    • Diagnostic Services: $25 (basic) or $40 (advanced imaging)

    其餘權益包含：全球醫療（Blue Cross Blue Shield Global Core）和旅遊醫療協助（AXA Travel Assistance）等。

    牙科保險 #

    CMU牙科保險內容可參考牙科保險權益摘要。保險內容需要注意以下幾點：

    1. Deductible: $100
    2. Maximum Benefit: $1,000
    3. 即使在In-Network治療仍需要負擔20%~50%的治療費用而非固定費用，因此務必在接受治療前問清楚價格。
    4. 牙齒矯正並未涵蓋。
    在接受任何診斷、治療前，你有權先向醫生、護理人員詢問價錢。

    眼科保險 #

    CMU眼科保險內容可參考眼科保險權益摘要。

    校外保險 #

    國際學生可購買其他符合學校最低要求的醫療保險（CMU最低要求，Pitt未明訂）。部分校外的保險公司提供學生限定的保險多是以價格導向（便宜為主）。

    切記一分錢一分貨

    如果想從學校方案外挑選適合自己的，非學校方案的醫療保險公司推薦名單：

    • UPMC
    • Aetna
    • United Healthcare
    • Blue Cross Blue Shield
    • Cigna

    變更保險 #

    變更保險包含中途加入或退出保險（含個人、配偶、受扶養人）。一般而言，為了避免等到生病才想投保，學校保險有限定的註冊期且不能任意退出。除非發生「合格生活事件」（Qualifying Life Event），才允許中途加入或退出保險。

    「合格生活事件」包含入學、結婚、生育、遷居、工作等都符合變更保險的要件，明細可參考HealthCare.gov清單。特別注意，生病、牙痛並不屬於「合格生活事件」。

    變更Pitt學校保險流程可參考Pitt人力資源辦公室網頁。

    變更CMU學校保險須郵電至shinsure@andrew.cmu.edu索取相關文件。退出CMU學校保險亦可填寫CMU退保表格（仍建議先email詢問，不確定更換保險公司後該表格是否有效）。

    CMU退保表格摘錄
    如果在註冊年度內4月1日之前取消學生醫療保險，並且保險公司為你或你的家人沒有支付重大索賠（包含處方、就診、急診等），就有資格按比例獲得退款。重大索賠是根據具體情況而定。

    常見問題 #

    學校是否有提供半年保險？

    CMU、Pitt有提供春季入學半年保險，但並未提供秋季學期的半年保險。校外保險則不一定。

    雖然學校並未直接提供半年保險，但畢業是屬於「合格生活事件」（Qualifying Life Event）之一，符合退出保險的要件。退出學校保險流程請參閱變更保險章節

    畢業不強制退出保險
    畢業並不要求一定要退出保險，可以評估下階段（如工作、回台）等需求再作決定。理論上，即使公司如同學校有特定的保險註冊期，但學校保險到期（或稱為失去保險）也屬於「合格生活事件」，符合中途加入保險的要件。

    學校保險的優點與缺點？ 優點：通常較為全面。低Deductible、Out-of-pocket Limit，高Coinsurance。承保原有疾病，無Waiting Period。
    缺點：承保內容固定、昂貴、起保日期固定、投保長度固定。
