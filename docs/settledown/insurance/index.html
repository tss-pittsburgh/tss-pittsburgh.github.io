    投保健康保險 #

    最後更新日期2020.08.13

    健康保險（Health Insurance）可分為醫療保險（Medical Plan）、牙科保險（Dental Plan）、眼科保險（Vision Plan）。學校保險昂貴但較為全面，部分校外的保險公司提供學生限定的保險多是以價格導向（便宜為主）。切記一分錢一分貨。

    名詞解釋 #

    健康保險術語：

    • Premium：即保費，通常是月付。
    • Deductible：保險公司開始賠付前，投保人須在投保一年內自行支付的累積費用。
    • Copayment：每次看病時須支付的固定金額，類似台灣的掛號費。
    • Coinsurance：投保人所須要支付醫療費用的百分比，類似台灣的部分負擔。
    • Out-of-pocket Limit：投保人在投保一年內最多須支付的金額，支付的金額包括Deductible、Copayment、Coinsurance等。滿足後，保險公司會100%報銷所有醫療費用。
    • Maximum Benefit：保險公司最高為投保人報銷的費用。依據險種不同，可以限定於單次、投保的一年內或終身。如果沒有上限，則為無。
    • In-/Out-of-network：In-network是指跟保險有合作關係的醫療院所，報銷比例高。Out-of-network的醫療院所報銷低或甚至不報銷。
    • Waiting Period：某些疾病或治療在投保一定時間後才會報銷。

    低Deductible、Copayment、Coinsurance、Out-of-pocket Maximum，高或無上限Maximum Benefit通常保費會較為昂貴。

    醫療保險種類：

    • Health Maintenance Organization (HMO)：投保人須要指定家庭醫生，專科診察須要家庭醫師轉診，只報銷In-network的醫療服（緊急狀況除外），自由度低。
    • Exclusive Provider Organization (EPO)：，專科診察不須經由轉診，但仍只報銷In-network的醫療服務（緊急狀況除外），自由度比HMO稍高。
    • Preferred Provider Organization (PPO)：專科診察不須經由轉診外，也報銷Out-of-network的醫療服務，但報銷費用會少In-network許多，自由度最高，保費一般較貴。
    • Point of Service (POS)：投保人須要指定家庭醫生，專科診察須要家庭醫師轉診。與PPO相同，報銷Out-of-network的醫療服務。計畫介於HMO與PPO之間。

    醫療保險 #

    留學生是強制購買醫療保險，但不限定於購買學校方案。CMU的醫療保險與Aetna合作，大學生和研究生方案一樣；Pitt的醫療保險與UPMC合作，提供General及Graduate 兩個方案，前者大學生、研究生皆可註冊，後者僅限研究生。Pitt研究生如果有Academic Appointment（RA, TA等）學校會免費提供Graduate Student Plan。如果變成Staff，CMU、Pitt皆有提供折扣的員工保險。

    牙科保險 #

    非強制性，建議購買。以Pitt Graduate Plan為例，100%報銷口腔檢查、洗牙、X-ray費用。對於補牙、根管等治療採用Copayment。一般無保險者，光檢查費用（不含補牙）就要$200。

    眼科保險 #

    非強制性。報銷驗光、眼鏡（配鏡架和鏡片）、隱形眼鏡等方面的福利。如果是眼睛疾病、治療，則是使用醫療保險。眼鏡、隱形眼鏡可在台灣多買幾份備份替代此保險。

    學校保險 #

    CMU和Pitt皆有提供學校的保險方案：

    保期 #

    Pitt：起保日期一般從9月開始，最晚投保日期為9/30，保期一年。8月的空窗期建議以旅遊險或留學險填補。（2020年由於疫情因素提供投保8/16~8/31的選項。）

    CMU：起保日期為8/1，最晚投保日期為9/11，保期1年。

    半年保險 #

    CMU、Pitt有提供春季入學半年保險，但並未提供秋季學期的半年保險。

    常見問題 #

    學校保險的優點與缺點？ 優點：通常較為全面。低Deductible、Out-of-pocket Limit，高Coinsurance。承保原有疾病，無Waiting Period。
    缺點：承保內容固定、昂貴、起保日期固定、投保長度固定。

    是否可購買校外保險？ 國際學生可購買其他符合學校最低要求的醫療保險（CMU最低要求，Pitt未明訂）。如果想從學校方案外挑選適合自己的，非學校方案的醫療保險公司推薦名單：UPMC、Aetna、United Healthcare、Blue Cross Blue Shield、Cigna。
