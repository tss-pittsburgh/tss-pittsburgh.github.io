    申辦駕照 / State ID #

    最後更新日期2022.03.30

    在美國日常生活中，常需要Photo ID作為身份、年齡證明（買酒、搭飛機）。由於學生證資料不全，學校以外的地方大多地方不能接受學生證作為Photo ID。雖說護照可以作為Photo ID，但一旦丟失就相當麻煩，不建議經常攜帶。為此，長期待美國都會建議申請美國各州頒發的Photo ID，分別為駕照（Driver’s License）和State ID。

    另外，國際駕照使用上不如美國駕照自由。以賓州為例，國際駕照有一年的效期，且須跟台灣駕照一起使用。部分州甚至不接受國際駕照，開到該州有變成無照駕駛的風險。美國駕照則允許在美國任何一州開車。台灣與賓州有協議，可以用台灣駕照免試申換賓州駕照。因此先在台灣考取台灣駕照會方便很多。

    I-20或DS-2019有效期須11個月（最好超過1年）才能申辦駕照或State ID。疫情期間，規定改為6個月以上。

    免試申換賓州駕照 #

    申換賓州駕照所須文件 #

    詳情與文件下載參照TECO連結：

    1. 台灣官方駕照譯本：由台灣駐紐約辦事處頒發，可現場辦理、郵寄辦理、委託他人辦理（須要公證，不推薦）。 應備文件：
      • 台灣駕照：現場辦理須準備台灣駕照正本及正反影本1份；郵寄辦理僅須正反影本1份，毋須郵寄正本。
      • 文件證明申請表。
      • 護照：現場辦理需要護照正本及護照資料頁影本1份，正本於發件時一併退回；郵寄辦理僅須護照資料頁影本1份，毋須郵寄正本。
      • 駕照英譯表格，填寫後將電子檔(.doc)寄至consular.ny@mofa.gov.tw。
      • 文件驗證規費：每份$15。
      • 如果選擇將台灣官方駕照譯本寄回（通常為郵寄辦理），須另附貼足郵票之回郵信封（限用USPS）。
      • 現場辦理可付現金；郵寄辦理者只接受Money Order（推薦）或Cashier’s Check，收款人抬頭為TECO in New York。
      • 可接受多人一起郵寄並一同寄回。建議一份郵寄最多寄5個人資料。

    TECO不再接受另附回郵郵資
    自2022年起，選擇將台灣官方駕照譯本寄回一律須附貼足郵票之回郵信封。以Money Order含附郵資的方式不再被接受。

    郵寄影本
    請勿在照片上寫字。如在照片上註明「此資料僅供紐約在台辦事處申辦臺灣駕照譯本用」等文字會被要求重寄影本。

    如何購買Money Order？
    建議在USPS郵寄前順便向行員購買Money Order後，一同放入信封內與文件一起郵寄。何謂和如何購買Money Order可參照美國境內轉帳章節。

    貼足郵票之回郵信封
    建議在USPS郵寄前順便向行員購買信封+郵票，並放入信封內與文件一起郵寄。郵票面額一般選擇Priority Mail即可（寄送時間約1~3個工作天、具有Tracking功能）。郵資以美國郵政官網USPS最新規定為準，詳情參考Retail Postage Price Calculator。

    不確定是否可要求寄送Prepaid Forever Priority Mail Flat Rate，此選項較買信封+郵票更便宜。

    1. SSN或SSN denial letter，可參照申請SSN章節。
    2. 最新出入境I-94紀錄，下載連結。
    3. 兩份居住證明（例如銀行帳單、水電費帳單、租約等，手機帳單不行）
    4. 護照及美國簽證
    5. 台灣駕照、國際駕照（兩者均要，不能過期）
    6. 身分文件（例如I-20），依據（非）移民身份有所不同
    居住證明
    官方規定的居住證明包括水電費帳單、租約、稅單（W-2 Form）等。實務上DMV會接受銀行帳單、Amazon包裹上的地址（不要帶整個箱子）。DMV辦事人員可能不會接受私人手寫租約、手機帳單，但有些辦事人員會接受Checking Account、Saving Account的帳單作為2份居住證明。如果不確定要提供什麼作為居住證明，可以把所有的地址相關的信件等文件帶去DMV給辦事人員挑選。此外，兩份居住證明地址要完全一致。

    換照流程 #

    1. 前往監理所（DMV），附近僅Downtown可以申辦。建議開門前就去排隊。
    2. 臨櫃時進行視力檢驗（測試矯正後視力）。檢測是否看到兩側閃爍點。
    3. 現場攝製證件照、簽名、繳交規費（2019年起接受刷卡）。
    4. 現場提供臨時駕照（效期15天，功能等同一般駕照），5~7天寄發正式駕照。

    駕照譯本不在系統
    駐紐約辦事處須將譯本email至賓州政府的專用信箱。除了可能是辦事人員不熟悉系統操作外，有時是駐紐約辦事處漏（晚）上傳或是州政府收信人員的遺漏以至於DMV查不到該譯本的有效性。發生時應聯絡駐紐約辦事處通知補上傳資料，可致電212-317-7300，如果打不通可嘗試領務專用電話212-486-0088。辦事處資料重新上傳後約48~72小時後資料會進入DMV系統，因此約3天後再去DMV辦理。若是仍找無資料，可能需要去DMV多試幾次或再要求駐紐約辦事處再寄一次信。

    尚未拿到SSN或非標準的Denial Letter
    Denial Letter原本是指Ineligibility for SSN（參照PUB 195NC）。然而疫情期間，很多人收到的回信並非是真正的Denial Letter，信中僅表明雖無法頒發SSN但也無法證明沒有資格申請SSN（如下圖）。實際上，Photo ID的申請文件上（DL-160 Form）僅要求Letter from SSA inticating that SSA did not make decision yet or SSA rejection Letter。因此，除了拿到這類Denial Letter的人外，已申辦SSN但尚未SSN的人也可以持申辦SSN的收據向DMV申請駕照。

    考取賓州駕照 #

    如果沒有台灣駕照，想直接考賓州駕照，考試分為兩部份：筆試（Knowledge Test）和路考（Road Test），詳情參考連結。

    考照流程 #

    1. 下載賓州駕駛員手冊閱讀。
    2. 完成Form DL-180（體檢表）。
    3. 參加筆試。
      • 攜帶Form DL-180、所須文件（同免試申換賓州駕照，排除台灣駕照相關文件）、費用（$35.5, 2022）。
      • 雖然有中文試題，但由於翻譯品質問題，建議仍選擇英文試題。
      • 通過後可以得到Learner Permit。有了Learner Permit後可在有駕駛執照的人陪同下開車。
      • 微信群有華人提供陪駕服務，約$60/hr。
    4. 預約路考。
      • 可線上（如果沒SSN須填簽證或I-94後四碼）或電話（717-412-5300）預約。
      • 一張Permit最多可以考3次，沒有時間間隔要求也沒有額外費用。超過須要填寫Form DL-31申請延展，費用$5。
      • 由於現場不提供車，必須要有有美國駕照的陪同者一起參加路考。
      • 路考當天須提供Permit、車輛紙本保險證明、紙本車輛登記證明、陪同者的有效美國駕照。由於租車公司通常不會提供相關文件，因此通常須要跟有認識的人借車。
      • 微信群有華人提供陪考服務，約$100/次。
    5. 獲得駕照。如果在PennDot駕照中心通過路考，即可於現場獲得駕照。

    駕照體檢 #

    主要有三個體檢單位。

    • 學校：CMU University Health Service；Pitt Wellness Center均提供體檢，皆須收費。
    • 初級保健：可由自己的家庭醫生完成體檢，依照保險負擔。如果剛好是Annual Wellness Visit則不須收費。詳情參照初級保健章節。
    • Urgent Care：部分UPMC Urgent Care提供體檢服務，不須預約。詳情參照UPMC網頁。

    路考內容 #

    • 車輛基本操作：按喇叭、車燈使用（近光燈、遠光燈、方向燈）、雨刷、警示燈、除霜器、停車制動器（手剎車）。
    • 路邊停車：車輛須完全停進24×8英尺空間，不能觸碰前後立柱以及路緣且不能調整超過3次。除了周一～周六08:30～16:30外，其餘時間可至路考中心現場練習。
    • 實地路考：每個考場都是固定路線。Penn Hill駕照中心的路線可參考網址。

    更新駕照 #

    通常駕照期限僅有4年或身份期限（例如I-20的Program End Date）。更新駕照所須文件：

    1. Form DL-143（網上下載）或DL60A/D（DMV會於過期前3個月發送更新邀請）
    2. SSN或SSN denial letter，可參照申請SSN章節。
    3. 最新出入境I-94紀錄，下載連結。
    4. 護照及美國簽證
    5. 身分文件（例如I-20），依據（非）移民身份有所不同，惟須注意申請OPT後也必須攜帶EAD。
    6. 兩份居住證明（如果有更換地址才須要）
    7. SSN（如果想同時獲得REAL ID才須要）

    申辦State ID #

    所須文件與考取賓州駕照類似，不須Form DL-180。

    REAL ID #

    何謂REAL ID？ #

    REAL ID不是一個獨立的ID，是指符合美國聯邦真實身份法案（REAL ID Act）規定下的身份證明文件，包含：

    • REAL ID版駕照或State ID
    • 護照
    • 永久居留卡
    • EAD卡
    • Global Entry卡
    • Government Employee ID
    • U.S. Military / Military Dependent ID
    • …

    REAL ID版駕照或State ID

    • 符合身份查核的規定下，部分州的Photo ID上會有額外註記（例如賓州會於ID右上角註記黃底白星）。
    • 賓州任何有效居民都可申請REAL ID版的駕照或State ID。
    REAL ID的駕照與State ID右上角會有星號標記

    Enhanced Driver’s Licnese

    • 主要用美加、美墨或加勒比海地區邊境通關的身分證明文件。
    • 僅限美國公民申請。
    • 符合REAL ID規定。

    REAL ID用途 #

    自2023.05.03起，搭乘美國國內班機或進入聯邦、軍事設施需要出示符合REAL ID法案的身份證明文件。

    • 單純駕駛或普通身份證明使用一般的駕照或State ID即可。

    加註REAL ID #

    在申辦駕照或State ID時，如果沒有主動說要申辦REAL ID版的駕照或State ID，承辦人員是不會主動辦理的。

    申換REAL ID所須文件，詳情參閱官方網站：

    • 護照及美國簽證
    • SSN或SSN denial letter
    • 兩份居住證明（未過期駕照、汽車登記證、汽車保險卡、水電費帳單等）

    常見問題 #

    除了駕照和State ID，有什麼其他身分文件可以當作Photo ID？
    • 護照
    • Green Card
    • Trusted Traveler ID （例如：Global Entry Card）
    • Government Employee ID
    • U.S. Military / Military Dependent ID
    • …
