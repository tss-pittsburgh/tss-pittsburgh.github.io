    申辦駕照 / State ID #

    最後更新日期2021.09.15

    在美國日常生活中，常需要Photo ID作為身份、年齡證明（買酒、搭飛機）。由於學生證資料不全，學校以外的地方大多地方不能接受學生證作為Photo ID。雖說護照可以作為Photo ID，但一旦丟失就相當麻煩，不建議經常攜帶。為此，長期待美國都會建議申請美國各州頒發的Photo ID，分別為駕照（Driver’s License）和State ID。

    另外，國際駕照使用上不如美國駕照自由。以賓州為例，國際駕照有一年的效期，且須跟台灣駕照一起使用。部分州甚至不接受國際駕照，開到該州有變成無照駕駛的風險。美國駕照則允許在美國任何一州開車。台灣與賓州有協議，可以用台灣駕照免試申換賓州駕照。因此先在台灣考取台灣駕照會方便很多。

    I-20或DS-2019有效期須11個月（最好超過1年）才能申辦駕照或State ID。疫情期間，規定改為6個月以上。

    如果符合美國聯邦政府的身份查核要求下，各州政府頒發的駕照和State ID會有額外註記，即是Real ID版的駕照或State ID。自2023.05.03起，搭乘美國國內班機或進入聯邦、軍事設施需要出示REAL ID，單純駕駛或普通身份證明使用一般的駕照或State ID即可。

    REAL ID的駕照與State ID右上角會有星號標記

    免試申換賓州駕照 #

    申換賓州駕照所須文件 #

    詳情與文件下載參照TECO連結：

    1. 台灣官方駕照譯本：由台灣駐紐約辦事處頒發，可現場辦理、郵寄辦理、委託他人辦理（須要公證，不推薦）。 應備文件：
      • 台灣駕照：現場辦理須準備台灣駕照正本及正反影本1份；郵寄辦理僅須正反影本1份，毋須郵寄正本。
      • 文件證明申請表。
      • 護照：現場辦理需要護照正本及護照資料頁影本1份，正本於發件時一併退回；郵寄辦理僅須護照資料頁影本1份，毋須郵寄正本。
      • 駕照英譯表格，填寫後將電子檔(.doc)寄至consular.ny@mofa.gov.tw。
      • 文件驗證規費：每份$15。
      • 郵寄辦理者須另附貼足郵票之回郵信封或回郵郵資（用於將台灣官方駕照譯本寄回）。回郵郵資是浮動的，費用詳情參考連結，通常是以快遞寄回（2021年費用為$26.5）。
      • 現場辦理可付現金；郵寄辦理者只接受Money Order（推薦）或Cashier’s Check，收款人抬頭為TECO in New York。
      • 可接受多人一起郵寄共攤一份郵資，並只要提供一張足額的Money Order。建議一份郵寄最多寄5個人資料。
    郵寄影本
    請勿在照片上寫字。如在照片上註明「此資料僅供紐約在台辦事處申辦臺灣駕照譯本用」等文字會被要求重寄影本。

    如何購買Money Order
    建議去USPS郵寄時順便向行員購買Money Order，購買後將Money Order放入信封內與文件一起郵寄。何謂和如何購買Money Order可參照美國境內轉帳章節。

    Money Order票面金額 = 文件規費（$15）×人數 + 快遞郵資（$26.5）

    1. SSN或SSN denial letter，可參照申請SSN章節。
    2. 最新出入境I-94紀錄，下載連結。
    3. 兩份居住證明（例如銀行帳單、水電費帳單、租約等，手機帳單不行）
    4. 護照及美國簽證
    5. 台灣駕照、國際駕照（兩者均要，不能過期）
    6. 身分文件（例如I-20），依據（非）移民身份有所不同
    居住證明
    官方規定的居住證明包括水電費帳單、租約、稅單（W-2 Form）等。實務上DMV會接受銀行帳單、Amazon包裹上的地址（不要帶整個箱子）。DMV辦事人員可能不會接受私人手寫租約、手機帳單，但有些辦事人員會接受Checking Account、Saving Account的帳單作為2份居住證明。如果不確定要提供什麼作為居住證明，可以把所有的地址相關的信件等文件帶去DMV給辦事人員挑選。此外，兩份居住證明地址要完全一致。

    換照流程 #

    1. 前往監理所（DMV），附近僅Downtown可以申辦。建議開門前就去排隊。
    2. 臨櫃時進行視力檢驗（測試矯正後視力）。檢測是否看到兩側閃爍點。
    3. 現場攝製證件照、簽名、繳交規費（2019年起接受刷卡）。
    4. 現場提供臨時駕照（效期15天，功能等同一般駕照），5~7天寄發正式駕照。

    駕照譯本不在系統
    駐紐約辦事處須將譯本email至賓州政府的專用信箱。除了可能是辦事人員不熟悉系統操作外，有時是駐紐約辦事處漏（晚）上傳或是州政府收信人員的遺漏以至於DMV查不到該譯本的有效性。發生時應聯絡駐紐約辦事處通知補上傳資料，可致電212-317-7300，如果打不通可嘗試領務專用電話212-486-0088。辦事處資料重新上傳後約48~72小時後資料會進入DMV系統，因此約3天後再去DMV辦理。若是仍找無資料，可能需要去DMV多試幾次或再要求駐紐約辦事處再寄一次信。

    尚未拿到SSN或非標準的Denial Letter
    Denial Letter原本是指Ineligibility for SSN（參照PUB 195NC）。然而疫情期間，很多人收到的回信並非是真正的Denial Letter，信中僅表明雖無法頒發SSN但也無法證明沒有資格申請SSN（如下圖）。實際上，Photo ID的申請文件上（DL-160 Form）僅要求Letter from SSA inticating that SSA did not make decision yet or SSA rejection Letter。因此，除了拿到這類Denial Letter的人外，已申辦SSN但尚未SSN的人也可以持申辦SSN的收據向DMV申請駕照。

    考取賓州駕照 #

    如果沒有台灣駕照，想直接考賓州駕照，考試分為兩部份：筆試（Knowledge Test）和路考（Road Test），詳情參考連結：

    1. 下載賓州駕駛員手冊閱讀。
    2. 完成Form DL-180（體檢表）。CMU可在University Health Service完成；Pitt在Wellness Center（非免費）。
    3. 攜帶Form DL-180及所須文件（同免試申換賓州駕照，排除台灣駕照相關文件）參加筆試，通過後可以得到Learner Permit。有了Learner Permit，就可以在有駕駛執照的人陪同下開車。
    4. 預約路考（線上或電話預約）。由於現場不提供車，是必須要有陪同者一起參加路考。車輛須提供保險證明、車輛登記證明、陪同者的有效美國駕照。由於租車公司通常不會提供，因此通常須要跟有認識的人借車。

    申辦State ID #

    所須文件與考取賓州駕照類似，不須Form DL-180。

    申換REAL ID #

    在申辦駕照或State ID時，如果沒有主動說要申辦REAL ID版的駕照或State ID，承辦人員是不會主動辦理的。

    申換REAL ID所須文件，詳情參閱官方網站：

    • 護照及美國簽證
    • SSN或SSN denial letter
    • 兩份居住證明（未過期駕照、汽車登記證、汽車保險卡、水電費帳單等）
