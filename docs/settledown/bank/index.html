    銀行開戶 #

    楊博元協編
    最後更新日期2021.06.05

    由於疫情因素，目前需要線上或打電話到該分行與Banker預約開戶。

    帳戶種類介紹 #

    所有帳戶都由兩組號碼構成：

    1. Routing Number：9位號碼，銀行代號，同一州的分行號碼一樣。
    2. Account Number：10~12位號碼，個人的帳戶號碼。

    美國帳戶主要可分為兩種：

    1. 支票賬戶（Checking Account）：主要用於提款以及與簽帳金融卡（Debit Card）、個人支票綁定扣款的帳戶。無利息或極低利率，基本可忽略。
    2. 儲蓄賬戶（Saving Account）：主要用來存錢，每月有限定的轉帳次數（約6次，超過要罰錢），有利息但利率極低基本可忽略。Debit Card或個人支票不使用此帳戶，且支票帳戶錢用完不會主動從儲蓄帳戶提錢，具有較高的安全性，可將比較大的金額存放在此。

    推薦銀行列表 #

    Pitt與CMU鄰近常見公車路線與銀行分佈

    Bank of America #


    美國銀行（Bank of America, BoA）： 2019年起在匹茲堡展設據點，目前據點偏少。

    • Advantage SafteBalance Banking (Checking account)管理費$4.95/月，24歲以下學生免管理費。
    • Advantage Plus Banking (Checking account)管理費$12/月，帳戶維持$1,500或24歲以下學生免管理費。
    • 由於開戶後，不論有沒有SSN或信用紀錄都能直接在Branch申請他們的信用卡，因此是目前相當多人的選擇，詳情參照信用卡章節。

    Chase #


    大通銀行（Chase）：2019年起在匹茲堡展設據點，目前據點偏少。

    • 學生帳戶限於25歲以下，畢業前免管理費最多5年。開戶獎勵$100。
    • 一般帳戶的Checking Account和Saving Account均有提供$100~$300開戶獎勵。基礎的Checking Account維持$1,500即免除管理費。
    • 開戶對於未來申請Chase信用卡有幫助。Chase的信用卡相當優秀，但幾乎需要較好、較長的信用紀錄。
    Chase開戶獎勵
    Checking Account的開戶獎勵條件通常要求有收入（payroll）轉入；Saving Account的開戶獎勵條件通常要求存入一定金額一段時間（例如存入$15,000維持90天）。需要Coupon且滿足開戶條件，網上可搜到或找他人Referral，開戶獎勵隨時間更動。

    PNC #


    PNC金融服務集團（PNC）： 匹茲堡本地發跡的銀行，分行據點以及在校園附近的ATM數量皆佔有絕對優勢。

    • 帳戶分為Spend、Reserve、Growth。Spend、Growth分別對應Checking、Saving，Reserve Account功能介於Checking與Saving之間，沒提款次數限制，但也不能直接被花用。
    • 任何學生都可以開學生帳戶，免管理費最多5年，Spend Account每學期免一次（國際）電匯手續費。

    Citibank #


    花旗銀行（Citibank）：匹茲堡地區沒有分行，須至鄰近的紐約、芝加哥、華盛頓特區開戶。

    • 有提供開戶獎勵。
    • 可以使用花旗全球速匯，參見國際匯款章節。2021年4月，花旗宣布出售台灣等多國消費金融，因此花旗全球速匯未來可能不適用。

    開戶流程 #

    1. 準備開戶材料

      • 包含美國簽證的護照
      • I-20
      • I94
      • 住址、E-mail及聯絡電話
      • 學生ID（沒有不影響開戶）
      • Social Security Number（沒有不影響開戶，也可事後補上）
      • 部分銀行（如花旗）要求第二證件（如台灣駕照）

    2. 填寫Form W-8 BEN
      非居民外籍人士（Nonresident Alien）開戶時必須填寫Form W-8 BEN；美國公民或居民外國人是填寫W-8。有時需要提醒Banker不要給錯表單。非居民外籍人士定義參照稅務章節。

    3. 領取Debit Card
      Debit Card約開戶一週後會寄至填寫地址。可以網路或線上開通。

      • Chase：PIN碼隨卡寄到住址。
      • PNC：現場填寫PIN碼。

    4. 辦理網路銀行
      開啟網路銀行功能，按照銀行E-mail或簡訊的指示並下載該銀行App。

      • Chase：需向Banker索取網銀帳密。
      • PNC：約開戶一周後將認證碼寄到地址。

    開戶注意事項 #

    • 務必記住自己的簽名樣式，會影響日後支票兌現。
    • 銀行會在開戶後提供幾張支票。若要整本支票的話，需要付款購買。
    • 建議向協助開戶的Banker索取名片，若網路銀行設定或有其他問題，直接致電處理通常較為迅速。
    • 美國以刷卡為主、很少用到現金，加上網路銀行方便（可手機存小額支票），因此並不常須用使用ATM或實體銀行。

    常見問題 #

    Checking Account或Saving Account到底要開哪個? Checking Account是一定要開，有Checking Account才會有Debit Card或使用支票。Saving Account屬於安全帳戶，一般人沒有辦法直接使用到這帳戶的金錢，如果有大筆暫時不會用到的金額可以考慮開。

    Debit Card卡號與Account Number是否是同一組號碼? Debit Card卡號與Account Number不一樣。Debit Card卡號是在進行網購的時候使用，大多數通過 Visa或Mastercard進行付款。如果卡片丟失，損毀，或者被盜刷，向銀行申請補辦或換卡，所拿到的新卡卡號會與舊卡不同。Account Number主要出現於支票或提供給公司做薪資發放用。

    Debit Card的PIN碼是做什麼的? PIN碼是提款或刷用卡片時所需要的密碼， 如被盜刷有通過輸入PIN碼， 銀行不會負責。 因此在輸入PIN時，需要小心被偷看。

    銀行帳戶是否有管理費? 除學生帳戶外，大部分美國的銀行若未滿足存放最低額度或有Direct Deposit（薪資入帳），每月會收取管理費。務必向行員問清楚各種權利義務如最低存款金額等。如果都沒存款，可能會被銀行關戶。

    是否能線上開戶? 非居民外籍人士僅能現場開戶，美國公民與居民外籍人士可以線上開戶。

    開銀行戶需不需要SSN? SSN不是銀行的必要項目，但如果有的話建議提供以及補登。
