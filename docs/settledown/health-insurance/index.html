    投保健康保險 #

    最後更新日期2022.12.20

    健康保險（Health Insurance）可分為醫療保險（Medical Plan）、牙科保險（Dental Plan）、眼科保險（Vision Plan）。國際生強制購買醫療保險，但不強制購買學校方案。

    健康保險術語、類型等基本介紹，參照健康保險攻略章節。學校健保保險內介紹，參照學生健康保險章節。

    常見問題 #

    學校是否有提供半年保險？ CMU、Pitt有提供春季入學半年保險，但並未提供秋季學期的半年保險。校外保險則不一定。畢業時可以退保，詳情參考畢業退保章節。

    學校保險的優點與缺點？ 優點：通常較為全面。低Deductible、Out-of-pocket Maximum，高Coinsurance。承保原有疾病，無Waiting Period。
    缺點：承保內容固定、昂貴、起保日期固定、投保長度固定。
