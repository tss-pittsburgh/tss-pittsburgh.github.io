    徵才/內推 #


    內推是一個互惠機制。被內推的人通常可以跳過第一關履歷的AI審核，運氣更好的可能直接履歷送到主管手中。如果被內推的人錄取並入職了，推薦人通常都可以獲得一筆可觀的獎金。此外，越多台灣人在一間公司，就越有機會互相提攜拉拔！

    如果想要提供職缺機會或長期內推的人，可以到左側選單中「意見回饋/徵才內推」填寫表單。

    聯絡內推注意事項
    簡單介紹自己的背景、學校、工作經驗以及想要內推的職缺全名及連結等。可以在Email或傳訊息註明是透過TSS網站得到的內推資訊。履歷、第三人稱自我介紹可以等連絡上推薦人再補傳給對方。
    不要急著寄履歷求內推
    建議先跟推薦人聯絡上後再寄送履歷及其他相關資料。由於許多公司會凍結重複申請同一職位的人，如果逕行寄送履歷給推薦人又聯絡不上推薦人時，就有可能導致未來重複申請。此外，務必確認是否已經投了相同職位。

    職缺 #

    如果職缺已失效，煩請到左側選單中「意見回饋/徵才內推」填寫表單告知。
