    內推平台 #


    內推是一個互惠機制。被內推的人通常可以跳過第一關履歷的AI審核，運氣更好的可能直接履歷送到主管手中。這裡作為一個共享平台，讓即將找工作的學弟妹有資源尋覓到適合的內推職位。往後越多台灣人在一間公司，就越有機會合作與互相提攜拉拔！如果想要提供職缺機會或長期內推的人（重複公司亦歡迎），可到左側選單中「意見回饋/徵才內推」填寫表單。

    內推表單 #

    基於隱私因素，內推人的聯絡方式將放於Google表單，其中Pitt與CMU表單的內容相同，校外表單的內容以內推人同意分享為限，為保護內推人隱私，請勿任意分享表單內容。

    Pitt在校生及校友可使用xxx@pitt.edu登入閱讀。

    • 校外表單
    聯絡內推注意事項
    簡單介紹自己的背景、學校、工作經驗以及想要內推的職缺全名及連結等。可以在Email或傳訊息註明是透過TSS網站得到的內推資訊。履歷、第三人稱自我介紹可以等連絡上推薦人再補傳給對方。

    不要急著寄履歷求內推
    建議先跟推薦人聯絡上後再寄送履歷及其他相關資料。由於許多公司會凍結重複申請同一職位的人，如果逕行寄送履歷給推薦人又聯絡不上推薦人時，就有可能導致未來重複申請。此外，務必確認是否已經投了相同職位。

    心懷感恩
    雖然推薦人通常都可因為內推的人入職而獲得一筆的獎金，但比起所付出的心力與投資報酬率，大多數學長姐都是出於熱心提攜後輩的。現在的內推系統所詢問的問題越來越詳盡，可能從要知道你和被內推人的關係到你知道被內推人有多少專業技能。往往需要耗時溝通、系統填表、追蹤面試進展、是否錄取、到最終入職才能修成正果。因此，推薦人看了你提供的資料後，可能因為覺得履歷經驗不足或是同個職缺已經內推了很多人而拒絕你的內推請求。不要氣餒，可以誠懇向推薦人尋求建議，也是相當有收穫的。
