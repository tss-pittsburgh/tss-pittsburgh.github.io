    工作 #

    最後更新日期2020.08.04

    在美國申請工作時，有合法的工作身分是很重要的，本節主要簡介F1簽證的工作樣態。獲取工作後，原則上可以申請SSN，詳情參照申請SSN章節。

    非法打工
    未經授權即在校外非法打工，一經發現，遭到遣返的可能性極高，切勿以身試罰。

    校內工作 #

    持F1簽證的人可在校內自由合法打工，不須Designated School Official（DSO）或美國公民及移民服務局（USCIS）批准。學期中一週不能超過20小時。有Funding的PhD學生，一般即是此類。如果想自行尋找校內工作，CMU學生可在Handshake、Pitt學生可在Talent Center搜尋工作。一般教學助理或研究型工作也可直接聯絡教授詢問。此外，在校內獲取工作一段時間後，是有機會獲得學費減免及學校保險補助的。

    CPT / OPT #

    Curriculum Practical Training (CPT)：給在讀F1學生的實習許可，通常須入學滿一年才能申請。CPT必須是完成學位大綱中要求必可不少的一部分並且工作與所學專業緊密相關。CPT須向學校DSO遞交申請，獲得經DSO批准的I-20後可以開始CPT工作。申請流程每年、每個學院甚至系所規定不一，請參照My OIS, Pitt或OIE, CMU查詢。

    Optional Practical Training (OPT)：給F1學生在其所學領域相關的實習工作許可。符合條件的學生可以申請長達12個月的OPT。OPT須向USCIS遞交申請，收到EAD卡後才能開始OPT工作。申請流程、表單每年均有微調變動，請參照My OIS, Pitt或OIE, CMU查詢。

    STEM OPT Extension：在STEM1領域獲得學位的F1國際學生，可以申請為期24個月的OPT延期。STEM OPT延期必須向USCIS申請，會收到新EAD卡。申請流程請參照My OIS, Pitt或OIE, CMU查詢。

    H1B CAP-GAP：在4月1日遞交H-1B申請的F1學生，若OPT在H-1B生效日（即10月1日）之前結束，則工作身份自動延長到9月30日，成為H1B Cap-Gap。H1B Cap-Gap須向DSO匯報並獲得新的I-20表，使得繼續合法工作。申請流程請參照My OIS, Pitt或OIE, CMU查詢。

    註1：STEM全稱為Science, Technology, Engineering, and Mathematics。

    常見問題 #

    如果來不及獲得SSN是否可以開始工作？ 根據美國社會安全局（SSA)和美國移民局（USCIS)說法，學生不需要SSN即可開始工作。 確認雇主的內部政策，如果遇到困難可向雇主提供美國聯邦指南。
