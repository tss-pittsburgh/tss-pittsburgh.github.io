    疫情生活指南 #

    最後更新日期2021.04.16

    2019年爆發新型冠狀病毒（COVID-19），各國、各州間均有不同的旅行禁令、檢疫要求，最新要求務必參照官方指示。

    入境美國與賓州旅行要求 #

    美國目前對台灣沒有旅行禁令，但自2021.01.26起登機前須檢具航班起飛前3日內核酸增幅檢測或抗源檢測報告（紙本或電子影本）或已感染且康復之證明文件，詳情參照美國CDC網站。

    自2021.03.01起，賓州政府取消州外旅遊限制、隔離要求。

    返台核酸檢測 #

    啟程地時間自2020.12.01起，入境台灣或於台灣機場轉機之旅客登機前均應檢附「表訂登機時間前3日（工作日）內COVID-19核酸檢驗陰性報告（紙本或電子影本）， 檢驗方法需採用分子生物學核酸檢測（例如：PCR、RT-PCR、NAA、NAT等），血清免疫學 （Immunoserology）檢驗抗原（Ag）或抗體（IgG／IgM）不符合本措施規範。 詳情參閱衛服部或駐紐約辦事處網站。

    台灣定義的3日內
    「3日內」檢驗報告日期是以「出報告日」及「工作日」計算，排除當地政府之國定假日。

    常見核酸檢測機構：

    • Pixel by LabCorp：有保險免費。檢測包寄至住處，自行採樣由FedEx寄回。
    • Curative：是否有保險均免費，沒保險的話檢查費用是由政府吸收。需填寫保險資料，未填則仍會被查詢保險紀錄。以廂型車作為檢查站，專人檢測。
    • CMU Tartan Testing：CMU教職員免費，每週可檢測1次。校內檢測站自行使用檢測包。
    • Pitt Quest Testing：Pitt學生於特定時間免費。檢測包寄至住處，自行採樣由FedEx寄回。
    • MedExpress：有保險免費。
    • CVS：費用$139，原則上2~3個工作天。CVS檢查站，在指導下自行使用檢測包。

    免費檢測頻率
    保險公司通常只要有醫囑就會支付檢測費用（免費的檢測通常是機構委請醫師下醫囑），但多短時間內會支付第二次費用仍是未知。

    避免Rapid Test
    CVS當日可獲得測試結果的Rapid Test屬於抗原檢測，不能作為登機前的檢驗報告
