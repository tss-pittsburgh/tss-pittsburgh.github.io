    疫情生活指南 #

    最後更新日期2021.06.04

    2019年爆發新型冠狀病毒（COVID-19），各國、各州間均有不同的旅行禁令、檢疫要求，最新要求務必參照官方指示。

    入境美國與賓州旅行要求 #

    • 美國目前對台灣沒有旅行禁令。
    • 抵美前14天若曾至（包含轉機或過境）中國大陸（除香港、澳門）、申根地區、英國、印度等外籍旅客禁止入境，詳情參閱美國CDC網站。
      • 美國公民、美國公民、永久居民以及其配偶、未成年眷屬不在此限），詳情參閱總統令。
      • 自2021.08.01，F1、M1、J1簽證不在此限（並未包含眷屬），詳情參閱總統令。
    • 自2021.01.26起登機前須檢具航班起飛前3日內核酸增幅檢測或抗源檢測報告（紙本或電子影本）或3個月內已感染且康復之證明文件，詳情參照美國CDC網站。
    • 自2021.03.01起，賓州政府取消州外旅遊限制、隔離要求。
    • 自2021.05.19起，已完全接種疫苗的人在美國境內旅遊不需要額外篩檢或隔離。

    COVID-19疫苗接種 #

    目前全美不論有沒有保險，任何人都可以「免費」施打COVID-19疫苗。目前美國FDA緊急授權的疫苗有Pfizer（輝瑞，2劑，每劑間隔至少21天，12歲以上）、 Moderna（莫德納，2劑，每劑間隔至少28天，18歲以上)、Johnson & Johnson（嬌生，1劑，18歲以上）。

    如何預約：線上預約為主，且方便。電話預約亦可。少數提供Walk-in。施打地點可能是醫院、藥局、體育中心或是賣場等。預約時就會告知這個地點會提供什麼疫苗牌子，基本上只要有提供都可以自由選擇。

    • CDC VaccineFinder
    • Allegheny County Vaccine Registration
    • Rite Aid
    • Giant Eagle Pharmancy（可以Walk-in）
    • CVS（可以Walk-in）
    • Costco

    所需文件：Photo ID（如護照或美國駕照）、保險卡（如果有保險就提供，沒有也非必要）、COVID-19疫苗卡（施打第二劑用）

    返台核酸檢測 #

    啟程地時間自2020.12.01起，入境台灣或於台灣機場轉機之旅客登機前均應檢附「表訂登機時間前3日（工作日）內COVID-19核酸檢驗陰性報告（紙本或電子影本）， 檢驗方法需採用分子生物學核酸檢測（例如：PCR、RT-PCR、NAA、NAT等），血清免疫學 （Immunoserology）檢驗抗原（Ag）或抗體（IgG／IgM）不符合本措施規範。 詳情參閱衛服部或駐紐約辦事處網站。

    台灣定義的3日內
    「3日內」檢驗報告日期是以「出報告日」及「工作日」計算，排除當地政府之國定假日。

    常見核酸檢測機構：

    • Pixel by LabCorp：有保險免費。檢測包寄至住處，自行採樣由FedEx寄回。
    • Curative：是否有保險均免費，沒保險的話檢查費用是由政府吸收。需填寫保險資料，未填則仍會被查詢保險紀錄。以廂型車作為檢查站，專人檢測。
    • CMU Tartan Testing：CMU教職員免費，每週可檢測1次。校內檢測站自行使用檢測包。
    • MedExpress：有保險免費。
    • CVS：費用$139，原則上2~3個工作天。CVS檢查站，在指導下自行使用檢測包。

    免費檢測頻率
    保險公司通常只要有醫囑就會支付檢測費用（免費的檢測通常是機構委請醫師下醫囑），但多短時間內會支付第二次費用仍是未知。

    避免Rapid Test
    CVS當日可獲得測試結果的Rapid Test屬於抗原檢測，不能作為登機前的檢驗報告
