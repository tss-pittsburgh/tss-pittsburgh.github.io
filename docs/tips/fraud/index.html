    詐騙防治 #

    劉又齊協編
    最後更新日期2020.08.04

    詐騙電話和電子郵件很常見，國際學生容易因為在等包裹或者正在處理F-1 / J-1、OPT等等相關事務就掉以輕心，以下是一些常見的詐騙手法：

    • 紐約辦事處或中國領事館有你的文件。
    • 紐約辦事處或中國領事館說你的銀行帳戶涉及洗錢。
    • 紐約辦事處說你的護照被盜用並且提供你上次入境的地點和日期或轉機的地點。
    • 美國公民及移民服務局（USCIS）或美國移民及海關執法局（ICE）說你違反移民法，有你的搜捕令，你需要匯錢請律師處理你的案件。
    • 國際學生要回報SSN。
    • 國稅局（IRS）要退稅給你或通知你有優惠學貸。
    • 美國公民及移民服務局（USCIS）或美國移民及海關執法局（ICE）說你未更新你的I-94。
    • 國土安全部公民與移民服務（USCIS）說你參與了犯罪組織或恐怖組織。
    • 水電公司、汽車保險等等單位說你欠款。
    • 人口普查。但人口普查調查員不會要求您提供公民身分、駕照、社會安全號碼、信用卡、銀行或其他財務資訊、金錢或捐贈、或PIN碼和密碼。

    以上都極為可能是詐騙。請不要經由電話或電子郵件提供你的移民身分、駕照、社會安全號碼、信用卡、銀行或其他財務資訊、金錢或捐贈、或PIN碼和密碼。
