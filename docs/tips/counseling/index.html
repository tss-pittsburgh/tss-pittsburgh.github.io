    心理諮商 #

    江孟純協編
    最後更新日期2021.01.23

    不論疫情封城或是學業壓力，身處他鄉求學往往會遇到不順心的事情。如果對生活持續感到不滿意、情緒感到不舒服，時間或朋友也無法解決時，尋求專業協助對生活與學業都會相當有幫助的。心理保健跟身體保健是一樣重要的，平常的時候就要保養求助。如果一小點點憂鬱的念頭持續了一陣子，就可以打電話來預約了。早期諮商，早期解決。如果完全沒有病史，建議優先找諮商師會談，避免直接尋求藥物控制。

    醫療保密：基於健康保險便利和責任法案(HIPAA)，無論是家人、朋友、學校都不能獲取你的醫療信息。因此無須擔心簽證、校園生活會因為心裡諮商或治療受到任何影響。

    職業名稱： 美國的系統之下，有不同證照的差異，包含Licensed psychologist, Licensed professional counselor, Licensed social worker，但是對於一般諮商需求來說並無差異，以下統稱諮商師。通常諮商師和心理治療師是不開藥的，以臨床會談為主。如果有使用藥物的需求則會轉診精神科醫師。

    校內諮商 #

    Pitt和CMU均有提供免費的諮商服務，主要屬於短期諮商的資源，透過24/7的電話預約（Pitt: 412-648-7930, CMU: 412-268-2922）。學校會在會談中提供尋找校外長期諮商的建議。

    流程：約診時會透過電話問些基本的問題，確認是否有急切的需求。以CMU為例，如果非急迫需求，第一次會談會在約5個工作天內；如果有較為急迫的諮商需求，會有諮商師馬上洽談；有更為緊急的情況，學校會派救護車協助。

    時間：一次諮商診療約50分鐘。

    校外諮商 #

    如果有過諮商經驗或想要建立長期諮商關係，校外諮商會是較佳的選擇，無須經由校內諮商轉介。

    流程：可以從Psychology Today與Thriving Campus尋找諮商師。諮商師的談話方式、風格皆有不同，因此建議可以多找幾個會談以尋找適合自己的諮商師 如果覺得初期會談發現與諮商師溝通效果不佳，可以更換諮商師。建立長期諮商關係後，如有需求會提供緊急約診的服務。

    費用：依保險不同。每次會談Pitt Graduate Student Medical Plan須負擔$5 copayment，CMU Aetna保險完全免費，兩者皆無使用次數限制。

    時間：一次諮商診療約50分鐘。通常一個星期1次，視情況需求，一個星期2~3次亦可。

    轉診：可轉移診療紀錄至其他諮商師。如果僅是在選諮商師階段，通常不需要轉移診療紀錄。
