    小費文化 #

    最後更新日期2021.05.24

    美國的小費文化是相當獨特的，美國的許多餐廳或其他服務業都用最低薪水來雇用服務生，因此對於服務人員來說小費是他們重要的收入來源。因此，大部分的「服務」都不成文規定地收取小費，學習怎麼給小費也是一種入境隨俗的表現。

    何時該給小費 #

    基本概念就是享受專屬服務時，就必須付出相對應的報酬，如餐廳吃飯倒水的服務生、酒吧裡的酒保等。此外，紐約對於小費文化在各個層面涉及更廣，屬於特殊區域。惟須注意，移民官、USPS郵差等公共雇員受法律約束是不能接受小費。

    該給多少小費 #

    絕大多數情況下，給稅前費用的15%（或$1以上）是普通標準，給10%或以下代表不滿意服務，給20%或以上則表示很滿意。下面列舉的金額或稅前價格的額外百分比僅供參考，可依據當下情況。

    餐廳 #

    • 一般餐廳：通常早、午餐為稅前15%，晚餐、假日為稅前18~20%。通常10%代表服務很不及格。通常特定餐廳或是多人聚餐會酌收18~20%的固定服務費，稱為Gratuity Included。
    • 酒吧：每杯飲料$1~$2或是按照一般餐廳百分比辦理。
    • 速食餐廳：通常不須額外付小費。如果用現金付款，有些人可能會跟店員說Keep the change（即把零錢留著吧！）當成小費。櫃檯前方偶爾會放投錢箱，如果很滿意店員服務，也可以投個$1~$2。
    • 手搖飲料：通常不須給小費。
    • 外帶：通常不須給小費。

    住宿 #

    • 行李搬運：每件行李$1~$2。
    • 客房清潔（鋪床、打掃房內）：依照飯店等級每次$1~$5。可以放在枕頭上或要在小費旁留張紙條，隨意亂放可能會被認為是私人財物。
    • 客房服務：一般情況索取毛巾、刮鬍刀等備品通常不須要。客房餐飲可給10~15%。

    交通 #

    • 叫車：計程車為15%。由於Uber、Lyft等叫車服務都是不是當場給付，因此很多人沒有給小費的習慣。
    • 外送：依距離或天氣給予$1~$3或10~20%。
    • 泊車人員：領車時給$2~$5。
    • 飯店的免費接駁車：一般不需要，如果服務員有幫忙上下行李，每件行李$1~$2。

    其他 #

    • 理髮：10~20%。
    • 擦鞋：$2~$3。
    • 導覽解說：如果是非聯邦公共雇員，可給予$5~$20。
    • 搬家、送貨上門（大型家具）工人：每人$3~$5。
    • 租屋處管理員：在匹茲堡沒有給管理員小費的傳統，而紐約有於節慶給小費的傳統。

    如何給予小費 #

    • 如果在餐廳以刷卡消費，可直接在刷卡單填寫金額並簽名放置桌上離去。
    • 如果在餐廳以現金消費，通常不會直接把小費遞給服務生或酒保，可選擇不用找零或把小費留在桌上或吧台上後離開。
    • 其他服務多是直接給現金。

    可否不給小費 #

    並沒有法律強制規定給小費，但如果沒給或給得過低可能會被服務生追問。如果實在不滿意服務生的服務時，苛扣小費並非良好的表達方式，而是給予正常的小費並直接向餐廳經理投訴。記住，給不給或給多少小費最終決定權還是在自身上，但如果將來還會來同樣餐廳吃飯，最好不要留下壞印象。
