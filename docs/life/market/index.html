    超市 #

    蔡思亭協編
    最後更新日期2021.09.08

    本篇超市泛指雜貨店（Grocery）與超級超市（Supermarket），另外匹茲堡附近仍有許多大賣場（Wholesale）以及藥妝店（Drugstore）。

    超市分佈 #


    點擊左上角可看到明細與分類選。
    ●：美國超市
    ●：亞洲超市
    ●：大賣場
    ●：零售/藥妝

    超市外送服務 #

    疫情期間，超市採購有不小的轉變，許多店家提供生鮮外送服務，以下主要為提供生鮮、雜貨外送服務的商家：

    百合超市 #


    匹茲堡當地中超，滿$30免運，可選擇配送時間，通常當日或隔日配送名額額滿。

    Whole Foods #


    Amazon Prime會員專屬服務，滿$35免運或是店內Pick up。通常為隔日送達。可於Amazon Hub Locker或Whole Foods店內退貨。

    Costco #


    提供Two-Day和Same-Day Delivery，Two-Day Delivery由Costco自行運營，滿$75免運；Same-Day是與Instacart合作，滿$35免運但酌收10%服務費。新鮮肉類、果蔬和海鮮等食品都可以當天送到。需要美國版Costco卡才行使用該服務。

    Instacart #



    有多家店舖選擇，可自行選擇配送時間以及配送方式。當日送達，滿$35非會員運費$3.99、會員免運，會員年費為$99。超過50lbs會有heavy fee。

    FreshGoGo #



    提供多樣的亞洲生鮮食材，滿$49免服務費，滿$99免運。每週六送貨。

    亞米網 #



    北美亞洲商品購物平台，目前尚未提供生鮮運送至匹茲堡。滿$49免運，配送時間約為1~2日，高峰期為3~4日。

    Walmart #



    提供Free Grocery Pickup和Free 2-Day Delivery，滿$35免運。會員則沒有最低消費要求，會員年費為$98。
