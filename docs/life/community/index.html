    匹茲堡台灣社群與社團 #

    劉又齊協編
    最後更新日期2021.05.24

    交流型社團 #

    全美台灣同鄉會匹茲堡分會 #

    全美台灣同鄉會匹茲堡分會（Taiwanese Association of America - Pittsburgh Chapter, FB社團、網站），簡稱匹茲堡台灣同鄉會，成立於1969年，是美國最大最久的台灣人社團-全美台灣同鄉會(TAA)的匹茲堡分會。宗旨是促進鄉親之情誼、互助和合作，關懷台灣社會，宣揚台灣文化以及促進台美文化交流。會員分佈從50、60第一代留學生到在學留學生，是一個以台灣人及台美人為榮、世代融合的大家庭。每年在農曆新年、母親節及中秋佳節等舉辦室內、外活動，有時也協助接待、安排台灣的文化及學校團體來表演或參加競賽。同鄉會有合唱團、並和 TSS in Pittsburgh及哲學星期五＠匹茲堡有很好的互動。如需急難救助，也可以聯絡台灣同鄉會。

    持會員卡至下列超市及餐廳享有5%優惠：百合 Lotus、佳惠 Oriental Market、好多多 Many More Asian Market、永發行 WFH Oriental Market、婷園 Ting’s Kitchen、天天見面 Everyday Noodles。

    大匹茲堡地區台灣人 #

    大匹茲堡台灣人（Taiwanese and Taiwanese Americans in greater Pittsburgh, FB社團）成立目的是給大匹茲堡區及周邊區域的台灣人交流生活資訊，舉凡食衣住行、生活瑣事、求職上的疑問，都歡迎提出討論。社團內有很多資深匹茲堡台灣人。

    匹茲堡台灣媽媽交流 #

    匹茲堡台灣媽媽交流（Pittsburgh Taiwanese Moms, FB社團）是住在匹茲堡地區的台灣家長們交流的地方，攜家帶眷來念書的同學可以加入這個社團。不管是生活、育兒、購物、找playdate、或是遇到的問題，都可以在這裡分享！常有台灣媽媽們分享照顧小孩的經驗、生活、學區兒童活動相關的討論或免費或低價的二手小物。

    學術型社團 #

    哲學星期五＠匹茲堡 #

    哲學星期五（FB社團）不定期邀請來自不同領域的專家帶領討論，例如：婚姻平權、牙齒保健、台灣文學、種族正義、勞工權益等。除了能聆聽專家演講之外，哲五也提供了讓大家互相認識的機會，是個交朋友的好地方！歡迎大家來當志工或是講者，有興趣的朋友歡迎臉書傳訊息與哲學星期五聯絡。

    Pittsburgh Taiwanese Pride for All #

    這是個台灣人和台美人以及其他支持平權、公平和 PRIDE for ALL的團體（FB社團）。每年六月「驕傲月」，匹茲堡台灣人會組成一小隊一起加入遊行，請大家密切關注！

    運動型社團 #

    匹茲堡台灣同鄉會壘球社 #

    匹茲堡台灣同鄉會壘球社（Pittsburgh Taiwanese Association Softball Club, FB社團）已經在匹茲堡地區成立多年，主要的目的是邀請大家一起來打球，男女老少不拘、參加者也不一定要只是台灣人，只要想打球就可以加入。過去幾年，壘球社有時會跟匹茲堡台灣學生會的壘球隊進行友誼賽，也會參加匹茲堡地區業餘的壘球比賽，打球之餘，也會出去吃吃喝喝或是進行戶外活動。

    匹茲堡排球聯盟 #

    每週六或週日早上打球，打完球去喝珍奶+吃飯（FB社團）。

    其他 #

    壘球、籃球、網球、棒球。詳情關注新生說明會。
