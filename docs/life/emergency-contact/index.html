    急難救助 #

    最後更新日期2022.04.17

    旅外國人、度假打工青年、留學生及臺灣僑胞於遭遇緊急危難事件時可透過以下方式獲取必要的關懷與救助。

    美國報警電話 #

    在緊急情況下，優先立即撥打 911 或當地的緊急電話號碼。

    緊急情況是指需要警察、消防部門或救護車立即協助的任何情況，包括：

    • 火災
    • 正在發生或已經發生的犯罪事件
    • 車禍，特別是有人受傷時
    • 醫療緊急情況，特別對於需要立即就醫的症狀

    當撥打 911 時，請準備好回答接聽者的問題，其中可能包括：

    • 緊急情況的位置，包括街道地址
    • 撥打的電話號碼
    • 緊急情況的性質
    • 有關緊急情況的詳細信息，例如對可能犯罪的人的身體描述、對任何可能正在燃燒的火災的描述，或者對有醫療緊急情況的人所經歷的傷害或症狀的描述

    簡訊911
    當無法撥打 911 時，可以使用簡訊服務至911。惟須注意以下幾點

    • 不要超過160個characters
    • 第一條簡訊應包含「緊急情況發生的地點」、「需要的幫助類型：消防、救護車、警察等」

    學校 #

    學校單位主要負責發生在校園事件為主，校外事件需撥打911。

    匹茲堡臺灣人急難救助協會 #

    為響應僑委會建構全球急難救助網的呼籲，匹茲堡臺灣鄕親在熱心鄉親及友人們的贊助下成立了急難救助協會，以便及時協助政府駐外單位，為旅外國人、度假打工青年、留學生及臺灣僑胞(甫至本地區工作、經商或依親之臺僑為主)於遭遇緊急危難事件時提供必要的關懷與救助。志工成員包含內、外科醫師、慈濟團隊等。

    主要聯絡人：

    • 理事長：郭向榮
    • 副理事長：林敏璋
    • 聯絡電話：412-475-8220
    • Email： pteaa412@gmail.com
