    郵寄/收件 #

    最後更新日期2020.09.20

    在美國，USPS是公家的郵政機構，UPS與FedEx則是主流的商業快遞公司。USPS在美國境內運輸仍是相當可靠、有效的。一般來說，商業快遞公司的售後服務會比USPS好。

    USPS #



    USPS全稱United States Postal Service，即美國郵局，是聯邦政府旗下的機構。普通信件寄送、郵筒、郵票、選票寄送都由USPS管轄，部分USPS也會為美國公民辦理護照。除了平日營業至17:00，部分的USPS辦公室週六早上至13:00仍有營業。以下郵資、費用僅作參考(2021年9月）。

    郵寄文件至政府機關推薦使用USPS
    部分政府機關僅能透過USPS郵寄（例如日本大使館）、回信（例如駐紐約辦事處）。此外，例如報稅或寄信至SSA，使用USPS可直接投遞到專用辦公室，一般商業快遞公司須投到限定的地址。

    美國境內郵寄 #

    包裹郵寄均有限重70 lbs、「最大邊長」和「另兩邊周長」的總和不得超過108"的限制。

    • First Class Mail：即普通郵件，最便宜的郵寄方式，依照郵寄物有不同價格。約1~3工作天送達。限重13oz以下。本身不附帶何追踪或保障，其他附加服務需要額外付費，例如Certified Mail（即掛號）、$5000以內的保險（限商品）。

      • 明信片（Postcard）：$0.4起
      • 信件（Letter）：$0.58（標準方形）、$0.78（其他形狀）起，限重3.5oz
      • 大信封（Large Envelope）：$1.16起
      • 小包裹（Small Package）：$4起

    • Priority Mail：即普通快遞，郵資$7.7起。約1~3工作天送達，週六也遞送。。附帶追蹤（USPS Tracking®）以及$50的保險（可額外購買超額保險）。部分物品不得運送，詳情參考連結 。

      郵資計算分為兩種：

      • 按重量、區域，可透過線上工具預估費用。
      • Flat Rate：使用USPS專用信封/紙箱，不論內容物郵資固定，詳情參考連結。

    • Priority Mail Express：USPS境內最快捷的郵寄服務，約1~2工作天送達，365天服務、隔夜投遞，超過2天退款。附帶追蹤、$100的保險（可額外購買超額保險）、遞送證明簽名記錄（寄送時要提出）。

      郵資計算分為兩種：

      • 按重量、區域，可透過線上工具預估價格。
      • Flat Rate：使用USPS專用信封/紙箱，不論內容物郵資固定，詳情參考連結。
    • 其他服務：詳情參照連結。
      • Certified Mail：可以追蹤信件、確認收件人是否簽收，費用為$3.75。
      • Return Receipt：可拿到收件人簽名的電子或紙本收據，費用為$1.85（電子）、$3.05（紙本）。使用此服務前必須同時購買Certified Mail。
      • Certificate of Mailing：郵寄證明，限郵局購買，費用為$1.65。
      • Signature Confirmation：提供收件人簽名收據以及投遞紀錄，收件人需要出示ID，費用為$2.9（線上付費）、$3.45（郵局購買）。
      • Registered Mail：USPS最安全的郵寄方式，可投保達$50,000，費用$13.75起。
      • Special Handling – Fragile：費用$12.15起。
    USPS Tracking®與Certified Mail不同
    Priority Mail和Priority Mail Express內含的USPS Tracking®僅具有追蹤信件功能，但收件人不需要簽收。Certified Mail同時具有追蹤信件並要求收件人簽收。

    國際郵寄 #

    包裹郵寄均有限重70 lbs、「最大邊長」和「另兩邊周長」的總和不得超過108"的限制。國際包裹不會主動報關，也就是不一定會被課稅。

    • First-Class Mail International：USPS最便宜的信封國際郵寄服務。郵資根據為明信片（Postcard）、信件（Letter, < 3.5oz）、大信封（Large Envelope, < 16oz）有所不同。由於無法追蹤信件去向，重要文件不推薦使用此服務。

    • Priority Mail Package International：USPS最便宜的包裹國際郵寄服務。約6~10工作天送達。附帶追蹤、$100的物品價值保險、$200的商品價值保險（可額外購買超額保險）。

      郵資計算分為兩種：

      • 按重量、國家，可透過線上工具預估費用。
      • Flat Rate：使用USPS專用信封/紙箱，不論內容物郵資固定，詳情參考連結。

    • Priority Mail Express International：約3~5工作天送達。附帶追蹤、$100的物品價值保險、$200的商品價值保險（可額外購買超額保險）。有提供Flat Rate Envelopes（限重4 lbs）。

    • Global Express Guaranteed：USPS最快捷的國際郵寄服務，約1~3工作天送達。附帶追蹤、$100的價值保險。

    Informed Delivery #

    USPS免費功能，需要線上免費註冊。收件者能在線上預覽郵件並管理即將抵達的包裹。使用者用戶可以從email、網頁帳戶、USPS Mobile app查詢Priority Mail®、Priority Mail Express® 和其他可追踪的包裹資訊，包含信件外部、追蹤訊息。缺點是同地址的信都可能會一併發送。

    USPS Electronic Signature Online #

    Informed delivery中的免費附屬功能，需先註冊Informed delivery後才能註冊。如果包裹需要簽收（例如台灣EMS包裹、Priority Mail Express、 保險金額超過$500等），收件人經由Informed delivery得知包裹訊息後，可以使用電子簽收以代替在家親自簽名。電子簽名在USPS 存檔一年，過期須更新。

    親自簽收
    如果沒有申辦Electronic Signature，像是台灣寄來的包裹都是要求親自簽收。郵政投遞的時候並不會打電話也不一定會按鈴通知，通常僅留下「PS Form 3849 Redelivery Notice」。可在這張紙條上簽名或親自到當地郵局領取。可以向物業/房東或鄰居詢問是哪個USPS辦公室負責你的居住區域。

    郵件轉寄 #

    當收件人搬家（暫時或永久）時，可以向USPS申請轉寄郵件。除了廣告郵件，所有USPS的郵件都會轉寄。申請方法分為兩種：

    • 線上：至Official USPS Change of Address填寫，並支付$1.1（身分驗證費）。
    • 現場：至任意USPS辦公室，向行員索取Mover’s Guide packet並填寫PS Form 3575（免費）。

    郵票/郵箱販售 #

    • 郵票與Envelope/Box在USPS辦公室、線上均有販售。

    • USPS Envelope/Box有分Flat rate版和一般版。Flat rate版封面會註明Flat Rate。


    • 郵票和Flat Rate Envelope/Box均有分普通版和Forever版。兩者當下價格一樣，但Forever版不會隨郵資變化而需要補交郵資。

    • 線上訂購郵票是免運但會被收取固定手續費（$1.4）；線上訂購Envelope/Box是免費、免運且無手續費（Forever版除外），詳情參考Postal Store購買連結。

      避免濫用/誤用USPS Envelope/Box
      不論是否使用過，使用USPS Envelope/Box到其他私人公司郵寄均是違反聯邦法律。其他濫用行為亦可能違反聯邦法律。

    商業快遞公司 #



    UPS是全球最大的商業快遞公司。國際包裹會主動報關，會按照保險金額課稅，但清關速度較USPS快。除了提供快遞服務以外，UPS也提供打包服務、Pack Ship Guarantee、公證服務（Notary Public）、拍大頭照等。

    FedEx是美國第二大的商業快遞公司。國際包裹會主動報關，會按照保險金額課稅，但清關速度較USPS快。除了提供快遞服務以外，FedEx也提供列印服務、拍大頭照等。

    注意照片尺寸
    一般UPS、FedEx拍攝預設是2"x2"，如果有其他尺寸需求（如台灣護照）記得提前告知。

    第三方快遞公司 #

    這類公司大多是與FedEx或UPS合作，提供更為優惠的費率。有任何第三方快遞公司推薦，歡迎到意見回饋提供。
