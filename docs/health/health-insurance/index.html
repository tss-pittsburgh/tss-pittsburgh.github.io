    健康保險攻略 #

    最後更新日期2022.01.17

    健康保險（Health Insurance）可分為醫療保險（Medical Plan）、牙科保險（Dental Plan）、眼科保險（Vision Plan）。學校健康保險資訊請參照投保健康保險章節。

    通用術語 #

    • Premium：即保費，通常是月付。
    • Deductible：保險公司開始賠付前，投保人須在投保週期內（通常1年內）先行支付的累積費用。
    • Copayment (Copy)：每次看病、治療時須支付的固定金額，類似台灣的掛號費。通常有Copayment的醫療服務是不需要先滿足Deductible要求。以Pitt Genernal Student Medical Plan為例，Office visits、Urgent care、領處方藥需要負擔Copayment但不需要先負擔$250的Deductible。
    • Coinsurance：投保人所須要支付醫療費用的百分比，類似台灣的部分負擔。使用前，投保週期內先要負擔完Deductible。
    • Cost sharing：Deductible、Copayment、Coinsurance合計。
    • Out-of-pocket Limit：投保人在保週期內最多須支付的Cost sharing總額，不含保險未涵蓋或拒絕支付的醫療費用。支付超過此金額後保險公司會100%報銷所有有涵蓋的醫療費用。
    • Maximum benefit：保險公司最高為投保人報銷的費用。依據險種不同，可以限定於單次、投保的一年內或終身。如果沒有上限，則為無。
      保費關聯
      Low Deductible、Low Copayment、Low Coinsurance、Low Out-of-pocket Maximum與High/Unlimited Maximum benefit通常保費會較為昂貴。
    • Coverage：保險所涵蓋、會報銷的醫療項目與服務。
    • Waiting Period：某些疾病或治療在投保一定時間後才會報銷。
    • Pre-existing Conditions：部分醫療保險會拒絕理賠購買保險前已經存在的疾病、傷害或醫療需求（例如懷孕等）。學校保險均沒有此限制。
    • In-/Out-of-network：In-network是指跟保險有合作關係的醫療院所，報銷比例高。Out-of-network的醫療院所報銷低或甚至不報銷。

      In-network facility ≠ In-network provider
      即使在In-network的醫院看診，但並代表每個在該醫院提供服務的醫生都是In-network。 例如：接受手術前要確保外科醫生是In-network。

      Balance billing vs Surprise billing
      醫療提供者向患者索取醫療服務定價與保險負擔金額的差額，主要發生在Out-of-network。如果投保人獲得非預期的Balance bill，稱為Surprise billing。Surprise billing通常發生在前往In-network facility看診但無意間接受Out-of-network醫療人員的醫療服務。

      No Surprises Act
      No Surprises Act自2022.01.01起生效，旨在避免投保人收到Surprise bill，包含緊急醫療以及In-network醫院、門診手術中心獲取的醫療服務。從雇主、Marketplace®或直接從保險公司購買的個人健康保險，均受到No Surprises Act保護。在符合規定的In-network facility看診，禁止部分醫療服務（例如麻醉、病理、放射科等）Balance billing，其餘醫療人員（例如外科醫師）除非獲取患者書面同意並放棄法案保護，否則患者不需負擔Balance bill。

    • Preauthorization：預授權，又稱Precertification、Prior approval。在接受部分醫療措施、手術前需要獲得保險公司的授權。
      • 何種醫療措施需要獲得預授權：非緊急況住院、手術、救護運輸；耐用醫療設備（Durable Medical Equipment，例如拐杖、輪椅）；處方藥提前補充等。
        緊急情況住院
        緊急情況住院須一定時間內（通常是48小時內）向保險公司補提出Preauthorization。
      • 審查內容：驗證是否有資格獲得保險理賠、審查診斷和治療計劃、評估護理在醫學上是否必要和適當、授權護理並為住院患者安排適當的住院時間。
      • 預授權責任：通常In-network的預授權是醫療提供者的責任，而Out-of-network的預授權是投保人的責任。如果In-network的預授權被保險公司拒絕，投保人是不需要負擔在收到通知前的醫療費用。相反，如果Out-of-network的預授權被保險公司拒絕，投保人需要自行負擔所有費用。
        未提交預授權
        雖然未提交預授權並不會一定導致被拒絕理賠，但獲取預授權可避免不可預期的醫療費用。

    依照不同看診、轉診模式，主要可分為4種類型：

    • Health Maintenance Organization (HMO)：投保人須要指定家庭醫生，專科診察須要家庭醫師轉診，只報銷In-network的醫療服務（緊急狀況除外），自由度低。
    • Exclusive Provider Organization (EPO)：專科診察不須經由轉診，但仍只報銷In-network的醫療服務（緊急狀況除外），自由度比HMO稍高。
    • Preferred Provider Organization (PPO)：專科診察不須經由轉診外，也報銷Out-of-network的醫療服務，但報銷費用會少In-network許多，自由度最高，保費一般較貴。
    • Point of Service (POS)：投保人須要指定家庭醫生，專科診察須要家庭醫師轉診。與PPO相同，報銷Out-of-network的醫療服務。計畫介於HMO與PPO之間。

    醫療保險術語 #

    • Preventive Care Services：預防醫學服務，包含年度體健、疫苗接種、常規性篩檢與驗血，通常不包含成人眼科與牙科。一般保險需要涵蓋的內容可參照HealthCare.gov，Pitt與CMU保險可參照UPMC、Highmark預防醫學項目。
      免費Preventive Care Services
      根據Afforable Care Act，保險提供的Preventive Care Services是不需要支付Deductible、Copayment和Coinsurance，可以完全免費使用，但必須在In-network使用。
    • Primary Care Provider Visit：初級保健看診，詳細內容可參照初級保健章節。
    • Specialist Visit：專科看診，例如皮膚科、婦產科。
    • Urgent Care Visit：緊急護理是指需要即時治療但通常沒有嚴重到需要急診的疾病或傷害。常見需要Urgent Care的情況可參考連結。
    • Emergency Room Services：急診，在一般常識下合理預期不立即就醫會導致嚴重危害生命、身體機能嚴重損害、器官等功能損害。例如：事故造成的身體傷害（刀傷、槍傷、車禍），嚴重的急性症狀或劇烈疼痛、突然的醫療狀況（呼吸困難、胸部持續疼痛或壓力、嘴唇或臉發青）。
    • Virtual Visit：線上看診（Telemedicine）。例如UPMC AnywhereCare、Teledoc、Amwell等。
    • Prescription Drug：處方藥。每個保險都會有自己的處方集（formulary或Preferred drugs），使用這些藥會較為便宜。按照來源通常分為原廠藥（Brand drugs）與學名藥（Generic drugs），使用學名藥會較為便宜。依照藥的類型分為零售（Retail）、特殊（Speciality）以及長期（Mail-order）處方籤。
    • Hospital Services：包含Inpatient（住院）、Outpatient（門診）、Observation stay（留院觀察，判斷是否需從門診過轉至住院的過程）。
    • Surgical Services：手術服務。
    • Diagnostic Services： 診斷服務，保險通常分為基礎診斷（例如：超音波、X光、驗血或病理切片等）與進階影像（例如：電腦斷層掃描、磁振造影、正子造影等）。
    • Rehabilitation Therapy Services：康復治療服務，包含物理（熱療、電療、光療等）與職能治療、語言治療、心臟復健等。
    • Dental Services related to accidental injury：意外下造成的牙齒損壞的修補。
    • Facility Fee：設施費，看診地在非該醫生所擁有的醫療院所可能需要支付的費用。醫院通常會收取設施費以支付維護該設施的費用。 設施費從十幾美元到數百美元不等，具體取決於使用的服務。
      注意Facility Fee
      大多數的醫療保險如CMU的Highmark並不承擔Facility Fee，因此到醫院看診可能要完全自費承擔Facility Fee。Pitt的UPMC醫療保險涵蓋Facility Fee。

    HSA #

    Health Savings Account (HSA)是一種儲蓄帳戶，HSA有以下特色：

    • 僅限購買高免賠額醫保計劃（High Deductible Health Plan, HDHP）開戶。以2022年為例，個人保險的Deductible需要在$1,400以上。
    • 允許將收入以稅前（Pre-tax）方式的存入，每年有存款上限。以2022年為例，55歲以下的個人帳戶能存入$3,650。
    • 通常有提供HSA Debit Card。
    • 在年滿65歲前，帳戶資金僅能用於支付自己、配偶以及被扶養人的符合資格的醫療費用（qualified medical expenses），否則會有罰款。合格的醫療費用可參照HSA Bank。
    • 存入的資金不限制當年用完，並且帳戶所賺的利息或投資營利均免稅。
    高免賠額醫保計劃購買限制
    留學生或J-1簽證通常不允許購買此類保險計畫。

    FSA #

    Flexible Spending Account (HSA)是一種雇主提供的特殊帳戶，有以下特色：

    • 不受醫療保險計畫限制。
    • 帳戶擁有者為雇主，通常離職後帳戶帳戶即作廢。
    • 允許將收入以稅前（Pre-tax）方式的存入，每年有存款上限。以2022年為例，Heathcare FSA個人帳戶能存入$2,850。
    • 帳戶資金僅能用於支付自己、配偶以及被扶養人的符合資格的醫療費用。由於帳戶屬於雇主，員工未必能隨意動用帳戶資金用於非合格醫療費用。
    • 帳戶資金必須當年用完，逾其作廢。最多允許2.5月的寬限期或轉存$570到下年度使用。
    • 無利息或投資用途。

    牙科保險術語 #

    牙科保險項目主要分為4類服務（可能根據保險公司有所差異）：

    • Class I：預防診斷類別，包含口腔檢查（Exam）、X-rays、常規洗牙（Cleanings）、塗氟（Fluoride Treatment）等，學校保險都是100%報銷且不計入Deductible。一般無保險的情況下，光檢查費用就要$200。
    • Class II：基礎治療，包含補牙（Fillings）、拔牙（Extractions）、根管治療（Root Canal/Endodontics）、深度洗牙（Periodontal Maintenance）等。大多數保險的報銷為70%~80%。
    • Class III：主要治療，包含烤瓷牙（Crown）、植牙（Implant）、假牙（Denture）。大多數保險的報銷為50%。
    • Class IV：牙齒矯正（Orthodontia），通常報銷與上述3類分開計算，通常僅限19歲以下。
    Pitt Undergraudate Dental Plan
    雖然全美任意牙醫均可看診，但報銷內容差。
    意外損傷
    部分醫療保險承保意外所致的牙齒損傷，例如Pitt和CMU提供的醫療保險。

    眼科保險 #

    報銷/折抵驗光、眼睛檢查（包含散瞳）、眼鏡（配鏡架和鏡片）、隱形眼鏡等方面的費用。惟須注意，除了眼鏡、隱形眼鏡可在台灣多買幾份備份替代此保險，眼睛疾病、治療是使用醫療保險而非眼科保險。

    眼睛疾病、治療是使用醫療保險而非眼科保險。

    變更保險 #

    變更保險包含中途加入或退出保險（含個人、配偶、受扶養人）。一般而言，為了避免等到生病才想投保，學校保險有限定的註冊期且不能任意退出。除非發生「合格生活事件」（Qualifying Life Event），才允許中途加入或退出保險。

    「合格生活事件」包含入學、結婚、生育、遷居、工作等都符合變更保險的要件，明細可參考HealthCare.gov清單。特別注意，生病、牙痛並不屬於「合格生活事件」。

    學校退保流程請參照畢業退保章節。
