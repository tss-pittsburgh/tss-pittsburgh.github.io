    看牙指南 #

    最後更新日期2022.02.01

    雖然台灣看牙是相當便宜與方便，但有時臨時的牙痛、補牙需求仍是需要在美國接受治療的。

    牙科保險/折扣計畫 #

    雖然牙科保險並非強制的，但牙科保險保費一年僅約$220並提供免費檢查與每半年一次的洗牙，推薦購買。如果沒有保險要檢查牙齒，可能光X光片就要$99。

    Pitt Undergraduate Dental Plan可斟酌考慮
    Pitt Undergraduate Dental Plan報銷福利相對一般，可依個人需求考慮是否選擇校外更好的保險。

    由於學校的牙科保險有註冊期的限定，如果臨時有看牙需求但沒有學校保險的情況下可考慮購買校外牙科保險或牙科折扣計畫（Dental Discount Plan）。

    • 校外牙科保險：如Delta Dental，購買校外保險前需要確定是否有Waiting Period。如果來不及等Waiting Period結束，可以告知診所狀況，診所可以協助以適當的方法規避Waiting Period。

    • 牙科折扣計畫：如DentalPlans.com，類似會員折扣計畫，費用為年付。通常沒有Waiting Period、Dedutible與Maximum Benefits，但限定於In-Network且沒有免費口腔檢查與護理。

    就醫流程 #

    指定初級牙醫（限HMO） #

    僅有HMO類型的牙醫保險需要指定初級牙醫（Primary dental office）。HMO所有看診都先需要在該診所進行，如果牙醫師認為需要轉診會開立轉診單。彈性較差但通常保險福利較佳。

    • Pitt Graudate and Professional Plan：屬於HMO，看診前需要設定Primary dental office（限賓州）。選取牙醫師可至United Concordia Dental的Find a Dentist網站，輸入地點與DHMO Concordia Plus General Dentist尋找，確認後須撥打866-357-3304註冊。

    預約牙醫 #

    選好牙醫診所後打電話預約即可。第一次預約時會索取保險信息。牙醫選擇可參考Google上的用戶評價或在TSS社團上詢問意見。

    • Pitt Undergraudate Plan：屬於Flex類型，可以全美任意的牙醫診所看診，沒有In-Network或Out-of-network差異。
    • CMU Blue Edge Dental Plan：屬於PPO類型，不需要指定初級牙醫，但有分In-Network與Out-of-network。Highmark In-network牙醫可至Highmark網站上尋找。

    洗牙/看診 #

    初診時記得提早15分鐘報到並攜帶保險卡，需要填寫填寫個人資訊、病史等表格。

    每家診所對初診病患會先安排拍全口X光、口腔檢查以及牙洗，一般需要約45分鐘。最後，牙醫會檢查口腔與X光片。如果沒問題，會繼續安排6個月後洗牙。如果牙齒有問題會安排「下一次」治療。如果有事類似補牙掉落等缺口，會做一個臨時填補。如果有保險的話，這些檢查項目是全免費的，也不會被付Deductible。

    美國與台灣常見差異：

    • 美國的補牙都會周圍區域上麻醉（先挨幾針）。
    • 補完牙可以馬上吃東西。

    初診通常不會治療
    如果在美國沒有看過牙醫的紀錄，等牙齒出了問題才看。縱使覺得狀況很簡單，但仍需先預約一次拍片、檢查、洗牙，然後等下一次正式治療（通常是1~2個月後）！

    洗牙好處
    美國的洗牙是由專門的洗牙師處理，過程非常仔細。每次洗牙後，還可以領牙刷、電動刷頭、牙膏、牙刷袋等免費樣品！

    費用 #

    • Class I：預防診斷類別，包含口腔檢查（Exam）、X-rays、常規洗牙（Cleanings）、塗氟（Fluoride Treatment）等。**預防診斷類別通常都是100%報銷且不計入Deductible。**一般無保險的情況下，光檢查費用就要$200。
    • Class II：基礎治療，包含補牙（Fillings）、拔牙（Extractions）、根管治療（Root Canal/Endodontics）、深度洗牙（Periodontal Maintenance）等。
    • Class III：主要治療，包含烤瓷牙（Crown）、植牙（Implant）、假牙（Denture）。
    • Class IV：牙齒矯正（Orthodontia），通常報銷與上述3類分開計算，通常僅限19歲以下。學校保險通常不報銷。

    以下為Pitt和CMU牙科保險方案：

    • Pitt Undergraduate Plan：報銷比例Class I 100%、Class II 50%、Class III 0%。報銷福利差。不同診所的收費有所差異，因此務必在接受治療前問清楚價格。
    • Pitt Graudate and Professional Plan：報銷比例Class I 100%、其餘Class II、Class III採用Copyament。在United Concordia網頁中的Covered Services可以查到所有治療的費用，在哪家接受治療價格都一樣（僅Pitt牙醫學院會有額外折扣）。常見費用：
      • 補牙：$18
      • 根管治療：約$150
      • 烤瓷牙：約$350

    • CMU Blue Edge Dental Plan：In-network報銷比例Class I 100%、Class II 80%、Class III 50%。不同診所的收費有所差異，因此務必在接受治療前問清楚價格。

    根管治療費用
    台灣的根管治療是由健保補助，但假牙冠是全自費項目，依照不同的材料費用有所不同。美國採用的烤瓷牙在台灣自費為2~3萬台幣，因此整套根管治療+烤瓷牙在美國處理並不會太貴。

    拔智齒費用
    台灣的拔智齒是由健保補助，美國需要負擔部分費用，因此推薦在台灣處理。

    免費/無保險看牙 #

    • 擔任牙科學校的志願者：牙科或口腔衛生學校定期需要志願者給學生練習或考核用途，從洗牙、補牙等任何常規診療。擔任志願者通常可以以相當便宜或免費接受服務，但代價就是給學生們練習。

    • Mission of Mercy Pittsburgh：定期提供免費基礎牙齒護理與治療給無法負擔的人。
