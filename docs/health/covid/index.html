    疫情生活指南 #

    最後更新日期2021.06.19

    2019年爆發新型冠狀病毒（COVID-19），各國、各州間均有不同的旅行禁令、檢疫要求，最新要求務必參照官方指示。

    入境美國與賓州旅行要求 #

    • 美國目前對台灣沒有旅行禁令。

    • 抵美前14天若曾至（包含轉機或過境）中國大陸（除香港、澳門）、申根地區、英國、印度等外籍旅客禁止入境，詳情參閱美國CDC網站。

      • 美國公民、美國公民、永久居民以及其配偶、未成年眷屬不在此限，詳情參閱總統令。
      • 自2021.08.01，F1、M1、J1簽證不在此限（並未包含眷屬），詳情參閱總統令。

    • 自2021.01.26起登機前須檢具航班起飛前3日內核酸增幅檢測或抗源檢測報告（紙本或電子影本）或3個月內已感染且康復之證明文件，詳情參照美國CDC網站與 陰性證明章節。

    • 自2021.03.01起，賓州政府取消州外旅遊限制、隔離要求。

    • 自2021.05.19起，已完全接種疫苗的人在美國境內旅遊不需要額外篩檢或隔離。

    COVID-19疫苗接種 #

    目前全美不論有沒有保險，任何人都可以「免費」施打COVID-19疫苗。目前美國FDA緊急授權的疫苗有Pfizer（輝瑞，2劑，每劑間隔至少21天，12歲以上）、 Moderna（莫德納，2劑，每劑間隔至少28天，18歲以上)、Johnson & Johnson（嬌生，1劑，18歲以上）。

    如何預約：線上預約為主，且方便。電話預約亦可。少數提供Walk-in。施打地點可能是醫院、藥局、體育中心或是賣場等。預約時就會告知這個地點會提供什麼疫苗牌子，基本上只要有提供都可以自由選擇。

    • PittCoVax：可以Walk-in，不限Pitt學生。
    • CDC VaccineFinder
    • Allegheny County Vaccine Registration
    • Rite Aid
    • Giant Eagle Pharmancy（可以Walk-in）
    • CVS（可以Walk-in）
    • Costco

    所需文件：Photo ID（如護照或美國駕照）、保險卡（如果有保險就提供，沒有也非必要）、COVID-19疫苗卡（施打第二劑用）

    返台核酸檢測 #

    啟程地時間自2020.12.01起，入境台灣或於台灣機場轉機之旅客登機前均應檢附「表訂登機時間前3日（工作日）內COVID-19核酸檢驗陰性報告（紙本或電子影本），詳情參閱衛服部常見問答Q49或駐紐約辦事處網站。

    台灣定義的3日內
    「3日內」檢驗報告日期是以「報告日」及「工作日」計算，排除當地政府之國定假日，不含搭飛機日。

    檢驗方法 #

    檢驗報告需採分子生物學核酸檢測，包含

    • PCR、Real-Time PCR、RT-PCR、RT-qPCR
    • NAAT（Nucleic acid Amplification Test）、NAT（Nucleic acid Test）
    • LAMP（Loop-Mediated isothermal Amplification）、RT-LAMP、
    • COVID-19 RNA test、SARS-CoV-2 RNA test
    • Molecular Diagnostics

    惟須注意，血清免疫學檢驗抗原(Antigen Test)或抗體（IgG或IgM）不符合檢驗要求。

    檢驗報告內容 #

    檢驗報告內容須包含

    • 登機者之護照姓名
    • 出生年月日(或護照號碼)
    • 疾病名稱「COVID-19或SARS-CoV-2」
    • 檢驗方法
    • 判讀結果「Negative、undetectable、陰性、未檢出……」
    • 採檢日及報告日。

    美國檢測機構 #

    大多數檢測機構無論是否有保險均免費，沒保險的話檢查費用是由聯邦政府吸收。如果有保險，需要提供保險資料，未填則仍會被查詢保險紀錄。

    常見核酸檢測機構：

    • CMU Tartan Testing：CMU教職員生免費，每週可檢測1次。校內檢測站自行使用檢測包。
    • Curative：以廂型車作為檢查站，專人檢測。結果約在1~2天出爐。
    • CVS：必須開車經Drive-thru車道，並在指導下自行使用檢測包。結果約在1~2天出爐。
    • Rite Aid：必須開車經Drive-thru車道，並在指導下自行使用檢測包。結果約在2~5天出爐。
    • Walgreen: 必須開車經Drive-thru車道，並在指導下自行使用檢測包。Diagnostic Lab Test (PCR)約48小時內出爐、Rapid Diagnostic Test約24小時內。
    • Pixel by LabCorp：檢測包寄至住處，自行採樣由FedEx寄回。
    • MedExpress：有保險免費，沒有保險的話會被收取$119~$229費用。結果約在2~3 天出爐。

    Drive-thru車道
    任何Drive-thru車道檢測均不允許步行參與。

    謹慎選擇Rapid Test
    當日可獲得測試結果通常屬於抗原檢測（Rapid Antigen Test），不能作為回台登機前的檢驗報告。Walgreen的Rapid Diagnostic Test是NAAT檢測，雖符合規定但需要準備有可能航空公司搞不清楚而耽誤行程。
