    COVID-19篩檢 #

    最後更新日期2022.02.01

    美國檢驗機構 #

    大多數檢驗機構無論是否有保險均免費，沒保險的話篩檢費用是由聯邦政府吸收。如果有保險，需要提供保險資料，未填則仍會被查詢保險紀錄。

    以下COVID-19檢驗機構均提供核酸檢驗，部分檢驗機構亦提供抗原檢驗：

    • CMU Tartan Testing：CMU教職員生免費，每週可檢驗1次。校內檢驗站自行使用檢驗包。
    • Curative：以廂型車作為檢查站。結果約在1~2天出爐。
    • Walgreen: 必須開車經Drive-thru車道，並在指導下自行使用檢驗包。Diagnostic Lab Test (PCR)約48小時內出爐；Rapid Diagnostic Test (ID NOW)約24小時內，通常在幾小時內出爐；Rapid Antigen Test (BinaxNOW)約1小時內出爐。
    • CVS：必須開車經Drive-thru車道，並在指導下自行使用檢驗包。Lab Test (PCR/ NAAT)結果約在1~2天出爐;Rapid-result (PCR)約幾小時內出爐;Rapid-result (antigen)約幾小時內出爐。
    • Rite Aid：必須開車經Drive-thru車道，並在指導下自行使用檢驗包。結果約在2~5天出爐。
    • Pixel by LabCorp：檢驗包寄至住處，自行採樣由FedEx寄回。
    • MedExpress：有保險免費，沒有保險的話會被收取$119~$229費用。結果約在2~3天出爐。

    Drive-thru車道
    任何Drive-thru車道檢驗均不允許步行參與，可開車、騎乘摩托車、腳踏車或滑板車。

    Walgreen Rapid Diagnostic Test
    Walgreen的Rapid Diagnostic Test (ID NOW)屬於NAAT檢驗，可用於返台，但據點、名額較少，需要提早預約。

    居家快篩試劑 #

    自2022.01.15起，擁有雇主贊助或個人健康保險的人，每個月每人最多可以報銷8次、家庭醫保最多報銷32次FDA批准的非處方COVID-19居家快篩試劑。這項規定並不適用於Medicare的投保人。

    不是按照購買盒數計算
    次數是按照實際可測試的次數，而非購買盒數。一個包裝盒中可包含多個測試。例如，兩個測試包裝在一個盒子中，會計算成2次測試。

    美國政府 #

    每個「家庭」都有資格訂購 4 次免費的COVID-19居家快篩試劑，透過USPS發送且無須運費。測試完全免費，不需要保險。訂購連結。

    同地址僅限4次
    由於訂購紀錄會依據地址留存，因此如果家裡有其他室友已經使用完額度，會導致訂購被拒。

    UPMC Health Plan #

    Pitt的學校保險。目前主要是先買後報銷的形式，FDA批准的居家快篩試劑均可，詳細清單參考連結，報銷流程可參閱UPMC Health Plan。

    Highmark Blue Shield #

    CMU的學校保險。2022.01.31 Highmark宣布新的報銷規定，詳情規定參考Highmark Health Plan網頁。

    可以透過以下兩種方式獲得居家快篩試劑，不需要先行付費、無須走報銷流程：

    1. 透過Express Scripts線上訂購。
    2. 在零售藥局的處方櫃台購買，需攜帶保險卡，限購買以下產品：
      • Binaxnow COVID-19 AG Self Test
      • Ellume COVID-19 Home Test
      • Flowflex COVID-19 AG Home Test

    其餘購買途徑需透過郵寄或線上報銷獲得退款。惟須注意並非所有的FDA批准的居家快篩試劑都可以被Highmark醫保報銷，詳細清單參考如下。

    返美COVID-19檢驗 #

    自2021.12.06起，前往美國班機登機前須檢具航班起飛前1日內核酸增幅檢驗或抗源檢驗報告（紙本或電子影本）或90天內已感染且康復之證明文件，詳情參照美國CDC網站。

    台灣機構檢驗 #

    詳情參照美國CDC網站。

    • 提供PRC採檢醫院參閱附件表四。選擇使用PCR檢驗，務必選擇「急件」，並確認航班是否趕得及。
    • 部分醫院亦提供抗原快篩（約1,000~2,500台幣）。

    居家快篩試劑 #

    美國允許使用居家快篩試劑（self-test或home test）作為返美檢驗證明，其主要規定包含試劑獲得FDA批准、檢測過程中必須以視訊等方式監督等。

    使用居家快篩試劑作為入境檢驗證明規定
    • The test must be a SARS-CoV-2 viral test (nucleic acid amplification test [NAAT] or antigen test) with Emergency Use Authorization (EUA) from the U.S. Food and Drug Administration (FDA) OR the relevant national authority where the test is administered.
    • The testing procedure must include a telehealth service affiliated with the manufacturer of the test that provides real-time supervision remotely through an audio and video connection. Some FDA-authorized self-tests that include a telehealth service may require a prescription.
    • The telehealth provider must confirm your identity, observe the sample collection and testing procedures, confirm the test result, and issue a report that meets the requirements of CDC’s Order (see “What information must be included in the test result?” below).
    • Airlines and other aircraft operators must be able to review and confirm your identity and the test result details. You must also be able to present the documentation of test results to U.S. officials at the port of entry and local/state health departments, if requested.

    目前主要有兩款符合美國入境規定的居家快篩試劑：

    BinaxNOW COVID-19 Antigen Home Test #


    Abbott與eMed合作的旅行用居家快篩試劑。目前僅在eMed和Optum線上販售，2套$70、3套$100、6套$150。使用流程可參考Trip+經驗分享。

    CVS、Walgreen或Amazon等零售藥局所販售的BinaxNOW COVID-19 Self Test是另一種產品，不能用於入境美國的檢驗證明。

    Ellume COVID-19 Home Test with Azova #


    Ellume與Azova合作的旅行用居家快篩試劑。目前僅在實體藥局販售（$39/套@CVS），使用時須在Azova網頁額外支付$20並預約進行視訊篩檢。

    返台COVID-19檢驗 #

    啟程地時間自2022.01.04起，入境台灣或於台灣機場轉機之旅客登機前均應檢附「表訂登機時間前2個日曆日內COVID-19核酸檢驗陰性報告（紙本或電子影本），報告起計日以「採檢日」為基準。詳情參閱衛服部常見問答Q49。

    檢驗方法 #

    檢驗報告需採分子生物學核酸檢驗，包含

    • PCR、Real-Time PCR、RT-PCR、RT-qPCR
    • NAAT（Nucleic acid Amplification Test）、NAT（Nucleic acid Test）
    • LAMP（Loop-Mediated isothermal Amplification）、RT-LAMP
    • COVID-19 RNA test、SARS-CoV-2 RNA test
    • Molecular Diagnostics
    血清免疫學檢驗抗原（Antigen Test）或抗體（IgG或IgM）不符合台灣入境檢驗要求。

    檢驗報告內容 #

    檢驗報告內容須包含

    • 登機者之護照姓名
    • 出生年月日(或護照號碼)
    • 疾病名稱「COVID-19或SARS-CoV-2」
    • 檢驗方法
    • 判讀結果「Negative、undetectable、陰性、未檢出……」
    • 採檢日及報告日。
